40+ downloadable textbooks spanning 10 subject areas. Every title is free under CC BY-SA 4.0. The breadth is the point—this framework handles Aristotle and algebra, Mencken and molecular biology, Machiavelli and middle-school math.

Browse the Library Course Catalog

Deliberate Variety

The range of subject matter is intentional. This collection includes Classical Christian texts alongside secular history, Nietzsche alongside monasticism, Mencken's provocations alongside elementary reading companions. The variety demonstrates that the framework is content-agnostic—it handles any subject, any perspective, any grade level.

Some titles will challenge readers from one direction; others will challenge them from another. That is by design. A framework that can only produce one kind of book isn't a framework—it's a template.

Disclaimers

AI-Assisted Content

All textbooks and multimedia were created through human-AI collaboration using Claude Code (Anthropic) and Google NotebookLM. Content may contain errors. These materials are provided as-is and should be reviewed before classroom use. This is a beta release and a work in progress.

Perspective & Viewpoint

Several titles are written from a Classical Christian educational perspective. Others are secular. Some include provocative thinkers (Mencken, Nietzsche, Machiavelli) whose views many will find challenging. The author is Catholic; Claude tends toward mainstream consensus. All perspectives, including mainstream ones, should be examined critically.
