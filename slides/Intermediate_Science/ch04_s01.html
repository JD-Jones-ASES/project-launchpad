Key Concepts (1/2)

Reference Frame

A coordinate system, including an origin and defined directions, against which the position and motion of an object are measured. Without a reference frame, "position" is meaningless.

Position ($x$)

The location of an object relative to the chosen origin of a reference frame. It is a vector quantity, meaning it includes both how far from the origin (magnitude) and in which direction (sign in 1D).

Distance

The total length of the path an object travels, irrespective of its direction. It is a scalar quantity, always positive, and only indicates the "how much ground covered."

Scalar vs. Vector

Fundamental categories of physical quantities. Scalars have only magnitude (size), while vectors have both magnitude and direction. Distance is a scalar; position and displacement are vectors.

Important Terms (1/2)

Reference Frame

A coordinate system with an origin and directions, used to describe an object's position and motion.

Origin

The chosen zero point within a reference frame from which all positions are measured.

Position ($x$)

The location of an object relative to the origin of a reference frame, typically measured in meters (m) and including a direction (e.g., + or -).

Distance

The total length of the path traveled by an object, always a positive value, measured in meters (m).

Scalar

A physical quantity that has magnitude (size) only, without an associated direction (e.g., distance, speed, mass, temperature).

Important Terms (2/2)

Displacement ($\Delta x$)

The change in an object's position, representing the straight-line distance and direction from its starting point to its ending point, measured in meters (m).

Vector

A physical quantity that has both magnitude (size) and direction (e.g., displacement, velocity, force).

Position-Time Graph ($x$-$t$ graph)

A graph showing an object's position on the vertical axis against time on the horizontal axis, used to visualize and analyze motion.

Distance: How Far You Travel

Definition

*Distance* is the total length of the path an object has traveled, regardless of any changes in direction.

Example

If you walk 100m to the cafeteria and then 100m back to your classroom, your total distance traveled is 200m. Even though you ended up where you started, you still covered that ground.

Scalar Quantity

Distance is a *scalar*. A scalar quantity has only *magnitude* (a numerical value with units) but no associated direction. Other examples include speed, mass, and temperature.

Displacement: How Far and Which Way

Definition

*Displacement* ($\Delta x$) is the change in an object's position. It measures the straight-line distance and direction from the starting point to the ending point.

Example

If you walk 100m to the cafeteria and then 100m back, your *distance* is 200m, but your *displacement* is 0m because you ended up at your starting point.

Vector Quantity

Displacement is a *vector*. A vector quantity has both *magnitude* (size) and *direction*.
