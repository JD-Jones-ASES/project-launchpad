Reading the Periodic Table: A Comprehensive Study Guide

Study Slides

Key Concepts (1/2)

Periodic Table as a Tool

The periodic table is a fundamental resource providing extensive information about each element.

Information in Each Square

Each element's entry typically includes its atomic number, element symbol, element name, and average atomic mass.

Predicting Ion Charges

Main-group elements form ions to achieve a stable octet (or duet) of valence electrons, mimicking noble gas configurations. This allows for prediction of common ion charges based on group number.

Electron Configuration

The periodic table can be used to infer an element's electron configuration, particularly the number of energy levels and valence electrons.

Key Concepts (2/2)

Average Atomic Mass

This value is the weighted average of the masses of all naturally occurring isotopes of an element, taking into account their relative abundances. It is not the mass of a single atom.

Oxidation States

These describe the charge an element takes on in compounds, which for main-group elements, largely aligns with the predicted ion charges.

Henry Moseley's Contribution

Moseley's work established that atomic number, not atomic mass, is the fundamental property defining an element's identity and its position on the periodic table, resolving inconsistencies.

Important Terms (1/3)

Atomic Number

The number of protons in an atom's nucleus, which uniquely defines an element.

Element Symbol

A one- or two-letter abbreviation for an element's name (e.g., C, Fe, Na).

Average Atomic Mass

The weighted average of the masses of all naturally occurring isotopes of an element, measured in atomic mass units (amu).

Atomic Mass Unit (amu)

A unit of mass used to express atomic and molecular weights.

Group Number

The number of a vertical column on the periodic table, which indicates the number of valence electrons for main-group elements (and thus similar chemical properties).

Important Terms (2/3)

Period Number

The number of a horizontal row on the periodic table, which indicates the number of occupied electron energy levels (shells) in an atom.

Valence Electrons

The electrons in the outermost occupied electron shell of an atom, involved in chemical bonding.

Octet Rule

Isotopes

Atoms of the same element (same atomic number) that have different numbers of neutrons and therefore different atomic masses.

Oxidation State

A number assigned to an element in a compound that represents the number of electrons lost or gained (or shared unequally) by an atom of that element.

Important Terms (3/3)

Main-Group Elements

Elements in Groups 1, 2, and 13-18 of the periodic table, characterized by predictable trends in electron configuration and reactivity.

Transition Metals

Elements in Groups 3-12, known for forming ions with multiple possible charges.

Metalloid

An element with properties intermediate between those of metals and nonmetals, typically found along the "staircase line" on the periodic table.

Information from Position

Group Number (Column)

For main-group elements (Groups 1, 2, 13–18), the group number tells you the number of **valence electrons**.

Period Number (Row)

The period number indicates the number of occupied **electron energy levels (shells)** in an atom of that element.

Metal, Nonmetal, or Metalloid

The elements are generally divided by a "staircase line" on the table.

Predicting Ion Charges (1/2)

Group 1 Metals

Lose 1 electron, forming ions with a +1 charge (e.g., Na$^+$).

Group 2 Metals

Lose 2 electrons, forming ions with a +2 charge (e.g., Ca$^{2+}$).

Group 13 Metals

Lose 3 electrons, forming ions with a +3 charge (e.g., Al$^{3+}$).

Group 15 Nonmetals

Gain 3 electrons, forming ions with a -3 charge (e.g., N$^{3-}$).

Group 16 Nonmetals

Gain 2 electrons, forming ions with a -2 charge (e.g., O$^{2-}$).

Predicting Ion Charges (2/2)

Group 17 Nonmetals (Halogens)

Gain 1 electron, forming ions with a -1 charge (e.g., Cl$^-$).

Group 18 Noble Gases

Already stable with a full octet (or duet), they generally do not gain or lose electrons and rarely form ions.
