Key Concepts (1/2)

What is the primary application of using proportions in similar figures?

To find unknown side lengths in similar figures, which is crucial for practical applications like indirect measurement.

Define the 'Scale Factor' of two similar figures.

The scale factor is the ratio of any pair of corresponding side lengths between the two similar figures.

If $\triangle ABC \sim \triangle DEF$ with $AB=6$ and $DE=9$, what is the scale factor from $\triangle ABC$ to $\triangle DEF$?

The scale factor is $AB/DE = 6/9 = 2/3$.

How does the order in which similar figures are listed affect their scale factor?

The scale factor depends on which figure is listed first. Reversing the order of the figures will result in the reciprocal of the original scale factor.

What is the standard algebraic technique for solving a proportion like $a/b = c/d$?

Cross-multiplication, which transforms the proportion into $ad = bc$.

Key Concepts (2/2)

Given $\triangle ABC \sim \triangle DEF$. If $AB=8$, $BC=12$, and $DE=10$, set up the proportion to find $EF$.

The proportion is $AB/DE = BC/EF$, which is $8/10 = 12/EF$.

For $\triangle ABC \sim \triangle DEF$, if $AB=8$, $BC=12$, and $DE=10$, what is the length of $EF$?

Using the proportion $8/10 = 12/EF$, cross-multiply: $8 \cdot EF = 10 \cdot 12 \implies 8EF = 120 \implies EF = 15$.

A flagpole casts a 12-foot shadow. At the same time, a 5.5-foot-tall student casts a 3-foot shadow. Set up a proportion to find the height ($h$) of the flagpole.

The proportion is $h/12 = 5.5/3$ (flagpole height/shadow = student height/shadow).

Calculate the height of the flagpole from the previous problem: a flagpole casts a 12-foot shadow, and a 5.5-foot student casts a 3-foot shadow.

Using $h/12 = 5.5/3$, cross-multiply: $3h = 5.5 \cdot 12 \implies 3h = 66 \implies h = 22$ feet.
