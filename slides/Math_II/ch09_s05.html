Data Displays and Measures of Spread

Study Slides

Key Concepts (1/2)

Measures of Center (Mean, Median, Mode)

These statistics provide a single value that attempts to describe the "typical" or central position of a data set.

Measures of Spread (Range, Interquartile Range)

These statistics describe how dispersed or varied the data values are.

Key Concepts (2/2)

Outliers

Data values that are significantly different from other values in the data set. An informal rule identifies potential outliers as values more than 1.5 times the IQR below Q1 or above Q3.

Important Terms (1/3)

Mean

The arithmetic average of a data set.

Median

The middle value of an ordered data set.

Mode

The value that occurs most frequently in a data set.

Range

The difference between the maximum and minimum values in a data set.

Histogram

A bar graph displaying the frequency distribution of data across consecutive intervals (bins).

Important Terms (2/3)

Bin

An interval used in a histogram to group data values.

Frequency

The number of times a particular data value or a value within a specific interval appears in a data set.

Five-Number Summary

A set of five values: minimum, first quartile (Q1), median (Q2), third quartile (Q3), and maximum, which summarize the distribution of a data set.

Quartile (Q1, Q2, Q3)

Values that divide an ordered data set into four equal parts.

Interquartile Range (IQR)

The difference between the third quartile and the first quartile ($Q_3 - Q_1$), representing the spread of the middle 50% of the data.

Important Terms (3/3)

Box Plot

A graphical representation of the five-number summary, showing the distribution and potential skewness of a data set.

Outlier

A data point that is significantly distant from other observations in a data set.

Measures of Spread: Range

The difference between the greatest (maximum) and least (minimum) values in a data set.
