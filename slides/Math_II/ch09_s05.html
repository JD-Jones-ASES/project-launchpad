Interquartile Range (IQR)

A measure of spread that represents the middle 50% of the data. It is calculated as the difference between the third quartile (Q3) and the first quartile (Q1): `IQR = Q3 - Q1`.

Key Concepts (2/2)

Box Plot (Box-and-Whisker Plot)

A visual representation of the five-number summary. It clearly shows the median, quartiles, and the overall spread and symmetry of the data, as well as potential outliers.

Outliers

Data values that are significantly different from other values in the set. Informally, a value is considered an outlier if it is more than `1.5 IQR` below Q1 or above Q3.

Important Terms (1/3)

Measures of Center

Statistical values (mean, median, mode) that describe the "middle" or "typical" value of a data set.

Mean

The arithmetic average of a data set.

Median

The middle value in an ordered data set.

Mode

The most frequently occurring value in a data set.

Range

The difference between the maximum and minimum values in a data set, indicating the total span of the data.

Important Terms (2/3)

Histogram

A type of bar graph that displays the frequency distribution of numerical data divided into intervals (bins).

Bins

The intervals or ranges into which data is grouped for a histogram.

Frequency

The number of times a particular data value or a value within a specific interval appears in a data set.

Five-Number Summary

A set of five descriptive statistics for a data set: minimum, first quartile (Q1), median (Q2), third quartile (Q3), and maximum.

Quartiles (Q1, Q2, Q3)

Values that divide an ordered data set into four equal parts. Q1 is the 25th percentile, Q2 (median) is the 50th percentile, and Q3 is the 75th percentile.

Important Terms (3/3)

Interquartile Range (IQR)

A measure of statistical dispersion, equal to the difference between the upper (Q3) and lower (Q1) quartiles.

Box Plot (Box-and-Whisker Plot)

A standardized way of displaying the distribution of data based on the five-number summary.

Outlier

A data point that lies an abnormal distance from other values in a random sample from a population.

Review of Measures of Center

Mean ($x$)

To calculate the mean, sum all the individual data points ($x_1, x_2, ..., x_n$) and then divide by the total number of data points ($n$).

Median

This is the middle value when the data set is arranged in numerical order.

Mode

The mode is the value that appears most often in the data set. A data set can have one mode (unimodal), multiple modes (multimodal), or no mode if all values appear with the same frequency.

Range

• Formula: `range = maximum value - minimum value`

• Maximum value = $30$

• Minimum value = $12$

• Range = $30 - 12 = 18$

Histograms (1/2)

• It looks like a bar graph, but the bars touch each other to signify that the data is continuous and grouped into intervals (often called bins).

• The horizontal axis represents the data intervals.

• The vertical axis represents the frequency (count) of data points falling into each interval.

• The height of each bar corresponds to the frequency for that specific bin.

• $60-69$: 3 students

• $70-79$: 8 students

Identifying Outliers

• More than `1.5 IQR` below the first quartile (Q1).

• More than `1.5 IQR` above the third quartile (Q3).

• Is $42 > 40$? Yes. Therefore, $42$ is an outlier.

Key Concepts (1/2)

What are the three basic measures of center reviewed in this section?

Mean, median, and mode.

Define the mean of a data set and provide its formula.

The sum of all data values divided by the number of values: $x = (x_1 + x_2 + ... + x_n) / n$.

How is the median of a data set found, especially when there's an even number of data points?

The median is the middle value when the data are ordered. If there's an even number of data points, it's the average of the two middle values.

What is the mode of a data set?

The value that appears most often in the data set.

Define the range of a data set and provide its formula.

The difference between the greatest and least values in the data set: `range = maximum - minimum`.

Calculate the mean, median, mode, and range for the data set: 12, 15, 18, 15, 22, 30, 15.

Ordered data: 12, 15, 15, 15, 18, 22, 30. Mean = 127/7 ≈ 18.1. Median = 15. Mode = 15. Range = 30 - 12 = 18.

In a histogram, what does the height of each bar represent?

The frequency (count) of data values that fall within that specific interval or bin.

List the five components of a data set's five-number summary.

Minimum (least value), First Quartile (Q1), Median (Q2), Third Quartile (Q3), Maximum (greatest value).

What is the first quartile (Q1)?

The median of the lower half of an ordered data set.

What is the third quartile (Q3)?

The median of the upper half of an ordered data set.
