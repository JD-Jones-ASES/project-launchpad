Important Terms (1/2)

Function Notation

A symbolic way to write functions, typically `f(x)`, where `f` is the name of the function and `x` is the input variable.

Input

The value or expression that is substituted into a function (represented by `x` in `f(x)`).

Output

The result obtained after evaluating a function with a given input (represented by `f(x)`).

Evaluate

To find the value of a function for a specific input by substituting the input into the function's rule and simplifying.

Domain

The set of all possible input values for which a function is defined (i.e., produces a real number output without undefined operations).

Important Terms (2/2)

Radicand

The expression or number under a radical sign (e.g., the `x` in `sqrt(x)`). For real numbers, the radicand of an even-indexed root must be non-negative.

Denominator

The bottom part of a fraction. The denominator of a fraction cannot be zero.

Introduction to Function Notation

• The `f` is the name given to the function. We can use other letters like `g`, `h`, etc.

• The `x` inside the parentheses `(x)` explicitly tells us that `x` is the input variable for this function.

• `f(x)` is read as "f of x" and represents the output of the function when `x` is the input. It does not mean "f times x."

• This notation makes it very clear what the input is and what the resulting output represents.

What is function notation, and what is its primary benefit?

How is the expression $f(x)$ read?

$f$ of $x$

Does $f(x)$ mean '$f$ times $x$'?

No, $f(x)$ represents the output of the function $f$ when the input is $x$.

How do you evaluate $f(a)$ for a given function $f(x)$?

To evaluate $f(a)$, replace every $x$ in the function's rule with $a$ and then simplify the expression.

Let $f(x) = 3x - 2$. Find $f(5)$.

$f(5) = 3(5) - 2 = 15 - 2 = 13$

Let $f(x) = 3x - 2$. Find $f(-1)$.

$f(-1) = 3(-1) - 2 = -3 - 2 = -5$

Key Concepts (2/2)

Let $f(x) = 3x - 2$. Find $f(0)$.

$f(0) = 3(0) - 2 = 0 - 2 = -2$

Let $g(x) = x^2 + 4x - 1$. Find $g(3)$.

$g(3) = (3)^2 + 4(3) - 1 = 9 + 12 - 1 = 20$

Let $g(x) = x^2 + 4x - 1$. Find $g(-2)$.

$g(-2) = (-2)^2 + 4(-2) - 1 = 4 - 8 - 1 = -5$

Let $h(x) = -2x + 7$. Find $h(4)$.

$h(4) = -2(4) + 7 = -8 + 7 = -1$

How do you evaluate a function when the input is an expression, such as $f(a+1)$?

Replace every instance of the variable (e.g., $x$) in the function's rule with the entire expression (e.g., $a+1$) and then simplify.

Let $f(x) = 2x + 5$. Find $f(a + 1)$.

$f(a+1) = 2(a+1) + 5 = 2a + 2 + 5 = 2a + 7$
