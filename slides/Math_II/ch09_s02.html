Study Slides

Key Concepts

Function Notation ($f(x)$)

A formal way to write functions, replacing the dependent variable $y$. It explicitly names the function (e.g., $f$) and its independent input variable (e.g., $x$), making it easier to read and eval...

Evaluating Functions

The process of substituting a specific value or expression for the input variable $x$ in a function's rule to find the corresponding output.

Evaluating with Expressions

When the input is an algebraic expression (e.g., $a+1$), you substitute the entire expression for $x$ and then simplify the resulting algebraic expression.

Working with Multiple Functions

Problems may involve evaluating and combining the outputs of several different functions, each evaluated independently.

Domain Restrictions

Limitations on the set of possible input values ($x$) for a function due to operations that are undefined in real numbers.

Important Terms

Function Notation

The mathematical convention of writing a function as $f(x)$ (read "$f$ of $x$"), where $f$ is the name of the function and $x$ is the input variable.

Input

The value or expression substituted into a function for the variable $x$.

Output

The result obtained after evaluating a function with a given input. In function notation, this is $f(x)$.

Domain

The complete set of all possible input values for which a function is defined and produces a real number output.

Radicand

The expression or number under a radical symbol (e.g., the $x+2$ in $\sqrt{x+2}$).

Understanding Function Notation

Purpose

Reading $f(x)$

It is read as "$f$ of $x$." It is crucial to remember that **$f(x)$ does NOT mean "$f$ times $x$."** It represents the output value of the function $f$ when $x$ is the input.
