Zero and Negative Exponents

Study Slides

Key Concepts

Zero Exponent Rule

Any nonzero base raised to the power of zero equals 1. This rule arises from simplifying a quantity divided by itself using the quotient rule (e.g., $a^m / a^m = a^{m-m} = a^0$, and $a^m / a^m = 1$).

Power of a Quotient Rule

When a fraction (quotient) is raised to a power, both the numerator and the denominator are raised to that power: $(a/b)^n = a^n/b^n$.

Negative Exponent with Fractions (Shortcut)

A negative exponent applied to a fraction means "flip" the fraction (take its reciprocal) and then raise it to the positive exponent: $(a/b)^{-n} = (b/a)^n$.

Combined Exponent Rules

All exponent rules (product, quotient, power to a power, and now zero/negative exponents and power of a quotient) can be applied together to simplify complex expressions.

Important Terms

Zero Exponent

An exponent of 0. For any nonzero number $a$, $a^0 = 1$.

Negative Exponent

An exponent that is a negative integer. For any nonzero number $a$ and positive integer $n$, $a^{-n} = 1/a^n$.

Base

The number or variable being multiplied by itself in an exponential expression. In $a^n$, $a$ is the base.

Reciprocal

For a number $x$, its reciprocal is $1/x$. Negative exponents are directly related to finding the reciprocal of the base.

Quotient Rule (revisited)

A previously learned rule, $a^m / a^n = a^{m-n}$, which is now expanded to include cases where $m \le n$ due to the introduction of zero and negative exponents.

The Zero Exponent

Derivation

We know that any nonzero quantity divided by itself equals 1. For example, $a^4/a^4 = 1$.

Using the Quotient Rule

If we apply the quotient rule to $a^4/a^4$, we get $a^{4-4} = a^0$.

Conclusion

Since both results must be equal, we conclude that $a^0 = 1$.

Combining these, we see that $a^{-3} = 1/a^3$.

Power of a Quotient

• $(a/b)^n = a^n/b^n$

• $(3/4)^3 = 3^3/4^3 = 27/64$

• $(x^2/y)^4 = (x^2)^4/y^4 = x^8/y^4$

• $(2/a)^{-3}$: This can be simplified by applying the negative exponent rule directly to the fraction: $(2/a)^{-3} = 1/(2/a)^3 = 1/(2^3/a^3) = 1/(8/a^3) = a^3/8$.

• Using this shortcut for $(2/a)^{-3}$: $(2/a)^{-3} = (a/2)^3 = a^3/2^3 = a^3/8$.

Key Concepts (1/2)

State the Zero Exponent Rule.

For any nonzero number $a$, $a^0 = 1$.

What is the condition on the base for the Zero Exponent Rule ($a^0=1$) to be valid?

The base $a$ must be a nonzero number.

Evaluate $7^0$.

$1$

Evaluate $(-4)^0$.

$1$. The base is $-4$, which is nonzero.

Evaluate $-4^0$ and explain the order of operations.

$-4^0 = -(4^0) = -(1) = -1$. The exponent applies only to the base immediately preceding it, which is $4$, not $-4$.

Evaluate $(3x)^0$ given that $x eq 0$.

$1$. The entire product $3x$ is the base, and it is nonzero.

Key Concepts (2/2)

What does a negative exponent indicate about the base number?

A negative exponent indicates taking the reciprocal of the base raised to the positive equivalent of that exponent.

State the rule for a negative exponent $a^{-n}$.

For any nonzero number $a$ and positive integer $n$, $a^{-n} = \frac{1}{a^n}$.

State the rule for a reciprocal with a negative exponent, $\frac{1}{a^{-n}}$.

For any nonzero number $a$ and positive integer $n$, $\frac{1}{a^{-n}} = a^n$.

What are the conditions on the base for negative exponent rules to be valid?

The base $a$ must be a nonzero number.

Rewrite $5^{-2}$ using a positive exponent and evaluate.

$5^{-2} = \frac{1}{5^2} = \frac{1}{25}$.

Rewrite $x^{-4}$ using a positive exponent.

$x^{-4} = \frac{1}{x^4}$.
