Important Terms

Sequence

An ordered list of numbers.

Term

Each individual number within a sequence.

Arithmetic Sequence

A sequence where the difference between consecutive terms is constant.

Common Difference ($d$)

The constant value obtained by subtracting any term from the term that immediately follows it in an arithmetic sequence ($d = a_{n+1} - a_n$).

Explicit Formula

A mathematical formula that directly calculates the value of the $n$th term of a sequence without needing to know the previous terms. For an arithmetic sequence, it is $a_n = a_1 + (n - 1)d$.

The Explicit Formula for an Arithmetic Sequence

• $a_n$ is the $n$th term you want to find.

• $a_1$ is the first term of the sequence.

• $n$ is the term number (e.g., for the 5th term, $n=5$).

• $d$ is the common difference.

What is a sequence in mathematics?

A sequence is an ordered list of numbers.

What is each number in a sequence called?

Each number in a sequence is called a term.

Define an 'arithmetic sequence'.

An arithmetic sequence is a sequence in which the difference between consecutive terms is constant.

What is the 'common difference' in an arithmetic sequence?

The common difference, denoted by $d$, is the constant difference between consecutive terms in an arithmetic sequence.

How do you calculate the common difference $d$ from two consecutive terms $a_n$ and $a_{n+1}$?

The common difference $d$ is calculated as $d = a_{n+1} - a_n$.

Is the sequence $3, 7, 11, 15, 19, \dots$ arithmetic? If so, state its common difference.

Yes, it is an arithmetic sequence. The common difference $d = 4$ ($7-3=4$, $11-7=4$, etc.).

State the explicit formula for the $n$th term of an arithmetic sequence.

The explicit formula for the $n$th term ($a_n$) of an arithmetic sequence with first term $a_1$ and common difference $d$ is $a_n = a_1 + (n - 1)d$.

In the explicit formula $a_n = a_1 + (n - 1)d$, what does the variable $n$ represent?

$n$ represents the term number or the position of the term in the sequence.

An arithmetic sequence has $a_1 = 5$ and $d = 3$. Find the 10th term ($a_{10}$).

Using the formula $a_n = a_1 + (n - 1)d$, $a_{10} = 5 + (10 - 1)(3) = 5 + 9(3) = 5 + 27 = 32$.

The sequence $12, 8, 4, 0, -4, \dots$ is arithmetic. Write its explicit formula for $a_n$.

First, find $a_1 = 12$ and $d = 8 - 12 = -4$. Then, substitute into $a_n = a_1 + (n - 1)d$: $a_n = 12 + (n - 1)(-4) = 12 - 4n + 4 = 16 - 4n$.
