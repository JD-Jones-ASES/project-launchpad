Important Terms (1/2)

Relation

A set of ordered pairs $(x, y)$.

Domain

The set of all $x$-values (inputs) in a relation.

Range

The set of all $y$-values (outputs) in a relation.

Function

A relation in which every input ($x$-value) is paired with exactly one output ($y$-value).

Input

The $x$-value in an ordered pair, representing an independent variable.

Important Terms (2/2)

Output

The $y$-value in an ordered pair, representing a dependent variable that depends on the input.

Vertical Line Test

A visual test applied to the graph of a relation: if no vertical line intersects the graph at more than one point, the relation is a function.

Understanding Relations

Domain and Range

For any relation, the set of all $x$-values is called the domain, and the set of all $y$-values is called the range. These sets define the scope of the relation.

Key Concepts (1/2)

What is a relation in mathematics?

A relation is any set of ordered pairs (x, y).

In a relation, what is defined as the domain?

The domain is the set of all x-values (inputs).

In a relation, what is defined as the range?

The range is the set of all y-values (outputs).

What is a function, and how does it differ from a general relation?

Is the set of ordered pairs {(1, 3), (2, 5), (3, 7), (4, 9)} a function? Explain why.

Yes, it is a function because each x-value appears exactly once.

Is the set of ordered pairs {(1, 4), (2, 5), (1, 6)} a function? Explain why.

No, it is not a function because the input x = 1 is paired with two different outputs (4 and 6).

Is the set of ordered pairs {(-2, 3), (0, 3), (4, 3)} a function? Explain why.

Yes, it is a function. Each x-value appears only once, even though different x-values share the same y-value.

What is the purpose of the Vertical Line Test?

The Vertical Line Test is used to determine if a graph represents a function.

State the rule for the Vertical Line Test.

A graph represents a function if and only if no vertical line intersects the graph at more than one point.

If a relation is given by the graph of a circle, is it a function? Justify your answer using the Vertical Line Test.

No. A vertical line can intersect a circle at two points (e.g., passing through the center), violating the Vertical Line Test.

A table shows inputs x = {-1, 0, 1, 2} and corresponding outputs y = {2, 5, 8, 11}. Does this table define a function? What are its domain and range?

Yes, it defines a function because every x-value appears exactly once. Domain: {-1, 0, 1, 2}, Range: {2, 5, 8, 11}.
