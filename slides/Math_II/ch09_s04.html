Important Terms (1/2)

Probability of an Event

A quantitative measure (a number between 0 and 1) expressing the likelihood of an event occurring.

Simple Event

An event that consists of a single outcome in a probability experiment (e.g., drawing a single card, rolling one die).

Compound Event

An event that combines two or more simple events (e.g., drawing two cards, flipping a coin and rolling a die).

Favorable Outcomes

The specific outcomes of an experiment that satisfy the conditions of the event being considered.

Total Outcomes

All possible outcomes that can result from a probability experiment.

Important Terms (2/2)

Complementary Event

The event that an original event does not happen. If $E$ is an event, its complement is denoted as "not $E$."

Independent Events

Events where the result of one event does not affect the probability or outcome of another event.
