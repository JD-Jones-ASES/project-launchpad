Key Concepts (1/2)

What fundamental geometric theorem is the Distance Formula based on?

The Pythagorean Theorem.

State the Distance Formula for two points $(x_1, y_1)$ and $(x_2, y_2)$.

$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$

When deriving the Distance Formula, how is the horizontal leg of the right triangle expressed using coordinates $P_1(x_1, y_1)$ and $P_2(x_2, y_2)$?

Horizontal leg $= |x_2 - x_1|$.

When deriving the Distance Formula, how is the vertical leg of the right triangle expressed using coordinates $P_1(x_1, y_1)$ and $P_2(x_2, y_2)$?

Vertical leg $= |y_2 - y_1|$.

Why does the specific order of the points (which one is $(x_1, y_1)$ and which is $(x_2, y_2)$) not affect the final distance calculated by the Distance Formula?

Because the differences $(x_2 - x_1)$ and $(y_2 - y_1)$ are squared, which removes any negative signs, making $(a-b)^2 = (b-a)^2$.

Find the distance between the points $A(1, 2)$ and $B(7, 10)$.

$d = \sqrt{(7-1)^2 + (10-2)^2} = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10$.

Key Concepts (2/2)

Calculate the distance between $C(-3, 4)$ and $D(5, -2)$.

$d = \sqrt{(5 - (-3))^2 + (-2 - 4)^2} = \sqrt{8^2 + (-6)^2} = \sqrt{64 + 36} = \sqrt{100} = 10$.

What is the distance between $(2, -1)$ and $(-4, 7)$?

$d = \sqrt{(-4 - 2)^2 + (7 - (-1))^2} = \sqrt{(-6)^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10$.

Find the distance between $(1, 3)$ and $(4, 8)$ and express your answer in simplified radical form.

$d = \sqrt{(4-1)^2 + (8-3)^2} = \sqrt{3^2 + 5^2} = \sqrt{9 + 25} = \sqrt{34}$. Since 34 has no perfect-square factors, it is already simplified.

What is a common error students make when using the Distance Formula regarding the differences in coordinates?

Forgetting to square the differences $(x_2 - x_1)$ and $(y_2 - y_1)$ before adding them, instead incorrectly computing $(x_2 - x_1) + (y_2 - y_1)$.
