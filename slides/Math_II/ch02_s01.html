Chapter 2, Section 1: Rational Numbers on the Number Line Study Guide

Study Slides

Key Concepts (1/2)

Definition of a Rational Number

A number that can be written in the form $a/b$, where $a$ and $b$ are integers and $b$ is not equal to zero.

Integers as Rational Numbers

Every integer $n$ is a rational number because it can be written as $n/1$.

Terminating Decimals

A decimal representation of a rational number that has a finite number of decimal places (it ends).

Repeating Decimals

A decimal representation of a rational number that has a block of one or more digits that repeats forever, often indicated by a bar over the repeating block.

Decimal Expansion Property

A number is rational if and only if its decimal expansion either terminates or repeats.

Key Concepts (2/2)

Plotting Rational Numbers

Every rational number corresponds to a unique point on the number line, and fractions can be plotted by dividing unit intervals into equal parts.

Density Property of Rational Numbers

Between any two distinct rational numbers, there is always another rational number, and in fact, infinitely many rational numbers.

Important Terms

Rational Number

Any number that can be expressed as a fraction $a/b$, where $a$ and $b$ are integers and $b \neq 0$. The set of rational numbers is denoted by $Q$.

Terminating Decimal

A decimal number that has a finite number of digits after the decimal point; it eventually comes to an end. Examples include $0.25$ or $0.375$.

Decimals and Rational Numbers

$0.75$

This is a terminating decimal. Since it ends, it is rational. It can be written as $3/4$.

$0.\overline{142857}$

This is a repeating decimal (the block "142857" repeats). Since it repeats, it is rational. It can be written as $1/7$.

$0.010010001...$ (the number of zeros increases each time)

This decimal neither terminates nor repeats (the pattern of zeros changes each time). Therefore, it is *not* a rational number.

Plotting Rational Numbers on the Number Line

Method

Divide the appropriate unit interval (the space between two consecutive integers) into equal parts corresponding to the denominator of the fraction.

Example

Plot $-3/2$, $2/5$, $1.75$ on the number line.
