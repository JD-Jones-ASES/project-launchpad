Solving One-Step and Two-Step Inequalities Study Guide

Study Slides

Key Concepts (1/2)

Inverse Operations for Isolation

To solve an inequality, you isolate the variable by using inverse operations, just as you would when solving an equation.

Addition and Subtraction Properties of Inequality

Adding or subtracting the same number on both sides of an inequality does not change the direction of the inequality symbol.

Multiplication and Division Properties of Inequality

Common Error

Forgetting to reverse the inequality sign when multiplying or dividing by a negative number is the most frequent mistake.

Order of Operations for Two-Step Inequalities

When solving two-step inequalities, first undo any addition or subtraction, then undo any multiplication or division.

Key Concepts (2/2)

Checking Solutions

To verify a solution, choose a value that satisfies your final inequality and substitute it into the original inequality to ensure it makes the statement true.

Important Terms

Inequality

A mathematical statement that compares two expressions using an inequality symbol (e.g., `<`, `>`, `≤`, `≥`).

Inverse Operations

Operations that undo each other, such as addition and subtraction, or multiplication and division. They are used to isolate a variable.

Isolate the Variable

The process of manipulating an equation or inequality to get the variable by itself on one side of the symbol.

Solution to an Inequality

The set of all numbers that make the inequality a true statement. This is often represented as an interval or by shading on a number line.

Key Concepts (1/2)

What is the primary goal when solving an inequality?

To isolate the variable using inverse operations.

What is the critical difference between solving an equation and solving an inequality?

You must reverse the inequality symbol when multiplying or dividing both sides by a negative number.

State the Addition Property of Inequality.

Adding the same number to both sides of an inequality does not change the direction of the inequality symbol. (e.g., If a < b, then a + c < b + c).

State the Subtraction Property of Inequality.

Subtracting the same number from both sides of an inequality does not change the direction of the inequality symbol. (e.g., If a < b, then a - c < b - c).

How does multiplying both sides of an inequality by a positive number affect the inequality symbol?

It does not change the direction of the inequality symbol. (e.g., If a < b and c > 0, then ac < bc).

How does dividing both sides of an inequality by a positive number affect the inequality symbol?

It does not change the direction of the inequality symbol.

Key Concepts (2/2)

What happens to the inequality symbol when multiplying both sides by a negative number?

The direction of the inequality symbol must be reversed. (e.g., If a < b and c < 0, then ac > bc).

What happens to the inequality symbol when dividing both sides by a negative number?

The direction of the inequality symbol must be reversed.

What is the most common mistake students make when solving inequalities?

Forgetting to reverse the inequality sign when multiplying or dividing by a negative number.

Solve the one-step inequality: $x - 4 > 7$.

Add 4 to both sides: $x > 11$.

When solving an inequality like $-3x \le 18$, what critical step must be performed due to the negative coefficient?

When dividing both sides by -3, the inequality sign must be reversed.

Solve the one-step inequality: $-3x \le 18$.

Divide by -3 and reverse the sign: $x \ge -6$.
