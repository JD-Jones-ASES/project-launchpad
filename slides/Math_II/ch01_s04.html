Key Concepts (1/2)

Inverse Operations

Exponents (squaring, cubing) and roots (square roots, cube roots) are inverse operations, meaning one "undoes" the other.

Square Root

Cube Root

For any number $a$, its cube root is a number $b$ such that $b^3 = a$. It is written as $\sqrt[3]{a}$.

Key Concepts (2/2)

Domain of Cube Roots

Unlike square roots, cube roots are defined for all real numbers, including negative numbers.

Estimating Non-Perfect Roots

For numbers that are not perfect squares or cubes, their roots can be estimated by finding the two consecutive whole numbers between which the root lies.

Product Property of Square Roots

For nonnegative numbers $a$ and $b$, the square root of their product can be split into the product of their square roots: $\sqrt{ab} = \sqrt{a} \cdot \sqrt{b}$.

Simplifying Square Roots

To simplify a square root, identify and factor out the largest perfect square from the radicand, then apply the product property.

Important Terms (1/2)

Square Root

A number that, when multiplied by itself, gives a specific nonnegative number. For a number $a$, the square root is $b$ such that $b^2 = a$.

Radical Sign

The symbol $\sqrt{\quad}$, used to indicate a root. For square roots, the small "2" (index) is usually omitted.

Radicand

The number or expression written under the radical sign.

Principal Square Root

The unique nonnegative square root of a nonnegative number. For example, $\sqrt{25} = 5$.

Cube Root

A number that, when multiplied by itself three times, gives a specific number. For a number $a$, the cube root is $b$ such that $b^3 = a$. It is written as $\sqrt[3]{a}$.

Important Terms (2/2)

Perfect Square

An integer that is the square of another integer (e.g., $1, 4, 9, 16, 25, \dots$).

Perfect Cube

An integer that is the cube of another integer (e.g., $1, 8, 27, 64, 125, \dots$, and also negative cubes like $-1, -8, -27, \dots$).

Introduction to Roots as Inverse Operations

Squaring

a number means multiplying it by itself (e.g., $7^2 = 7 \times 7 = 49$).

Finding a square root

asks, "What number, when multiplied by itself, gives this result?" (e.g., $\sqrt{49}$ asks "What number squared equals 49?").

Definition and Notation

• We write this as $\sqrt{a} = b$.

• The symbol $\sqrt{\quad}$ is called the radical sign.

• The number under the radical sign, $a$, is called the radicand.

Principal Square Root

• For example, both $5$ and $-5$ are square roots of $25$ because $5^2 = 25$ and $(-5)^2 = 25$.

• The symbol $\sqrt{a}$ always denotes the principal square root, which is the nonnegative root. So, $\sqrt{25} = 5$.

• To indicate the negative square root, a negative sign is placed outside the radical, like $-\sqrt{25} = -5$.

Evaluating Square Roots

• $\sqrt{49} = 7$ because $7^2 = 49$.

• $-\sqrt{81} = -9$ because $\sqrt{81} = 9$.

• $\sqrt{0} = 0$ because $0^2 = 0$.

• $\sqrt{144} = 12$ because $12^2 = 144$.

Domain Restriction for Real Numbers

• There is no real number whose square is negative. For instance, $\sqrt{-4}$ has no real value, because no real number multiplied by itself gives $-4$.

Definition and Notation

• We write this as $\sqrt[3]{a} = b$. The small "3" is called the index of the root.

Key Difference: Negative Numbers

• For example, $\sqrt[3]{-8} = -2$ because $(-2)^3 = (-2) \times (-2) \times (-2) = 4 \times (-2) = -8$.

• This is possible because an odd number of negative factors results in a negative product.

Evaluating Cube Roots

• $\sqrt[3]{27} = 3$ because $3^3 = 27$.

• $\sqrt[3]{-64} = -4$ because $(-4)^3 = -64$.

• $\sqrt[3]{125} = 5$ because $5^3 = 125$.

Important Caution: The Product Property does NOT apply to addition!

• For instance, $\sqrt{9 + 16} = \sqrt{25} = 5$.

• But, $\sqrt{9} + \sqrt{16} = 3 + 4 = 7$.

• Since $5 e 7$, the sum property does not hold. Only the product property works for multiplication.

What is the definition of a square root of a nonnegative number $a$?

The square root of a nonnegative number $a$ is the nonnegative number $b$ such that $b^2 = a$. It is written as $a = b$.

In the expression $x$, what are the names for the symbol $x$ and the number $x$?

The symbol $x$ is called a radical sign, and the number $x$ under it is called the radicand.

What is the principal square root of $25$?

$5$, because $5^2 = 25$ and $5$ is nonnegative.

How would you write and evaluate the negative square root of $81$?

The negative square root of $81$ is written as $-81$, which evaluates to $-9$.

For what kind of numbers $a$ is the expression $a$ defined in the set of real numbers?

$a$ is defined only when $a \ge 0$. There is no real number whose square is negative.

What is the definition of the cube root of a number $a$?

The cube root of a number $a$ is the number $b$ such that $b^3 = a$. It is written as $[3]a = b$.

How does the domain of cube roots differ from that of square roots for real numbers?

Square roots are defined only for nonnegative numbers, while cube roots are defined for all real numbers (positive, negative, and zero).
