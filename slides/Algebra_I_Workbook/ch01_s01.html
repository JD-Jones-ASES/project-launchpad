The Real Number System

Absolute Value and Opposites

Opposites

Two numbers that are the same distance from zero but on opposite sides (e.g., the opposite of $-8$ is $8$).

Mathematical Properties

Commutative Property

The order of numbers in addition or multiplication does not change the result ($a + b = b + a$; $ab = ba$).

Associative Property

The grouping of numbers in addition or multiplication does not change the result ($(a + b) + c = a + (b + c)$).

Glossary of Important Terms (1/3)

Absolute Value

The non-negative distance of a number from zero on the number line.

Additive Inverse

The opposite of a number; the value that, when added to the original number, results in zero.

Algebraic Expression

A mathematical phrase that can contain ordinary numbers, variables, and operators.

Associative Property

A rule stating that the way in which numbers are grouped in addition or multiplication does not change their sum or product.

Commutative Property

A rule stating that the order in which numbers are added or multiplied does not change the result.

Glossary of Important Terms (2/3)

Distributive Property

A property that allows one to multiply a sum by multiplying each addend separately and then adding the results.

Integer

The set of whole numbers and their negative opposites.

Irrational Number

A real number that cannot be expressed as a ratio of two integers; its decimal form is non-terminating and non-repeating.

Multiplicative Inverse (Reciprocal)

A number that, when multiplied by the original number, results in one.

Natural Numbers

The set of positive integers used for counting ($1, 2, 3, \dots$).

Glossary of Important Terms (3/3)

Rational Number

Any number that can be expressed as the quotient $\frac{a}{b}$ of two integers, where $b$ is not zero.

Real Number

The value that represents any quantity along a continuous number line, encompassing both rational and irrational numbers.

Variable

A symbol, usually a letter, used to represent one or more numbers in a mathematical expression.

Whole Numbers

The set of natural numbers plus zero.
