Notations for Solutions

Inequality Notation

Direct use of symbols (e.g., $x > 5$).

Interval Notation

Uses parentheses `()` for excluded values (open circles) and brackets `[]` for included values (closed circles). Infinity ($\infty$) always uses parentheses.

Set-Builder Notation

A formal way to describe a set (e.g., $\{x \mid x < 7\}$).

Graphing on a Number Line

Open Circle

Used for $>$ or $<$ symbols. It indicates the boundary value is not part of the solution.

Closed Circle

Used for $\geq$ or $\leq$ symbols. It indicates the boundary value is included.

Direction

An arrow pointing right indicates "greater than"; an arrow pointing left indicates "less than."

Line Segments

A solid line connecting two circles represents a compound inequality where the solution lies between two values.

Solving Linear Inequalities

The Reversal Rule

You must reverse the inequality sign whenever you multiply or divide both sides by a negative number.

Standard Steps

Compound Inequalities

AND Inequalities

The solution must satisfy both conditions. These often describe a range (e.g., $-5 < x + 3 < 9$). In interval notation, this is a single interval.

OR Inequalities

The solution satisfies at least one of the conditions. These are expressed as a union ($\cup$) of two intervals (e.g., $x > 7$ or $x < -4$).

Glossary of Important Terms (1/2)

Absolute Value

The distance of a number from zero on a number line, always expressed as a non-negative value.

At Least

Mathematical equivalent of "greater than or equal to" ($\geq$).

At Most

Mathematical equivalent of "less than or equal to" ($\leq$).

Boundary Lengths/Values

The specific points at which a condition changes from true to false; often found in quality control or tolerance problems.

Compound Inequality

Two inequalities joined by "and" or "or."

Glossary of Important Terms (2/2)

Interval Notation

A method of writing solution sets using parentheses and brackets to show the start and end points of the set.

No Solution

A result that occurs when an absolute value is set equal to a negative number or when an inequality has no values that satisfy its conditions.

Set-Builder Notation

A shorthand used to describe a set by stating the properties that its members must satisfy (e.g., $\{x \mid x > 0\}$).

Union ($\cup$)

A symbol used in interval notation to combine two separate solution sets, typically used in "OR" compound inequalities.

Key Concepts (1/2)

How is 'all real numbers greater than $7$' written as an inequality?

$x > 7$

How is 'all real numbers at least $12$' expressed as an inequality?

$x \ge 12$

How is 'a number $n$ is no more than $25$' written as an inequality?

$n \le 25$

What inequality represents all real numbers strictly between $-4$ and $6$, including both endpoints?

$-4 \le x \le 6$

What inequality is represented by an open circle at $5$ with an arrow pointing to the right on a number line?

$x > 5$

What inequality is represented by a closed circle at $-2$ with an arrow pointing to the left on a number line?

$x \le -2$

Key Concepts (2/2)

What inequality is represented by a closed circle at $0$ and a closed circle at $8$ connected by a solid line?

$0 \le x \le 8$

What inequality is represented by an open circle at $-6$ and a closed circle at $1$ connected by a solid line?

$-6 < x \le 1$

Describe the graph of $x > -3$ on a number line.

An open circle at $-3$ with an arrow pointing to the right.

Describe the graph of $-1 \le x < 5$ on a number line.

A closed circle at $-1$ and an open circle at $5$ connected by a solid line.

Why is $x = 4$ not a solution to $2x - 3 > 5$?

Substituting $4$ results in $5$, and $5$ is not strictly greater than $5$.

Write $x < 7$ in set-builder notation.

$\{x \mid x < 7\}$

Quiz Question 2

B. $-3 < x \le 4$

Quiz Question 3

B. $x = -7$ and $x = -1$

C. $x = -11$ and $x = 3$

D. $x = -3$

Hint: Isolate the absolute value expression first and consider if the resulting value is possible.

Quiz Question 4

Which interval notation correctly represents the solution to $|x - 5| > 3$?

B. $(2, 8)$

C. $(-\infty, 2] \cup [8, \infty)$

Hint: Think about whether this inequality describes values that are 'far from' or 'close to' the center point.

Quiz Question 5

A. $x \ge 95$

B. $x \ge 85$

C. $x \ge 80$

D. $x \ge 90$

Hint: The sum of all three scores divided by three must be greater than or equal to the target average.
