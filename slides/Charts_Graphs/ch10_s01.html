Chapter 10, Section 1: Histograms

Study Slides

Key Concepts (1/2)

Histograms vs. Bar Charts

Histograms display continuous numerical data grouped into intervals (bins), while bar charts display categorical (discrete) data.

Continuous Data

Data that can take any value within a range (e.g., heights, temperatures). It's the type of data histograms are designed for.

No Gaps Between Bars

A defining feature of histograms. Since continuous data has no natural breaks, the bars representing consecutive intervals touch, signifying the flow of data across the range.

Bins (Intervals)

Equal-width ranges into which continuous data is divided. Each bar on a histogram represents the frequency of data values falling within a specific bin.

Distribution

Describes how data values are spread out, showing where data clusters, is sparse, and its overall pattern. Histograms visually represent this distribution.

Key Concepts (2/2)

Shapes of Distribution

Reading a Histogram

Involves understanding axis labels, identifying bin intervals, reading bar heights (frequencies), describing the distribution's shape, and estimating the center and spread of the data.

The Critical Difference: No Gaps!

Continuous data

can take any value within a given range (e.g., 164 cm, 164.5 cm, 164.9 cm). There are no natural breaks between one value and the next in a continuous set.
