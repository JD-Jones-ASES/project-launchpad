Key Concepts (1/3)

Data Illusions

Charts can create "optical illusions" using data, where the visual design misrepresents the actual information, even if the numbers themselves are correct.

Critical Chart Evaluation

Understanding these techniques allows viewers to "see through" manipulative designs and interpret data accurately, rather than being swayed by distorted visuals.

Truncated Y-Axis

Exaggerating small differences by starting the value axis (y-axis) at a value other than zero.

3D Distortion

Using three-dimensional effects in charts (especially pie and bar charts) to make elements in the foreground appear larger than identical elements in the background due to perspective.

Cherry-Picked Time Range

Selectively choosing a specific timeframe in a line graph to support a particular narrative while omitting contradictory historical data.

Key Concepts (2/3)

Inconsistent Intervals

Using uneven spacing for ticks on the x-axis (especially in time-series charts) to visually distort the rate of change represented by lines.

Area and Volume Distortion

Misrepresenting data values by scaling icons or bubbles by height or radius instead of by area, leading to an exaggerated visual size relative to the actual data difference.

Missing Context

Presenting data without crucial comparison points, sources, dates, baselines, or units, making the data's significance impossible to assess properly.

Dual-Axis Deception

Using two different y-axes with independent scales to make unrelated variables appear to have a strong correlation or move together perfectly.

Key Concepts (3/3)

Misleading Percentages

Presenting percentage changes that seem dramatic but are based on small absolute values, or showing percentages that inappropriately add up to more than 100%.

Important Terms (1/2)

Truncated Axis

A numerical axis (typically the y-axis) that does not start at zero, which can visually exaggerate the differences between data points.

Cherry-Picking

The act of intentionally selecting only the data points or a specific range of data that supports a desired conclusion, while omitting other data that might contradict it.

Dual-Axis Chart

A chart that uses two separate y-axes (one on the left and one on the right), often with different scales, which can be manipulated to create a false visual correlation between two data series.

Sequential Color Scale

A color palette used in data visualization that progresses smoothly from light to dark within a single hue, making it easy to perceive a clear order from low to high values.
