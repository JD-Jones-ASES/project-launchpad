Chapter 10, Section 2

Understanding and Interpreting Box Plots

Study Slides

Key Concepts (1/3)

Box Plot (Box-and-Whisker Plot)

A graph that visually summarizes a data set using its five-number summary.

Five-Number Summary

The five key values that describe the distribution of a data set: minimum, first quartile (Q1), median, third quartile (Q3), and maximum.

Minimum

The smallest value in a data set.

First Quartile (Q1)

The value below which 25% of the data falls. It marks the left edge of the box.

Median

The middle value of an ordered data set, separating the lower 50% from the upper 50%. It is represented by a line inside the box.

Key Concepts (2/3)

Third Quartile (Q3)

The value below which 75% of the data falls (or above which 25% of the data falls). It marks the right edge of the box.

Maximum

The largest value in a data set (excluding outliers).

Interquartile Range (IQR)

The difference between the third quartile (Q3) and the first quartile (Q1) ($IQR = Q3 - Q1$). It represents the range of the middle 50% of the data.

Whiskers

Lines extending from the box to the minimum and maximum values, illustrating the spread of the lower and upper 25% of the data (excluding outliers).

Outlier Dots

Individual data points plotted beyond the whiskers, indicating values that are unusually far from the rest of the data.

Key Concepts (3/3)

Limitations

Box plots do not show the number of data points or the detailed shape of the distribution as a histogram does.

The Box

The Whiskers

These are lines that extend from each end of the box out to the minimum and maximum values of the data, *excluding* any outliers. They show the spread of the lowest 25% and highest 25% of the data.

Outlier Dots

Any data points that fall unusually far outside the typical range of the data (beyond the whiskers) are plotted as individual dots. These extreme values are called outliers.

Common Misconceptions to Avoid

Median is always the middle of the box

The median represents the middle of the *data*, not necessarily the visual center of the box. Its position within the box gives clues about the skewness of the middle 50%.

Long whisker means many data points

A long whisker simply means the data extends to an extreme value, covering 25% of the data points, just like a short whisker. It does not imply a higher *density* of points in that region.

Box plots show individual counts

Box plots *do not* show the number of individual data points. This is a key limitation; you cannot tell how many total data points a box plot represents just by looking at it.

Example: Class Test Scores Interpretation

Five Values

Clearly identified from the plot (55, 72, 80, 88, 98).

IQR

$88 - 72 = 16$ points. This means the middle 50% of students scored within a 16-point range (from 72 to 88).

Box Interpretation

The box from 72 to 88 shows the central 50% of scores. The median (80) is roughly in the center of the box, suggesting the scores in this middle range are fairly symmetrically distributed around 80.

Whiskers Interpretation

Outlier

The dot at 42 highlights an unusually low score, significantly below the rest of the class.
