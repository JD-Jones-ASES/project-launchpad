Chapter 9, Section 1: Reading Scatter Plots

Study Slides

Key Concepts (1/2)

Purpose of a Scatter Plot

A scatter plot is used to display two measurements for individual items to see if there's a relationship between the variables.

Individual Data Points

Each dot on a scatter plot represents a single item (e.g., a person, an experiment) and shows its corresponding value for two different variables—one on the x-axis and one on the y-axis.

No Connected Dots

Unlike line graphs, the dots on a scatter plot are not connected. Each dot is an independent observation, and the order of dots does not imply a sequence or change over time.

Identifying Variables and Scales

Understanding the title, axis labels, and scales is crucial for interpreting what the scatter plot is representing.

Directions of Patterns (Trends)

Key Concepts (2/2)

Reading Specific Values

To read the values for a specific dot, trace vertically down to the x-axis and horizontally left to the y-axis.

Outliers

An outlier is a data point that stands apart from the general pattern of the other dots, suggesting an unusual observation.

Important Terms (1/2)

Scatter Plot (or Scatter Graph/Scatter Diagram)

A graph that displays data as a collection of individual dots, where each dot represents an item's values for two different variables.

Variable

A characteristic, number, or quantity that can be measured or counted. In a scatter plot, two variables are typically measured for each item.

x-axis

The horizontal axis on a graph, typically used for the independent variable or the first measurement in a pair.

y-axis

The vertical axis on a graph, typically used for the dependent variable or the second measurement in a pair.

Positive Trend (Upward Pattern)

A pattern in a scatter plot where, as the x-variable increases, the y-variable also tends to increase. The dots rise from the lower-left to the upper-right.

Important Terms (2/2)

Negative Trend (Downward Pattern)

A pattern in a scatter plot where, as the x-variable increases, the y-variable tends to decrease. The dots fall from the upper-left to the lower-right.

No Trend (No Pattern)

A situation in a scatter plot where the dots are randomly scattered, indicating no clear relationship or direction between the two variables.

Outlier

A data point in a scatter plot that lies significantly far away from the other data points, not fitting the general pattern or trend.

What is a Scatter Plot?

• One value is read on the horizontal (x) axis.

• The other value is read on the vertical (y) axis.

Key Difference from a Line Graph

Line Graph

Connects dots to show how one variable changes over time. The order of the dots matters (e.g., Monday, Tuesday, Wednesday).

Scatter Plot

Directions of Patterns Explained

Upward Pattern (Positive Trend)

Think of a staircase going up. If students who study more tend to get higher test scores, that would likely show a positive trend.

Downward Pattern (Negative Trend)

Think of a slide going down. If people who exercise more tend to have lower heart rates, that might show a negative trend.

No Pattern (No Trend)

Imagine throwing confetti in the air; it lands randomly. If the month a student was born has no impact on their favorite color, there would be no trend.

What does each individual dot on a scatter plot represent?

Each dot represents a single item (e.g., a person, a country, a game) and shows the values of two different variables for that specific item.

What is the key difference between a scatter plot and a line graph regarding how their data points are treated?

On a scatter plot, the dots are NOT connected; each dot stands alone. On a line graph, dots are connected to show change over time or sequence.

When is a line graph typically used?

A line graph is used to show how one variable changes over time, where the order of data points is meaningful (e.g., Monday, Tuesday, Wednesday).

What type of relationship does a scatter plot primarily explore?

A scatter plot shows whether two variables are related to each other, indicating a possible association or trend between them.

Imagine a scatter plot with 'Height (inches)' on the x-axis and 'Shoe Size' on the y-axis. What does a dot at position (65, 9) signify?

This dot represents one item (e.g., a student) who is 65 inches tall and wears a size 9 shoe.
