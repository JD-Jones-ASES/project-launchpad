Glossary of Important Terms (1/2)

Axis of Symmetry

A vertical line ($x = \frac{-b}{2a}$) about which a parabola is symmetric.

Domain

The set of all possible input values (all real numbers for quadratic functions).

Parabola

The U-shaped curve that represents the graph of a quadratic function.

Range

The set of all possible output values, determined by the vertex and the direction the parabola opens.

Standard Form

The expression of a quadratic function as $f(x) = ax^2 + bx + c$.

Glossary of Important Terms (2/2)

Vertex

The highest or lowest point of a parabola; the point where the curve changes direction.

Vertex Form

The expression of a quadratic function as $f(x) = a(x - h)^2 + k$, where $(h, k)$ explicitly identifies the vertex.

$x$-intercept

The point(s) where the parabola crosses the $x$-axis, found by solving $f(x) = 0$.

$y$-intercept

The point where the parabola crosses the $y$-axis, found by evaluating $f(0)$.

Key Concepts (1/2)

What is a parabola?

The U-shaped curve that is the graph of a quadratic function $f(x) = ax^2 + bx + c$ (with $a eq 0$). It is symmetric about a vertical line called its axis ...

What is the vertex of a parabola?

The point where the curve changes direction. It is the lowest point if the parabola opens upward, or the highest point if it opens downward.

How does the sign of the leading coefficient $a$ determine the direction a parabola opens?

If $a > 0$, the parabola opens upward. If $a < 0$, the parabola opens downward.

If $a > 0$, is the vertex a minimum or maximum point?

A minimum (because the parabola opens upward).

If $a < 0$, is the vertex a minimum or maximum point?

A maximum (because the parabola opens downward).

What is the formula for the axis of symmetry for a quadratic function $f(x) = ax^2 + bx + c$?

$x = -b/(2a)$.

Key Concepts (2/2)

How do you find the coordinates of the vertex for a quadratic function $f(x) = ax^2 + bx + c$?

The vertex is the point $(-b/(2a), f(-b/(2a)))$. First find the axis of symmetry, then evaluate the function at that x-value.

What is the first step in graphing a quadratic function $f(x) = ax^2 + bx + c$?

Find the axis of symmetry: $x = -b/(2a)$.

What is the third step in graphing a quadratic function $f(x) = ax^2 + bx + c$?

Find the y-intercept by setting $x = 0$ to get $f(0) = c$. The point is $(0, c)$.

How do you find the x-intercepts (if they exist) for a quadratic function $f(x) = ax^2 + bx + c$?

Solve the quadratic equation $ax^2 + bx + c = 0$.

What is the domain of any quadratic function $f(x) = ax^2 + bx + c$?

All real numbers, or $(-\infty, \infty)$.

If $a > 0$, what is the range of the quadratic function $f(x) = ax^2 + bx + c$?

$[y_v, \infty)$, where $y_v$ is the y-coordinate of the vertex.
