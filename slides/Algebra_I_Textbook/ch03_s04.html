Glossary of Important Terms

Key Concepts (1/2)

What is the geometric definition of the absolute value of a number?

The number's distance from zero on the number line.

For any real number $a$, what is the result of $|a|$ if $a \ge 0$?

$a$

For any real number $a$, what is the result of $|a|$ if $a < 0$?

$-a$

According to the definition of distance, what is the minimum possible value for $|a|$?

$0$

When solving the equation $|x| = 5$, what specific distance are you looking for from zero?

A distance of $5$.

How many solutions exist for the equation $|x| = 5$?

Two solutions.

Key Concepts (2/2)

What are the two solutions for the equation $|x| = 5$?

$x = 5$ and $x = -5$

In the property for solving $|ax + b| = c$, what must be true about $c$ for the equation to have two solutions?

$c > 0$

If $c > 0$, what two linear equations are equivalent to $|ax + b| = c$?

$ax + b = c$ and $ax + b = -c$

How many solutions are there for the equation $|ax + b| = 0$?

Exactly one solution.

If $c = 0$, what single linear equation is equivalent to $|ax + b| = c$?

$ax + b = 0$

Under what condition for $c$ does the equation $|ax + b| = c$ have no solution?

$c < 0$
