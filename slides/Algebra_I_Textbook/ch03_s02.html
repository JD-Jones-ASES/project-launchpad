Glossary of Key Terms (1/2)

Combine Like Terms

The process of simplifying an expression by adding or subtracting terms that have the same variable raised to the same power.

Distributive Property

A mathematical property used to multiply a single term by two or more terms inside a set of parentheses (e.g., $a(b + c) = ab + ac$).

Interval Notation

A method of writing down the set of all real numbers between two endpoints using parentheses $()$ or brackets $[]$.

Inverse Operations

Operations that undo each other, such as addition and subtraction or multiplication and division, used to isolate a variable.

Multi-Step Inequality

An inequality that requires more than one operation (such as distributing, combining terms, and using inverse operations) to solve for the variable.

Glossary of Key Terms (2/2)

Reverse (Flip) the Symbol

The requirement to change the direction of an inequality sign (e.g., from $<$ to $>$) specifically when multiplying or dividing both sides by a negative number.

Key Concepts (1/2)

Which operations are used to isolate the variable when solving a multi-step inequality?

Inverse operations.

How does the process of solving an inequality generally compare to solving an equation?

It is very similar, primarily using inverse operations to isolate the variable.

According to the Addition Property of Inequality, if $a < b$, then $a + c <$ ____.

$b + c$

According to the Subtraction Property of Inequality, if $a < b$, then $a - c <$ ____.

$b - c$

True or False: Adding or subtracting the same number on both sides of an inequality changes the direction of the symbol.

False.

Under the Multiplication Property, if $c > 0$ and $a < b$, then $ac$ ____ $bc$.

$<$

Key Concepts (2/2)

Under the Multiplication Property, if $c < 0$ and $a < b$, then $ac$ ____ $bc$.

$>$

What action requires the reversal of the inequality symbol when solving an inequality?

Multiplying or dividing both sides by a negative number.

According to the text, what is one of the most common errors in algebra regarding inequalities?

Forgetting to reverse the inequality symbol when multiplying or dividing by a negative number.

If you divide both sides of $-2x > 6$ by $-2$, what is the resulting inequality?

$x < -3$

What is the first step to solve the inequality $3x - 7 > 5$?

Add $7$ to both sides.

Solve the inequality $3x > 12$ for $x$.

$x > 4$
