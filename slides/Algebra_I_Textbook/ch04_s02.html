Relations and Functions

Study Slides

Understanding Relations

• Lists of ordered pairs

Domain and Range

Domain

The set of all first coordinates, or $x$-values.

Range

The set of all second coordinates, or $y$-values.

Defining a Function

• No $x$-value can be repeated with a different $y$-value.

• One input must result in exactly one output.

• Note: Multiple different $x$-values can point to the same $y$-value (e.g., $(0, 1), (1, 1), (2, 1)$ is a function).

The Vertical Line Test

Rule

A graph is a function if and only if every vertical line intersects the graph at most once.

Failure

If a vertical line crosses the graph at two or more points, a single $x$-value is paired with more than one $y$-value, meaning the relation is not a function.

Function Notation

Meaning

$f(x)$ represents the output of the function $f$ when the input is $x$.

Caution

$f(x)$ does not mean "$f$ times $x$." It is a notation for evaluation, not multiplication.

Glossary of Terms

Key Concepts (1/2)

In mathematics, what is the definition of a 'relation'?

Any set of ordered pairs.

List four common ways a relation may be described.

A list of ordered pairs, a table, a graph, or an equation.

The set of all first coordinates ($x$ -values) in a relation is called the _____.

Domain

The set of all second coordinates ($y$ -values) in a relation is called the _____.

Range

When stating the domain and range from a list of ordered pairs, how should repeated values be handled?

Each value should be listed only once.

State the domain of the relation $(-3, 2), (0, 5), (1, -1), (4, 5)$.

$-3, 0, 1, 4$

Key Concepts (2/2)

State the range of the relation $(-3, 2), (0, 5), (1, -1), (4, 5)$.

$-1, 2, 5$

In the relation $(-3, 2), (0, 5), (1, -1), (4, 5)$, why is the number $5$ only listed once in the range?

The range is a set of unique $y$-values and does not include repeats.

A relation where every element of the domain is paired with exactly one element of the range is called a _____.

Function

What is the requirement for $x$-values in a relation to be classified as a function?

No $x$-value can be repeated with a different $y$-value.

The _____ is a visual method used to determine if a graph in the coordinate plane represents a function.

Vertical line test

According to the vertical line test, a graph is a function if every vertical line intersects the graph at _____ point(s).

At most one
