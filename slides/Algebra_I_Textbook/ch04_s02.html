Chapter 4, Section 2

Relations and Functions

Study Slides

Key Concepts Review (1/2)

Domain

The set of all first coordinates, or $x$-values.

Range

The set of all second coordinates, or $y$-values.

Rule

A graph is a function if and only if every vertical line intersects the graph at most once.

Failure

If a vertical line crosses the graph at two or more points, a single $x$-value is paired with more than one $y$-value, meaning the relation is not a function.

Meaning

$f(x)$ represents the output of the function $f$ when the input is $x$.

Key Concepts Review (2/2)

Caution

$f(x)$ does **not** mean "$f$ times $x$." It is a notation for evaluation, not multiplication.
