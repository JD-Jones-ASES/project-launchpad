Glossary of Key Terms (1/2)

Consistent System

A system of equations that has at least one solution (includes both independent and dependent systems).

Dependent System

A system that has an infinite number of solutions because the equations represent the same line.

Inconsistent System

A system of equations that has no solution because the lines are parallel.

Independent System

A system that has exactly one unique solution, represented by the single point where two lines intersect.

Ordered Pair

A pair of coordinates $(x, y)$ used to identify a point on a coordinate plane and a potential solution to a system.

Glossary of Key Terms (2/2)

Slope-Intercept Form

A way of writing a linear equation (typically $y = mx + b$) that clearly identifies the slope ($m$) and the $y$-intercept ($b$), useful for graphing.

System of Linear Equations

A set of two or more linear equations sharing the same variables.

Key Concepts (1/2)

What is the definition of a system of linear equations?

A set of two or more linear equations that use the same variables.

What is the standard algebraic form of a system with two equations and two variables?

$\begin{cases} a_1 x + b_1 y = c_1 a_2 x + b_2 y = c_2 \end{cases}$

In the system form $\begin{cases} a_1 x + b_1 y = c_1 a_2 x + b_2 y = c_2 \end{cases}$, what do the symbols $a_1, b_1, c_1, a_2, b_2, c_2$ represent?

Real-number constants.

What constitutes a solution for a system of two equations in two variables?

An ordered pair $(x, y)$ that makes both equations true.

Graphically, what represents the solution of a system of two linear equations?

The point where the two lines intersect.

How many solutions does a system have if the lines intersect at exactly one point?

One solution.

Key Concepts (2/2)

How is a system classified if it has exactly one solution?

Consistent and independent.

How many solutions does a system have if the lines are parallel?

No solution.

How is a system classified if it has no solution?

Inconsistent.

How many solutions does a system have if the equations describe the same line?

Infinitely many solutions.

How is a system classified if the lines are the same?

Consistent and dependent.

In a system with no solution, how do the slopes of the lines compare?

The slopes are the same.
