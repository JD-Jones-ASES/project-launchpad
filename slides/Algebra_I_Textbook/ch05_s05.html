Systems of Linear Inequalities

Study Slides

Graphing a System of Inequalities

Identifying the Vertex

The vertex of a solution region is found by calculating the intersection of the boundary lines. This is done by setting the equations of the boundaries equal to each other and solving for $x$ and $y$.

Inclusion of the Vertex

A vertex is only included in the solution region if it lies on the intersection of two solid lines. If one or both of the boundary lines are dashed, the vertex is not included.

Short-Answer Practice Questions (1/2)

Q: What determines whether a boundary line should be drawn as solid or dashed?

A solid line is used when the inequality includes "or equal to" ($\le$ or $\ge$). A dashed line is used for strict inequalities ($<$ or $>$).

Q: How is the solution region of a system defined?

The solution region is the intersection or overlap of the half-planes created by each individual inequality in the system.

Q: If the point (0, 0) is tested in the inequality $y \le 2x + 1$ and the result is true, which region should be shaded?

The half-plane containing the origin $(0, 0)$ should be shaded. In this specific case, that is the region below the solid line $y = 2x + 1$.

Short-Answer Practice Questions (2/2)

Q: When does a system of linear inequalities have no solution?

A system has no solution when there is no overlapping region between the individual inequalities. An example of this is two parallel strips that never intersect.

Q: How do you mathematically determine if a specific ordered pair is a solution to a system?

Substitute the $x$ and $y$ values of the ordered pair into every inequality in the system. If the point makes every inequality true, it is a solution.

Glossary of Important Terms

Key Concepts (1/2)

What is the definition of a solution to a system of linear inequalities?

An ordered pair $(x, y)$ that satisfies every inequality in the system.

What term describes the set of all solutions representing the intersection of half-planes in a system of inequalities?

The solution region.

When graphing a single linear inequality, what is the first step regarding the boundary?

Graph the boundary line defined by the equation version of the inequality.

Which inequality symbols require the use of a solid boundary line?

$\le$ (less than or equal to) or $\ge$ (greater than or equal to).

Which inequality symbols require the use of a dashed boundary line?

$<$ (less than) or $>$ (greater than).

What does a solid boundary line indicate about the points located on that line?

The points are included in the solution set.

Key Concepts (2/2)

What does a dashed boundary line indicate about the points located on that line?

The points are not included in the solution set.

What is the purpose of choosing a test point not on the boundary line?

To determine which half-plane contains the solutions and should be shaded.

Why is the origin $(0, 0)$ frequently used as a test point?

It simplifies the substitution and arithmetic calculations.

Under what condition do you shade the half-plane that contains the chosen test point?

When substituting the test point into the inequality results in a true statement.

Under what condition do you shade the half-plane that does NOT contain the chosen test point?

When substituting the test point into the inequality results in a false statement.

For the inequality $y \le 2x + 1$, what type of line is used for the boundary $y = 2x + 1$?

A solid line.
