Glossary of Key Terms (1/2)

Closed Circle

A mark on a number line graph used to indicate that the boundary value is included in the solution set (used with $\leq$ or $\geq$).

Inequality

A mathematical sentence that compares two expressions using symbols such as $<, >, \leq,$ or $\geq$.

Interval Notation

A method of describing a solution set using parentheses and brackets to indicate included and excluded endpoints.

Open Circle

A mark on a number line graph used to indicate that the boundary value is not included in the solution set (used with $<$ or $>$).

Set-Builder Notation

A shorthand way to write a solution set by stating the condition that all variables in the set must meet (e.g., $\{x \mid x > 2\}$).

Glossary of Key Terms (2/2)

Solution Set

The collection of all values that make a given inequality true.

Key Concepts (1/2)

In mathematics, what is a sentence that compares two expressions using symbols like $<$ or $>$ called?

An inequality

Which mathematical symbol is used to represent 'less than'?

$<$

Which mathematical symbol is used to represent 'greater than'?

$>$

Which mathematical symbol is used to represent 'less than or equal to'?

$\le$

Which mathematical symbol is used to represent 'greater than or equal to'?

$\ge$

What is the term for the set of all values that make an inequality true?

The solution set

Key Concepts (2/2)

Term: Inequality

Definition: A mathematical sentence that compares two expressions using symbols such as $<$, $>$, $\le$, or $\ge$.

Concept: Solution Set

Definition: The set of all variable values that satisfy an inequality. Example: For $x > 3$, the set of all numbers greater than $3$.

How is the solution set of an inequality represented visually?

By shading a region on a number line.

When graphing an inequality, what does an open circle at the boundary point represent?

The boundary value is not included in the solution set.

Which two inequality symbols require an open circle on a number line graph?

$<$ and $>$

When graphing an inequality, what does a closed circle at the boundary point represent?

The boundary value is included in the solution set.
