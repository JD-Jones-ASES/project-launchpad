Set Notation and the Real Numbers

Study Slides

Set Fundamentals

Notation

$a \in A$ indicates that $a$ is an element of set $A$; $a \notin A$ indicates it is not.

Roster Notation

Elements are listed explicitly between braces, e.g., $\{1, 2, 3\}$.

Set-Builder Notation

Elements are defined by a rule, e.g., $\{x \mid x \text{ is a positive integer and } x < 5\}$. The vertical bar ($\mid$) is read as "such that."

The Empty Set ($\emptyset$)

A unique set containing no elements.

The Number-Set Hierarchy

Set Name. Symbol

Description

Natural Numbers. $\mathbb{N}$

Counting numbers starting from 1.

Integers. $\mathbb{Z}$

Natural numbers, their negatives, and zero.

Rational Numbers. $\mathbb{Q}$

Numbers expressible as a ratio $p/q$ ($q \neq 0$). Decimals terminate or repeat.

Irrational Numbers. $\mathbb{I}$

Numbers that cannot be ratios; decimals are non-terminating and non-repeating.

Real Numbers. $\mathbb{R}$

The set of all rational and irrational numbers combined.

Interval Notation

Closed Interval $[a, b]$

Includes the endpoints; represented by filled circles on a number line.

Open Interval $(a, b)$

Excludes the endpoints; represented by open circles.

Infinity Rules

$\infty$ and $-\infty$ are never used with square brackets because infinity is not a real number.

Shading

On a number line, the region between endpoints is shaded to represent all real numbers in that range.

Set Operations

Union ($A \cup B$)

Includes all elements belonging to $A$, to $B$, or to both. (Keyword: "or").

Intersection ($A \cap B$)

Includes only elements belonging to both $A$ and $B$ simultaneously. (Keyword: "and").

Glossary of Important Terms (1/3)

Element ($\in$)

An individual object that belongs to a set.

Empty Set ($\emptyset$)

A set with no members; also referred to as the null set.

Integers ($\mathbb{Z}$)

The set containing all whole numbers, their opposites (negatives), and zero.

Intersection ($\cap$)

The set operation that identifies the overlap between two sets.

Interval Notation

A method of writing a continuous range of numbers using brackets and parentheses.

Glossary of Important Terms (2/3)

Irrational Number

A real number that has a non-terminating, non-repeating decimal expansion and cannot be written as a simple fraction.

Natural Numbers ($\mathbb{N}$)

The set of positive counting numbers $\{1, 2, 3, \dots\}$.

Rational Number ($\mathbb{Q}$)

Any number that can be expressed as $p/q$ where $p$ and $q$ are integers and $q \neq 0$.

Real Numbers ($\mathbb{R}$)

The set containing every point on the number line; the combination of rational and irrational numbers.

Roster Notation

A method of describing a set by listing every element within braces.

Glossary of Important Terms (3/3)

Set

A well-defined collection of objects.

Set-Builder Notation

A method of describing a set by stating the specific rule or property its elements must satisfy.

Union ($\cup$)

The set operation that combines all elements from two or more sets.
