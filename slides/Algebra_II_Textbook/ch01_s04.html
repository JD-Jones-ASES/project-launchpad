Key Concepts (1/2)

What is the geometric interpretation of absolute value?

The absolute value of a real number $x$, written $|x|$, is the distance from $x$ to $0$ on the number line.

Provide the formal definition of the absolute value of $x$.

$|x| = x$ if $x \ge 0$, and $|x| = -x$ if $x < 0$.

What is always true about the value of an absolute value expression?

The absolute value of any real number is always non-negative; $|x| \ge 0$ for every real $x$.

State two useful algebraic identities involving absolute value.

$|-x| = |x|$ and $|xy| = |x||y|$. (Also $|x/y| = |x|/|y|$ for $y e 0$).

What are the possible solutions for an absolute value equation of the form $|E| = n$ when $n > 0$?

There are two possible solutions: $E = n$ or $E = -n$.

How many solutions does the equation $|E| = n$ have when $n = 0$?

Exactly one solution: $E = 0$.

Key Concepts (2/2)

What happens if you try to solve an absolute value equation where the absolute value is set equal to a negative number, e.g., $|E| = -2$?

There is no solution, because an absolute value expression can never be negative ($|E| \ge 0$ always holds).

Define an 'extraneous solution' in the context of solving equations.

An extraneous solution is a value produced algebraically during the solving process that does not satisfy the original equation and therefore must be discarded.

How do you solve an absolute value equation where both sides have an absolute value, such as $|A| = |B|$?

Set $A = B$ or $A = -B$. Two quantities have equal absolute values if they are equal or are exact opposites.

Geometrically, what does the inequality $|E| < m$ represent (for $m > 0$)?

It represents all real numbers for which the expression $E$ is within a distance $m$ from zero. This results in a bounded interval.

Geometrically, what does the inequality $|E| > m$ represent (for $m > 0$)?

It represents all real numbers for which the expression $E$ is farther than a distance $m$ from zero. This results in two outward rays.
