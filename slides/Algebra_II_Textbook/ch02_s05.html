Systems of Linear Equations

Study Slides

Key Concepts (1/2)

System of Linear Equations

A set of two or more linear equations in two or more variables that are to be solved simultaneously.

Solution to a System

An ordered pair $(x, y)$ that satisfies all equations in the system. Graphically, this represents the point(s) where the lines intersect.

Substitution Method

An algebraic technique to solve systems by solving one equation for one variable and substituting that expression into the other equation.

Elimination (Addition) Method

An algebraic technique to solve systems by multiplying equations by constants to create opposite coefficients for one variable, then adding the equations to eliminate that variable.

Applications

Systems of linear equations are powerful tools for modeling and solving real-world problems by translating conditions into equations.

Important Terms (1/2)

System of Linear Equations

A collection of two or more linear equations considered together. For example, $a_1 x + b_1 y = c_1$ and $a_2 x + b_2 y = c_2$.

Solution to a System

An ordered pair $(x, y)$ that makes all equations in the system true.

Substitution Method

An algebraic method for solving systems where one variable is isolated in one equation and its expression is substituted into the other equation.

Elimination (Addition) Method

An algebraic method for solving systems where equations are manipulated (multiplied by constants) and then added together to eliminate one variable.

Consistent System

A system of equations that has at least one solution.

Important Terms (2/2)

Inconsistent System

A system of equations that has no solution.

Dependent System

A system of equations where the equations are multiples of each other, resulting in infinitely many solutions.

Independent System

A system of equations where the equations are distinct and not multiples of each other (can be consistent or inconsistent).

Key Concepts (1/2)

Define a "system of linear equations" in two unknowns.

A pair of linear equations, such as $a_1 x + b_1 y = c_1$ and $a_2 x + b_2 y = c_2$, that are required to be satisfied simultaneously.

What is a "solution" to a system of linear equations?

An ordered pair $(x, y)$ that satisfies both equations in the system simultaneously.

Graphically, what does a solution to a system of two linear equations represent?

A point of intersection between the two lines represented by the equations.

What are the three possible geometric outcomes for the solution set of a system of two linear equations?

1. The lines intersect at exactly one point. 2. The lines are parallel. 3. The equations describe the same line.

Apply the Substitution Method to solve for $x$ in the system: $y = 2x - 3$, $3x + y = 7$.

Substitute $y = 2x - 3$ into the second equation: $3x + (2x - 3) = 7 \Rightarrow 5x - 3 = 7 \Rightarrow 5x = 10 \Rightarrow x = 2$.

Key Concepts (2/2)

What is the complete solution to the system $y = 2x - 3$, $3x + y = 7$?

$(x, y) = (2, 1)$. (Substitute $x=2$ into $y=2x-3$ gives $y=2(2)-3=1$. Check: $3(2)+1=7$ and $1=2(2)-3$ are true.)

To solve $3x - 2y = 11$, $5x + 4y = 1$ using elimination, what is the first step to eliminate $y$?

Multiply the first equation by $2$ to get $6x - 4y = 22$. Then the $y$-coefficients $(-4y, +4y)$ are opposites.

What is the result of adding the modified equations $6x - 4y = 22$ and $5x + 4y = 1$ when using the Elimination Method?

$11x = 23$.

What is a common error to avoid when multiplying an equation by a constant in the Elimination Method?

Forgetting to multiply every term (including the right-hand side) by the constant, especially when dealing with negative signs.

Describe a "consistent and independent" system of linear equations, including its graphical representation and number of solutions.

The lines intersect at exactly one point, resulting in one unique solution.
