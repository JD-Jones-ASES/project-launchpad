Key Concepts

Linear Functions as Models

Many real-world processes exhibit a constant rate of change, making linear functions ($f(x) = mx + b$) ideal for modeling them.

Solving Direct Variation Problems

To find $k$, substitute a known $(x,y)$ pair into $y=kx$ and solve for $k$. Once $k$ is found, the model can be used to find other values.

Important Terms (1/2)

Linear Model

An equation in the form $f(x) = mx + b$ used to represent a real-world situation where a quantity changes at a constant rate.

Slope ($m$)

In a linear model, the slope represents the constant rate of change. It indicates how much the dependent variable changes for each unit change in the independent variable.

Y-intercept ($b$)

In a linear model, the y-intercept represents the initial value or the value of the dependent variable when the independent variable is zero.

Independent Variable ($x$)

The input variable in a function; its value is chosen or controlled.

Dependent Variable ($y$ or $f(x)$)

The output variable in a function; its value depends on the independent variable.

Important Terms (2/2)

Direct Variation

A relationship between two variables, $y$ and $x$, such that $y = kx$ for some non-zero constant $k$. We say "$y$ varies directly as $x$" or "$y$ is directly proportional to $x$."

Constant of Variation (or Constant of Proportionality)

The non-zero constant $k$ in a direct variation equation ($y = kx$), which represents the constant ratio between $y$ and $x$.

Building a Linear Model: The Formula $f(x) = mx + b$ (1/2)

Independent Variable ($d$)

Distance in miles.

Dependent Variable ($C(d)$)

Total cost in dollars.

Rate of Change ($m$)

$1.80 per mile. This is how the cost changes for each additional mile. So, $m = 1.80$.

Initial Value ($b$)

Flat fee of $3.00. This is the cost when you haven't traveled any miles ($d=0$). So, $b = 3.00$.

Function

$C(d) = 1.80d + 3.00$.
