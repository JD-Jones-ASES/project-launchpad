Key Concepts (1/2)

Power Function Definition

A function defined as $f(x) = x^n$, where $n$ is a positive integer, called the **degree**.

Odd-Degree Power Functions

End Behavior

Describes the trend of the function's output ($f(x)$) as the input ($x$) becomes extremely large positive ($x \to \infty$) or extremely large negative ($x \to -\infty$).

Impact of Degree on Output

For $x > 1$, higher powers produce larger outputs ($x^4 > x^2$). For $0 < x < 1$, higher powers produce smaller outputs ($x^4 < x^2$).

Key Concepts (2/2)

Transformations

Power functions can be transformed (shifted, stretched, reflected) just like other basic functions. Vertical shifts can alter a function's symmetry.

Anchor Points

All power functions $y = x^n$ pass through $(0,0)$ and $(1,1)$. Even-degree functions also pass through $(-1,1)$, while odd-degree functions pass through $(-1,-1)$. These are useful for sketching.

Important Terms (1/2)

Power Function

A function of the form $f(x) = x^n$, where $n$ is a positive integer.

Degree

The positive integer exponent $n$ in a power function $f(x) = x^n$.

Even Function

A function $f(x)$ where $f(-x) = f(x)$ for all $x$ in its domain. Its graph is symmetric about the $y$-axis.

Odd Function

A function $f(x)$ where $f(-x) = -f(x)$ for all $x$ in its domain. Its graph has $180^\circ$ rotational symmetry about the origin.

End Behavior

The behavior of the graph of a function as $x$ approaches positive infinity ($x \to \infty$) or negative infinity ($x \to -\infty$).

Important Terms (2/2)

Symmetry about the y-axis

A graph can be folded along the y-axis, and the two halves match exactly. Characteristic of even functions.

Rotational Symmetry about the Origin

A graph looks the same after being rotated $180^\circ$ around the origin. Characteristic of odd functions.
