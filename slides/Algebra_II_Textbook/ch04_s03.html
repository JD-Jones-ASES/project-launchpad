Square Root Functions

Study Slides

Key Concepts

Domain Restriction

The most fundamental concept for square root functions is that the expression under the square root (the radicand) cannot be negative in the real number system. This restricts the function's domain.

Starting Point (Endpoint)

Unlike lines or parabolas, square root functions have a definite starting point on their graph, which is the leftmost point of their domain.

Standard Form of Transformed Square Root Functions

The general form for transformed square root functions is $f(x) = a\sqrt{x-h} + k$. Understanding the roles of $a$, $h$, and $k$ is crucial for graphing and analyzing these functions.

Important Terms

Square Root Function

The basic function $f(x) = \sqrt{x}$. Its domain is $[0, \infty)$ and its range is $[0, \infty)$. Its graph starts at the origin $(0,0)$ and curves gradually upward to the right.

Radical Function

Any function containing a variable under a radical sign. The general transformed square root function in this section is of the form $f(x) = a\sqrt{x-h} + k$, where $a e 0$.

Radicand

The expression under the radical sign. For a real result, the radicand of a square root must be non-negative.

Starting Point (Endpoint)

For a function in the form $f(x) = a\sqrt{x-h} + k$, the starting point of the graph is $(h, k)$.

Key Concepts (1/2)

What is the defining characteristic of a square root function regarding its domain?

The domain is restricted to non-negative inputs because the square root of a negative real number is not a real number.

Define the basic square root function, $f(x)$, and state its domain and range.

The basic square root function is $f(x) = \sqrt{x}$. Its domain is $[0, \infty)$ and its range is $[0, \infty)$.

What is a 'radical function'?

A radical function is any function containing a variable under a radical sign.

What is the general transformation form for a radical function as discussed in this section?

The general transformation form is $f(x) = a\sqrt{x - h} + k$, where $a$, $h$, and $k$ are real numbers with $a eq 0$.

How do you find the domain of a square root function of the form $f(x) = \sqrt{ax+b}$?

What is the effect of 'h' on the graph of $g(x) = a\sqrt{x-h} + k$?

'h' causes a horizontal shift: right $h$ units if $h > 0$, or left $|h|$ units if $h < 0$. The domain starts at $x = h$.

Key Concepts (2/2)

What is the effect of 'k' on the graph of $g(x) = a\sqrt{x-h} + k$?

'k' causes a vertical shift: up $k$ units if $k > 0$, or down $|k|$ units if $k < 0$. The range starts at $y = k$.

What is the effect of 'a' on the graph of $g(x) = a\sqrt{x-h} + k$ if $a < 0$?

If $a < 0$, the graph is reflected over the x-axis, meaning it will curve downward from the starting point.

What is the effect of the magnitude of 'a' on the graph of $g(x) = a\sqrt{x-h} + k$?

If $|a| > 1$, the graph is stretched vertically. If $0 < |a| < 1$, the graph is compressed vertically.

What are the coordinates of the starting point (endpoint) of the graph of $g(x) = a\sqrt{x-h} + k$?

The starting point of the graph is $(h, k)$.

Find the domain of $f(x) = \sqrt{x - 5}$.

Set $x - 5 \ge 0$, so $x \ge 5$. Domain: $[5, \infty)$.

Find the domain of $g(x) = \sqrt{3x + 12}$.

Set $3x + 12 \ge 0$, so $3x \ge -12$, giving $x \ge -4$. Domain: $[-4, \infty)$.
