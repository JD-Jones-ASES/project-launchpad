Arithmetic Sequence

An arithmetic sequence is an ordered list where the difference between any two consecutive terms is constant. This constant difference is known as the common difference ($d$).

Geometric Sequence

A geometric sequence is an ordered list where each term after the first is found by multiplying the previous one by a fixed, non-zero number. This fixed multiplier is called the common ratio ($r$).

$n$th-Term Formula for an Arithmetic Sequence

If the first term is $a_1$ and the common difference is $d$, the formula to find any term $a_n$ is $a_n = a_1 + (n - 1)d$. This formula highlights the linear growth of arithmetic sequences.

Important Terms (1/2)

Sequence

An ordered list of numbers, denoted by $a_1, a_2, a_3, \dots$, where each number is called a term.

Term ($a_n$)

A specific number in a sequence. $a_n$ refers to the $n$th term in the sequence.

Arithmetic Sequence (or Arithmetic Progression)

A sequence where the difference between any term and its preceding term is constant.

Common Difference ($d$)

The constant value found by subtracting any term from its succeeding term in an arithmetic sequence ($d = a_{n+1} - a_n$).

Geometric Sequence

A sequence where the ratio of any term to its preceding term is constant.

Important Terms (2/2)

Common Ratio ($r$)

The constant non-zero value found by dividing any term by its preceding term in a geometric sequence ($r = \frac{a_{n+1}}{a_n}$).

Key Concepts (1/2)

What is a sequence?

An ordered list of numbers, called terms, denoted by $a_1, a_2, a_3, \dots$, where $a_n$ represents the $n$th term.

From a function perspective, how can a sequence be viewed?

A sequence is a function whose domain is the set of positive integers (or natural numbers), where $a(n) = a_n$.

What are the two most important families of sequences discussed in this section?

Arithmetic sequences (characterized by a constant difference) and geometric sequences (characterized by a constant ratio).

Define an arithmetic sequence.

An arithmetic sequence (or arithmetic progression) is a sequence in which the difference between consecutive terms is constant.

What is the 'common difference' in an arithmetic sequence? How is it denoted and calculated?

It is the constant difference between consecutive terms, denoted by $d$. It is calculated as $d = a_{n+1} - a_n$ for all $n \ge 1$.

How do you determine if a given sequence is arithmetic?

Subtract each term from the next. If all differences equal the same constant value, $d$, then the sequence is arithmetic.

Key Concepts (2/2)

State the $n$th-Term Formula for an Arithmetic Sequence.

$a_n = a_1 + (n - 1)d$, where $a_1$ is the first term and $d$ is the common difference.

How is an arithmetic sequence related to a linear function?

An arithmetic sequence is a linear function of $n$. Plotting $(n, a_n)$ produces points that lie on a line with slope $d$ and $y$-intercept $a_1 - d$.

An arithmetic sequence has $a_1 = 7$ and common difference $d = -3$. Write a formula for $a_n$.

$a_n = 7 + (n-1)(-3) = 7 - 3n + 3 = 10 - 3n$.

For the arithmetic sequence $a_n = 10 - 3n$, find the 10th term ($a_{10}$).

$a_{10} = 10 - 3(10) = 10 - 30 = -20$.

For the arithmetic sequence $a_n = 10 - 3n$, which term equals $-50$?

Set $10 - 3n = -50$. Solving for $n$ gives $-3n = -60$, so $n = 20$. The 20th term equals $-50$.

Define a geometric sequence.

A geometric sequence is a sequence in which each term after the first is obtained by multiplying the previous term by a fixed nonzero constant.
