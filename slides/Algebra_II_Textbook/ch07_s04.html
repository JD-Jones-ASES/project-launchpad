Applications

Exponential equations are widely used to model real-world phenomena such as population growth, radioactive decay, compound interest, and doubling time for investments.

Important Terms (1/2)

Exponential Equation

An equation where the variable is found in the exponent.

Base

In an exponential expression $b^x$, $b$ is the base, the number being raised to a power.

Exponent

In an exponential expression $b^x$, $x$ is the exponent, indicating how many times the base is multiplied by itself.

Common Logarithm ($\log$)

A logarithm with base 10, typically written without the base subscript ($\log x$ means $\log_{10} x$).

Natural Logarithm ($\ln$)

A logarithm with base $e$ (Euler's number), typically written as $\ln x$ (meaning $\log_e x$).

Important Terms (2/2)

Power Rule for Logarithms

A key property of logarithms stating that $\log_b(M^p) = p \log_b(M)$. This rule allows us to bring an exponent down as a multiplier.

Isolate

To manipulate an equation algebraically so that a specific term or variable is alone on one side of the equals sign.

Back-substitute

The process of replacing a temporary variable (like $u$ in a substitution method) with its original expression to solve for the primary variable.

Key Concepts (1/2)

What defines an exponential equation?

An exponential equation is one in which the variable appears in an exponent.

State the Same-Base Property for solving exponential equations.

If b^(f(x)) = b^(g(x)) with b > 0 and b ≠ 1, then f(x) = g(x).

What is the primary strategy when using the Same-Base Method?

Rewrite both sides of the equation with the same base, then set the exponents equal to each other.

Solve 2^(3x) = 2^(x+4).

3x = x + 4 => 2x = 4 => x = 2.

Solve 9^x = 27 using the Same-Base Method.

(3^2)^x = 3^3 => 3^(2x) = 3^3 => 2x = 3 => x = 3/2.

Solve 4^(x-1) = 8^(x+2).

(2^2)^(x-1) = (2^3)^(x+2) => 2^(2x-2) = 2^(3x+6) => 2x - 2 = 3x + 6 => x = -8.

When should you use the 'Take the Logarithm' strategy to solve an exponential equation?

When the equation cannot be rewritten with the same base on both sides.

Which logarithm rule is crucial when solving exponential equations by taking logarithms of both sides?

The Power Rule: log_b(M^p) = p log_b(M) (or ln(M^p) = p ln M).

Solve 5^x = 80, rounding to four decimal places.

ln(5^x) = ln(80) => x ln 5 = ln 80 => x = ln(80) / ln(5) ≈ 2.7228.

Does it matter if you use the common logarithm (LOG) or the natural logarithm (LN) when solving exponential equations?

No, both will give the same answer. The choice often depends on the context (e.g., LN in science/calculus, LOG in engineering) or calculator preference.

Solve 3e^(2x) - 1 = 74.

3e^(2x) = 75 => e^(2x) = 25 => ln(e^(2x)) = ln(25) => 2x = ln 25 => x = (ln 25) / 2.
