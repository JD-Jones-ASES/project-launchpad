Inverse Functions

Study Slides

Key Concepts (1/2)

One-to-One Function

A function where every element of the range corresponds to exactly one element of the domain. In simpler terms, different inputs always produce different outputs ($f(a) = f(b) \implies a = b$).

Horizontal Line Test

A graphical method to determine if a function is one-to-one. If any horizontal line intersects the graph of a function at most once, the function is one-to-one.

Inverse Function ($f^{-1}$)

For a one-to-one function $f$, its inverse function $f^{-1}$ reverses the mapping. If $f(x) = y$, then $f^{-1}(y) = x$. Crucially, applying a function and its inverse in sequence brings you back to...

Domain and Range Swap

The domain of a function $f$ becomes the range of its inverse $f^{-1}$, and the range of $f$ becomes the domain of $f^{-1}$.

Algebraic Method for Finding an Inverse

A systematic procedure involving replacing $f(x)$ with $y$, swapping $x$ and $y$, solving for the new $y$, and then replacing $y$ with $f^{-1}(x)$.

Key Concepts (2/2)

Verification of Inverses

Two functions $f$ and $g$ are inverses of each other if and only if their compositions result in the identity function: $f(g(x)) = x$ AND $g(f(x)) = x$.

Important Terms (1/2)

One-to-One Function

A function $f$ for which $f(a) = f(b)$ implies that $a = b$. This means distinct inputs always map to distinct outputs.

Horizontal Line Test

A graphical test to determine if a function is one-to-one. A function passes the test if no horizontal line intersects its graph at more than one point.

Inverse Function ($f^{-1}$)

The function that reverses the action of a given one-to-one function $f$. It takes outputs of $f$ back to their original inputs. The notation $f^{-1}(x)$ is not the same as $1/f(x)$.

Domain

The set of all possible input values (independent variable) for which a function is defined.

Range

The set of all possible output values (dependent variable) that a function can produce.

Important Terms (2/2)

Composition of Functions

An operation that combines two functions such that the output of one function becomes the input of the other. Denoted as $(f \circ g)(x) = f(g(x))$.

Definition

A function $f$ is one-to-one if for any two distinct inputs, $a$ and $b$, their outputs are also distinct. Formally, $f(a) = f(b)$ implies $a = b$.

Definition and Properties of an Inverse Function

Domain and Range Swap

The domain of $f^{-1}$ is the range of $f$ (D_{f^{-1}} = R_f), and the range of $f^{-1}$ is the domain of $f$ (R_{f^{-1}} = D_f). This is a crucial consequence of "undoing" the function.

Composition Property

Notation Alert

Remember that $f^{-1}(x)$ denotes the inverse function, not the reciprocal $1/f(x)$. These are generally not equal.

Finding the Inverse Algebraically

Linear Function

$f(x) = 3x - 5$

Rational Function

$g(x) = \frac{2x+1}{x-3}$, where $x eq 3$.

Square Root Function

$h(x) = \sqrt{x-1}$, where $x \ge 1$.

Verifying Inverse Functions

• $(f \circ g)(x) = f(g(x)) = x$

• $(g \circ f)(x) = g(f(x)) = x$

• $f(g(x)) = f(\frac{x+5}{3}) = 3(\frac{x+5}{3}) - 5 = (x+5) - 5 = x$.

• $g(f(x)) = g(3x-5) = \frac{(3x-5)+5}{3} = \frac{3x}{3} = x$.

Key Concepts (1/2)

What mathematical concept captures the idea of "undoing" a function?

The inverse function.

Define a one-to-one function.

A function $f$ is one-to-one if different inputs always produce different outputs, meaning $f(a) = f(b)$ implies $a = b$.

How can you graphically determine if a function is one-to-one?

Use the Horizontal Line Test: A function is one-to-one if and only if every horizontal line intersects its graph in at most one point.

If $f$ is a one-to-one function with domain $D$ and range $R$, describe its inverse function $f^{-1}$.

Its inverse function $f^{-1}$ has domain $R$ and range $D$, and satisfies $f^{-1}(f(x)) = x$ for all $x \in D$, and $f(f^{-1}(y)) = y$ for all $y \in R$.

What is the meaning of the notation $f^{-1}(x)$?

It denotes the inverse function of $f(x)$. It is not the same as the reciprocal $1/f(x)$.

Find the inverse of $f(x) = 3x - 5$.

$f^{-1}(x) = \frac{x+5}{3}$.

When finding the inverse of a rational function like $g(x) = \frac{2x+1}{x-3}$, what happens to the excluded values (domain restrictions)?

The excluded values swap. If $x e 3$ for $g(x)$, then the inverse $g^{-1}(x)$ will have $x e 2$ (which was the range restriction of $g(x)$).
