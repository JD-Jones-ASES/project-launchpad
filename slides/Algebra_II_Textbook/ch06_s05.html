x-Intercepts

Key Concepts (2/2)

y-Intercepts

The point where the graph crosses the y-axis, meaning the x-value is zero. This is found by evaluating $f(0)$.

Sign Analysis

A method used to determine where the graph lies above or below the x-axis within intervals defined by vertical asymptotes and x-intercepts. This helps in sketching the curve's behavior.

Important Terms

Vertical Asymptotes (Review and Refinement)

Holes (Removable Discontinuities)

Factor

Numerator: $(x-2)(x+1)$. Denominator: $(x-2)(x+2)$.

Function becomes

$f(x) = \frac{(x-2)(x+1)}{(x-2)(x+2)}$.

Cancellation

The factor $(x-2)$ cancels.

Hole

Since $(x-2)$ canceled, there's a hole at $x=2$.

Vertical Asymptote

The remaining denominator factor is $(x+2)$. Set $x+2=0$, so $x=-2$. This factor did not cancel, so there's a VA at $x=-2$.

x-Intercepts

Factor numerator

$(x-2)(x-3)$.

Simplified form

$\frac{(x-2)(x-3)}{x-1}$ (no common factors, so no holes).

Set numerator to zero

$(x-2)(x-3) = 0$.

Solutions

$x=2$ and $x=3$.

x-intercepts

$(2,0)$ and $(3,0)$. (Neither $x=2$ nor $x=3$ makes the denominator $x-1$ zero, so they are valid intercepts.)
