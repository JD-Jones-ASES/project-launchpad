Function Notation

Writing Functions

When $y$ is a function of $x$, we use the notation $y = f(x)$, read as "$f$ of $x$".

Important Note

$f(x)$ does not mean $f$ multiplied by $x$. It indicates the value of the function $f$ at the input $x$.

Evaluating Functions

To find the output for a specific input, substitute that input value for every occurrence of $x$ in the function's formula.

One-to-One Functions

Definition

A function is one-to-one if distinct inputs always produce distinct outputs. That is, if $f(x_1) = f(x_2)$, then $x_1$ must equal $x_2$. Equivalently, no two different inputs produce the same output.

Horizontal Line Test

This test helps determine visually if a function is one-to-one.

Significance

One-to-one functions are important because they are the only functions that have inverse functions.

How do you determine if a table of values defines a function?

A graph contains the points (-1, 2), (-1, -3), and (2, 4). Does this graph represent a function? Explain.

No, it does not. The input x = -1 is paired with two different outputs, y=2 and y=-3. This violates the definition of a function (and fails the VLT).

What does the notation y = f(x) mean?

It means that y is a function of x, and f(x) denotes the output of the function f when the input is x.

Is f(x) multiplication of f by x?

No, f(x) is not multiplication. It's function notation where f is the name of the function and x is the input.

Given f(x) = 2x^2 - 3x + 1, find f(0).

f(0) = 2(0)^2 - 3(0) + 1 = 1.

Given f(x) = 2x^2 - 3x + 1, find f(-2).

f(-2) = 2(-2)^2 - 3(-2) + 1 = 2(4) + 6 + 1 = 8 + 6 + 1 = 15.
