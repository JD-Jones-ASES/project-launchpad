Inequalities in Word Problems

Real-world scenarios can be translated into inequalities by identifying key phrases such as "at most" ($\le$), "at least" ($\ge$), "no more than" ($\le$), and "no less than" ($\ge$).

Important Terms (1/2)

Inequality

A mathematical statement asserting that two expressions are not equal or that one is larger/smaller than the other.

Solution Set

The collection of all real numbers that, when substituted into an inequality, make the inequality a true statement.

Interval Notation

A concise way to represent sets of real numbers over a continuous range, using parentheses `()` for exclusive endpoints (not included) and square brackets `[]` for inclusive endpoints (included).

Compound Inequality

A mathematical statement consisting of two or more inequalities joined by the words "and" or "or."

AND Compound Inequality

A compound inequality where the solution must satisfy *all* conditions simultaneously. The solution is the intersection of the individual solution sets.

Important Terms (2/2)

OR Compound Inequality

A compound inequality where the solution must satisfy *at least one* of the conditions. The solution is the union of the individual solution sets.

Identity (Inequality)

An inequality that is true for all possible real number values of the variable (e.g., $x+1 > x$). Its solution set is $(-\infty, \infty)$.

Contradiction (Inequality)

An inequality that is never true for any real number value of the variable (e.g., $x+1 < x$). Its solution set is $\emptyset$ (the empty set).

The Critical Rule: Reversing the Inequality Sign

Example

If you have $-2x < 6$, dividing by $-2$ means you must flip the sign: $x > -3$.

Common Error

Forgetting this rule is the most frequent mistake in solving inequalities.

Properties of Inequality

Addition/Subtraction

If $a < b$, then $a + c < b + c$ and $a - c < b - c$. (The inequality direction remains unchanged.)

Multiplication/Division by a Positive Number

If $a < b$ and $c > 0$, then $ac < bc$ and $\frac{a}{c} < \frac{b}{c}$. (The inequality direction remains unchanged.)

Multiplication/Division by a Negative Number

If $a < b$ and $c < 0$, then $ac > bc$ and $\frac{a}{c} > \frac{b}{c}$. (The inequality direction *must be reversed*!)

"AND" Compound Inequalities

Definition

Both inequalities must be true simultaneously. The solution is the **intersection** of the individual solutions.

Compact Form

Often written as $a < x < b$, meaning $a < x$ AND $x < b$.

Solving Strategy

Perform operations on all three parts of the inequality at the same time.

"OR" Compound Inequalities

Definition

At least one of the inequalities must be true. The solution is the **union** of the individual solutions.

Solving Strategy

Solve each inequality separately, then combine their solution sets using the union symbol ($\cup$).

Special Cases: Identities and Contradictions

Identity

If solving leads to a true statement like $5 > 2$, the inequality is an identity, and its solution set is $(-\infty, \infty)$.

Contradiction

If solving leads to a false statement like $0 > 7$, the inequality is a contradiction, and its solution set is $\emptyset$ (the empty set).
