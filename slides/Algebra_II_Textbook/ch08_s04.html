Key Concepts

Important Terms (1/2)

Piecewise Function

A function defined by two or more expressions, each applied to a specified interval of the domain.

Breakpoint

A value of $x$ in a piecewise function where the definition (the rule/formula) of the function changes.

Jump Discontinuity

A type of discontinuity in a piecewise function where the graph "jumps" from one value to another at a breakpoint, meaning the left-hand and right-hand limits at that point are not equal.

Floor Function ($\lfloor x \rfloor$) (Greatest Integer Function)

A function that returns the greatest integer less than or equal to $x$.

Ceiling Function ($\lceil x \rceil$)

A function that returns the least integer greater than or equal to $x$.

Important Terms (2/2)

Cube Root Function

A function of the form $f(x) = \sqrt[3]{x}$, which finds the number that, when cubed, equals $x$. It has a domain and range of all real numbers.

Gaussian Function (Bell Curve)

A symmetric, bell-shaped function, commonly written as $f(x) = ae^{-(x-h)^2/(2\sigma^2)}$, used to model data distributions, especially the normal distribution in statistics.

Normal Distribution

A common continuous probability distribution in statistics, characterized by its bell-shaped curve (Gaussian function).

Key Concepts (1/2)

What is a piecewise function?

A function defined by two or more expressions, each applied to a specified interval of the domain. It uses different rules on different parts of the domain.

How do you evaluate a piecewise function at a given value x=a?

Determine which interval of the domain contains 'a', then substitute 'a' into only the formula corresponding to that piece.

When graphing a piecewise function, how do you represent endpoints that are included versus excluded?

Use a solid dot (•) at an endpoint that is included (≤ or ≥). Use an open dot (○) at an endpoint that is excluded (< or >).

What are the conditions for a piecewise function to be continuous at a breakpoint x=c?

Key Concepts (2/2)

What is a step function?

A step function is a piecewise-constant function whose graph consists of horizontal segments that 'jump' at certain values.

Define the floor function.

The floor function ⌊x⌋ (also called the greatest integer function) returns the greatest integer less than or equal to x.

Define the ceiling function.

The ceiling function ⌈x⌉ returns the least integer greater than or equal to x.

Evaluate ⌊3.7⌋, ⌈-2.1⌉, and ⌊5⌋.

⌊3.7⌋ = 3, ⌈-2.1⌉ = -2, ⌊5⌋ = 5.

Domain: (-∞, ∞); Range: (-∞, ∞); Symmetry: Odd function (f(-x) = -f(x)), symmetric about the origin.
