Identities and Contradictions

Special types of equations where the variable cancels out. An identity is always true (solution set is all real numbers), while a contradiction is never true (empty solution set).

Important Terms (1/3)

Equation

A mathematical statement asserting that two algebraic expressions are equal.

Solution (or Root)

A value of the variable that makes an equation a true statement when substituted.

Solution Set

The set of all values that are solutions to a given equation.

Linear Equation in One Variable

Any equation that can be written in the form $ax + b = 0$, where $a$ and $b$ are real numbers and $a e 0$.

Equivalent Equations

Equations that have exactly the same solution set.

Important Terms (2/3)

Addition Property of Equality

If $a = b$, then $a + c = b + c$.

Subtraction Property of Equality

If $a = b$, then $a - c = b - c$.

Multiplication Property of Equality

If $a = b$, then $ac = bc$.

Division Property of Equality

If $a = b$ and $c e 0$, then $a/c = b/c$.

Identity

An equation that is true for every real number; its solution set is all real numbers ($\mathbb{R}$).

Important Terms (3/3)

Contradiction

An equation that is never true for any real number; its solution set is the empty set ($\emptyset$ or {}).

Conditional Equation

An equation that is true for some but not all real numbers; most equations encountered in algebra fall into this category.

Literal Equation

An equation involving two or more variables, where the task is to isolate one specific variable in terms of the others (e.g., solving $A = lw$ for $l$ yields $l = A/w$).

The Role of Properties of Equality

Addition Property

You can add any real number ($c$) to both sides of an equation. If $a=b$, then $a+c=b+c$.

Subtraction Property

You can subtract any real number ($c$) from both sides of an equation. If $a=b$, then $a-c=b-c$.

Multiplication Property

You can multiply both sides of an equation by any non-zero real number ($c$). If $a=b$, then $ac=bc$.

Division Property

You can divide both sides of an equation by any non-zero real number ($c$). If $a=b$ and $c e 0$, then $a/c=b/c$.

Key Concepts (1/2)

What is an equation?

An equation is a statement that two expressions are equal.

Define 'solution' (or 'root') of an equation.

A solution (or root) of an equation is a value of the variable that makes the equation true.

What is a solution set?

The solution set is the set of all values of the variable that make the equation true.

What is the standard form of a linear equation in one variable x?

A linear equation in one variable x is any equation that can be written in the form ax + b = 0, where a, b are real numbers and a ≠ 0.

When are two equations considered equivalent?

Two equations are equivalent if they have exactly the same solution set.

State the Addition Property of Equality.

If a = b, then a + c = b + c. (You may add the same quantity to both sides of an equation.)

Key Concepts (2/2)

State the Multiplication Property of Equality.

If a = b, then ac = bc. (You may multiply both sides of an equation by the same quantity.)

State the Division Property of Equality.

If a = b and c ≠ 0, then a/c = b/c. (You may divide both sides of an equation by the same non-zero quantity.)

What is the first step when solving an equation that contains fractions?

Clear any fractions by multiplying both sides of the equation by the least common denominator (LCD).

After clearing fractions, what is the next step in the general strategy for solving a linear equation?

Use the distributive property to remove all parentheses.

Once all parentheses are removed, what should you do with terms on each side of the equation?

Combine like terms on each side of the equation.

What is the goal after combining like terms on each side of a linear equation?

Collect all variable terms on one side of the equation and all constant terms on the other side.
