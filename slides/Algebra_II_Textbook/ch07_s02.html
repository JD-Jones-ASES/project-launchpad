Exponential Functions

Key Concepts

Definition of an Exponential Function

A function of the form $f(x) = b^x$, where the base $b$ is a positive constant not equal to 1 ($b > 0, b \ne 1$), and the exponent $x$ can be any real number.

Transformations

Like other functions, exponential functions can be shifted vertically or horizontally, stretched, or reflected. For $g(x) = b^{x-h} + k$, $h$ causes a horizontal shift and $k$ causes a vertical shift.

The Natural Base *e*

An important irrational number approximately equal to 2.71828. It arises naturally in many growth and decay processes and is the base for the natural exponential function, $f(x) = e^x$.

Important Terms (1/3)

Exponential Function

A function where the independent variable is the exponent, typically written as $f(x) = b^x$ for a positive base $b \ne 1$.

Base ($b$)

The constant value that is raised to the power of the variable $x$ in an exponential function.

Exponent ($x$)

The variable in an exponential function, indicating the power to which the base is raised.

Exponential Growth

Occurs when the base $b > 1$, causing the function's output to increase at an accelerating rate.

Exponential Decay

Occurs when the base $0 < b < 1$, causing the function's output to decrease at an accelerating rate towards zero.

Important Terms (2/3)

Natural Base *e*

An irrational mathematical constant approximately 2.71828, fundamental in calculus and various scientific applications.

Natural Exponential Function

The exponential function with base $e$, written as $f(x) = e^x$.

Compound Interest

Interest calculated on the initial principal and also on all accumulated interest from previous periods.

Principal ($P$)

The initial amount of money invested or borrowed.

Annual Interest Rate ($r$)

The stated interest rate per year, expressed as a decimal in formulas.

Important Terms (3/3)

Compounding Frequency ($n$)

The number of times interest is calculated and added to the principal within one year.

Time ($t$)

The duration, in years, for which the money is invested or borrowed.

Amount ($A$)

The final value of an investment or loan after a certain period, including both principal and accumulated interest.

Horizontal Asymptote

A horizontal line that the graph of a function approaches as the input (x-value) tends towards positive or negative infinity.

Key Properties of $f(x) = b^x$ (1/2)

Domain

The set of all possible $x$-values is $(-\infty, \infty)$. You can raise any positive number (except 1) to any real power.

Range

The set of all possible $y$-values is $(0, \infty)$. The output of $b^x$ will always be positive, but it will never actually reach zero.

Y-intercept

For any base $b$, the $y$-intercept is always $(0, 1)$. This is because $f(0) = b^0 = 1$.

One-to-One

Exponential functions pass the Horizontal Line Test, meaning for every unique $y$-value, there is only one unique $x$-value. This property is important for inverse functions (logarithms).
