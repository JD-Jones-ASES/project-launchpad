Exponential Functions

Key Concepts

Definition of an Exponential Function

A function of the form $f(x) = b^x$, where the base $b$ is a positive constant not equal to 1 ($b > 0, b e 1$), and the exponent $x$ can be any real number.

Transformations

Like other functions, exponential functions can be shifted vertically or horizontally, stretched, or reflected. For $g(x) = b^{x-h} + k$, $h$ causes a horizontal shift and $k$ causes a vertical shift.

The Natural Base e

An important irrational number approximately equal to 2.71828. It arises naturally in many growth and decay processes and is the base for the natural exponential function, $f(x) = e^x$.

Important Terms (1/3)

Exponential Function

A function where the independent variable is the exponent, typically written as $f(x) = b^x$ for a positive base $b e 1$.

Base ($b$)

The constant value that is raised to the power of the variable $x$ in an exponential function.

Exponent ($x$)

The variable in an exponential function, indicating the power to which the base is raised.

Exponential Growth

Occurs when the base $b > 1$, causing the function's output to increase at an accelerating rate.

Exponential Decay

Occurs when the base $0 < b < 1$, causing the function's output to decrease at an accelerating rate towards zero.

Important Terms (2/3)

Natural Base e

An irrational mathematical constant approximately 2.71828, fundamental in calculus and various scientific applications.

Natural Exponential Function

The exponential function with base $e$, written as $f(x) = e^x$.

Compound Interest

Interest calculated on the initial principal and also on all accumulated interest from previous periods.

Principal ($P$)

The initial amount of money invested or borrowed.

Annual Interest Rate ($r$)

The stated interest rate per year, expressed as a decimal in formulas.

Important Terms (3/3)

Compounding Frequency ($n$)

The number of times interest is calculated and added to the principal within one year.

Time ($t$)

The duration, in years, for which the money is invested or borrowed.

Amount ($A$)

The final value of an investment or loan after a certain period, including both principal and accumulated interest.

Horizontal Asymptote

A horizontal line that the graph of a function approaches as the input (x-value) tends towards positive or negative infinity.

Introduction to Exponential Functions

• The base $b$ must be a positive number ($b > 0$).

• The base $b$ cannot be equal to 1 ($b e 1$). If $b=1$, then $f(x) = 1^x = 1$, which is a constant function, not exponential.

• The exponent $x$ can be any real number.

Key Properties of $f(x) = b^x$ (1/2)

Domain

The set of all possible $x$-values is $(-\infty, \infty)$. You can raise any positive number (except 1) to any real power.

Range

The set of all possible $y$-values is $(0, \infty)$. The output of $b^x$ will always be positive, but it will never actually reach zero.

Y-intercept

For any base $b$, the $y$-intercept is always $(0, 1)$. This is because $f(0) = b^0 = 1$.

One-to-One

Exponential functions pass the Horizontal Line Test, meaning for every unique $y$-value, there is only one unique $x$-value. This property is important for inverse functions (logarithms).

Key Concepts (1/2)

What is the defining characteristic of an exponential function compared to a polynomial?

In an exponential function (e.g., $b^x$), the variable is the exponent, while in a polynomial (e.g., $x^3$), the variable is the base.

State the general form of an exponential function and the conditions on its base.

An exponential function has the form $f(x) = b^x$, where the base $b > 0$ and $b eq 1$.

What is the domain of any basic exponential function $f(x) = b^x$?

The domain is $(-\infty, \infty)$.

What is the range of any basic exponential function $f(x) = b^x$?

The range is $(0, \infty)$.

What is the y-intercept for any basic exponential function $f(x) = b^x$?

The y-intercept is $(0, 1)$ for every base $b$.

What is the horizontal asymptote for the basic exponential function $f(x) = b^x$?

The horizontal asymptote is $y = 0$.

Key Concepts (2/2)

Under what condition does an exponential function $f(x) = b^x$ exhibit growth?

An exponential function exhibits growth when its base $b > 1$ (the function is increasing).

Under what condition does an exponential function $f(x) = b^x$ exhibit decay?

An exponential function exhibits decay when its base $0 < b < 1$ (the function is decreasing).

What does it mean for an exponential function to be 'one-to-one'?

It means the function passes the Horizontal Line Test, so each output value corresponds to exactly one input value.

If $f(x) = 3^x$, what is $f(-1)$?

$f(-1) = 3^{-1} = 1/3$.

If $f(x) = 3^x$, what is $f(1/2)$?

$f(1/2) = 3^{1/2} = \sqrt{3} \approx 1.732$.

How does the graph of $g(x) = 2^{x-3}$ relate to the graph of $f(x) = 2^x$?

The graph of $g(x)$ is the graph of $f(x)$ shifted 3 units to the right.
