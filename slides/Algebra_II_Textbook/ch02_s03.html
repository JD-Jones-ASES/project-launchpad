Linear Functions

Key Concepts

Three Forms of Linear Equations

Lines can be expressed in Point-Slope form ($y - y_1 = m(x - x_1)$), Slope-Intercept form ($y = mx + b$), or Standard form ($Ax + By = C$), each useful depending on the given information.

Writing Equations from Given Information

Graphing Using Intercepts

An efficient method for graphing linear equations (especially in standard form) by finding the points where the line crosses the x-axis (x-intercept) and y-axis (y-intercept).

Linear Functions

Defined as $f(x) = mx + b$, where $m$ is the slope and $b$ is the y-intercept. All linear functions (with $m \neq 0$) are one-to-one.

Important Terms (1/2)

Point-Slope Form

The equation of a line with slope $m$ passing through the point $(x_1, y_1)$ is $y - y_1 = m(x - x_1)$.

Slope-Intercept Form

The equation of a line with slope $m$ and y-intercept $b$ is $y = mx + b$.

Standard Form

The standard form of a linear equation is $Ax + By = C$, where $A$, $B$, and $C$ are integers, $A \geq 0$, and the greatest common factor of $A$, $B$, and $C$ is 1.

x-intercept

The point where a line crosses the x-axis. Found by setting $y = 0$ in the equation and solving for $x$.

y-intercept

The point where a line crosses the y-axis. Found by setting $x = 0$ in the equation and solving for $y$.

Important Terms (2/2)

Parallel Lines

Lines that have the **same slope** and never intersect.

Perpendicular Lines

Lines that intersect at a right angle (90 degrees). Their slopes are **negative reciprocals** of each other (e.g., if one slope is $m$, the perpendicular slope is $-1/m$).

Linear Function

A function that can be written in the form $f(x) = mx + b$, where $m$ and $b$ are constants. Its graph is a straight line.

Writing Equations of Lines (3/3)

Summary for Slopes

The slope is crucial because it completely defines the direction of a line. Knowing the slope and just one point on the line is sufficient to write its unique equation.

Graphing Lines Using Intercepts

x-intercept

The point where the line crosses the x-axis. At this point, the y-coordinate is always 0.

y-intercept

The point where the line crosses the y-axis. At this point, the x-coordinate is always 0.

Process

Example (from text)

Graph $2x - 3y = 6$ using intercepts.

Special Case

If both intercepts are $(0, 0)$ (meaning the line passes through the origin), you'll need to find a second point by substituting any convenient non-zero value for $x$ and solving for $y$.

Linear Functions as Functions

Definition

A linear function is any function that can be written in the form $f(x) = mx + b$.

Graph

The graph of a linear function is always a straight line. The constant $m$ represents the slope, and $b$ represents the y-intercept.
