Key Concepts (1/2)

What is the geometric definition of the absolute value of a number $x$?

The absolute value of a number $x$, written $x$, is the distance from $x$ to $0$ on the number line.

How is the absolute value of a real number $x$ defined algebraically (piecewise)?

$x = cases x & if x \geq 0, -x & if x < 0. cases$

Is the absolute value of a number ever negative? Explain.

No, the absolute value $x$ is always non-negative ($x \geq 0$) because it represents distance, which is never negative.

What is the standard absolute value function, and what are its domain and range?

The standard absolute value function is $f(x) = x$. Its domain is $(-\infty, \infty)$ and its range is $[0, \infty)$.

Describe the characteristic graph shape and vertex of the absolute value function $f(x) = x$.

The graph of $f(x) = x$ is a V-shape with its vertex at the origin $(0, 0)$.

Is $f(x) = x$ an even or odd function? How does its graph demonstrate this?

It is an even function because $f(-x) = -x = x = f(x)$ for all $x$. Its graph is symmetric about the $y$-axis.

Key Concepts (2/2)

What is the general form for transformations of an absolute value function?

The general form is $y = a\,x - h + k$.

In the general form $y = a\,x - h + k$, what transformation does $h$ control?

$h$ shifts the vertex horizontally: right $h$ units if $h > 0$, or left $|h|$ units if $h < 0$.

In the general form $y = a\,x - h + k$, what transformation does $k$ control?

$k$ shifts the vertex vertically: up $k$ units if $k > 0$, or down $|k|$ units if $k < 0$.

In the general form $y = a\,x - h + k$, what does the sign of $a$ indicate?

If $a > 0$, the V opens upward. If $a < 0$, it opens downward (a reflection over the $x$-axis).

In the general form $y = a\,x - h + k$, what does the magnitude of $a$ (i.e., $|a|$) indicate?

If $|a| > 1$, the graph is steeper (vertical stretch). If $0 < |a| < 1$, the graph is wider (vertical compression).

What are the vertex and axis of symmetry for an absolute value function in the form $f(x) = a\,x - h + k$?

The vertex is $(h, k)$ and the axis of symmetry is the vertical line $x = h$.
