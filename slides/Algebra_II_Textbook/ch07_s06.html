Logarithmic Functions

Study Slides

Key Concepts (1/2)

Definition of a Logarithmic Function

A function of the form $f(x) = \log_b(x)$, where $b > 0$ and $b \neq 1$.

Inverse Relationship

A logarithmic function $f(x) = \log_b(x)$ is the inverse of the exponential function $g(x) = b^x$. This means their graphs are reflections of each other across the line $y = x$.

Key Concepts (2/2)

Domain Restriction

For any logarithmic function $f(x) = \log_b(g(x))$, its argument $g(x)$ must be strictly positive ($g(x) > 0$). This is the most crucial step when finding the domain.

Important Terms (1/2)

Logarithmic Function

A function $f(x) = \log_b(x)$ that is the inverse of the exponential function $g(x) = b^x$.

Base (of a logarithm)

The number $b$ in $\log_b(x)$. It must be positive and not equal to 1.

Argument (of a logarithm)

The expression inside the logarithm, denoted by $x$ in $\log_b(x)$ or $g(x)$ in $\log_b(g(x))$. This expression must always be greater than zero.

Inverse Functions

Two functions $f$ and $g$ are inverse functions if $f(g(x)) = x$ and $g(f(x)) = x$. Their graphs are symmetric with respect to the line $y=x$.

Vertical Asymptote

A vertical line that the graph of a function approaches but never touches. For $f(x) = \log_b(x)$, the y-axis ($x=0$) is the vertical asymptote.

Important Terms (2/2)

Domain

The set of all possible input values (x-values) for which the function is defined. For logarithmic functions, this is determined by ensuring the argument is positive.

Range

The set of all possible output values (y-values) of a function. The range of a basic logarithmic function is all real numbers.

x-intercept

The point where the graph of a function crosses the x-axis, meaning the y-value is 0. For basic logarithmic functions, this is $(1,0)$.

y-intercept

The point where the graph of a function crosses the y-axis, meaning the x-value is 0. Basic logarithmic functions do not have a y-intercept because $x=0$ is a vertical asymptote.

Transformations of Logarithmic Functions (1/2)

Vertical Shift

$f(x) + k$ moves the graph vertically. $k > 0$ shifts up, $k < 0$ shifts down. The domain and vertical asymptote remain unchanged.

Horizontal Shift

Reflection across the x-axis

$-f(x)$ flips the graph upside down. The domain and vertical asymptote are unchanged.

Reflection across the y-axis

$f(-x)$ flips the graph left-to-right. This **does affect** the vertical asymptote and domain. The new domain will be $(-\infty, 0)$ for a basic reflected log.
