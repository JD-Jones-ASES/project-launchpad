Measures of Central Tendency

Single numbers that represent the center or "typical" value of a dataset.

Measures of Spread (Variation)

Numbers that describe how spread out or clustered the data values are.

Weighted Average

An average where each value contributes differently based on an assigned "weight" or importance factor.

Important Terms (1/3)

Arithmetic Mean ($x$-bar)

Often just called the "mean" or "average." Calculated by summing all values and dividing by the total number of values.

Outliers

Values in a dataset that are significantly higher or lower than most of the other values. They can heavily influence the mean.

Median

The central value in a dataset that has been ordered from least to greatest. If there's an even number of values, it's the average of the two middle values.

Mode

The value that occurs with the greatest frequency in a dataset. A dataset can be unimodal, bimodal, multimodal, or have no mode.

Range

A simple measure of spread, calculated as the difference between the maximum (largest) and minimum (smallest) values in a dataset.

Important Terms (2/3)

Weighted Average ($x_w$)

An average where individual values are multiplied by a weight (importance factor) before summing and dividing by the sum of the weights.

Frequency Distribution

A table that organizes data into a set of non-overlapping classes (intervals) and shows the number of observations (frequency) falling into each class.

Class Width

The difference between the lower limits of two consecutive classes in a frequency distribution.

Frequency

The count of data values that fall within a particular class in a frequency distribution.

Relative Frequency

The proportion or percentage of data values that fall within a particular class, calculated as (Class Frequency / Total Frequency).

Important Terms (3/3)

Cumulative Frequency

The running total of frequencies, representing the number of data values up to and including a particular class.

Key Concepts (1/2)

What is the primary purpose of statistical summary measures?

To summarize a collection of numbers with a single value, indicating whether typical values are high/low and if they are spread out or clustered.

Define the arithmetic mean.

The sum of all values in a data set divided by the number of values. It's often called the 'average' and represented by $x$ (x-bar).

What is the formula for calculating the mean of a data set?

$x = \frac{x_1 + x_2 + ... + x_n}{n}$, where $x_i$ are the individual values and $n$ is the number of values.

How do you calculate the mean from a frequency table?

Multiply each value by its frequency, sum these products, and then divide by the total frequency (sum of all frequencies).

Define the median.

The median is the middle value of a data set when the values are arranged in order from least to greatest.

Key Concepts (2/2)

How do you find the median when there is an odd number of values in a data set?

Arrange the values in order and select the single middle value, located at the position $(n+1)/2$.

How do you find the median when there is an even number of values in a data set?

Arrange the values in order and calculate the average of the two middle values, located at positions $n/2$ and $n/2 + 1$.
