Key Concepts (1/2)

Proposition

A declarative sentence that is either absolutely true or absolutely false, but cannot be both or neither. This forms the basic building block of propositional logic.

Logical Connective

A word or phrase that joins two or more propositions (or modifies one) to create a new, larger proposition.

AND (Conjunction)

A connective that links two propositions; the combined statement is only true if *both* individual propositions are true. It uses the symbol $\land$.

OR (Disjunction)

A connective that links two propositions; the combined statement is true if *at least one* of the individual propositions is true. In logic, "OR" is always **inclusive**, meaning it is true even if...

NOT (Negation)

A connective that changes the truth value of a single proposition. If the original proposition is true, its negation is false, and vice versa. It uses the symbol $\lnot$.

Antecedent

Consequent

Introduction to Propositional Logic

Shift in Focus

Chapters 1-3 examined arguments as a whole. Propositional logic (Chapters 4-7) "zooms in" to analyze how individual sentences are connected.

Core Idea

It's the study of how simple statements are combined using specific words like "and," "or," "not," and "if then."

What Is a Proposition?

Definition

A proposition is a statement that has a definite truth value. It must be able to be labeled as *true* or *false*.

Exclusions

Sentences that are commands ("Close the door!"), questions ("What time is lunch?"), or exclamations ("Wow, that was amazing!") are *not* propositions because they cannot be assigned a truth value.

Test

If you can reasonably determine whether a sentence is true or false, it's a proposition.
