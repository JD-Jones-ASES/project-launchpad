Important Terms

Logic Puzzle

A problem that can be solved entirely through reasoning, using a given set of clues to find the one correct answer.

Grid Puzzle

A type of logic puzzle that uses a grid or table to represent items from different categories and determine which items match based on clues.

Elimination

The process of ruling out options that cannot be correct, narrowing down possibilities until the right answer is the only one left.

Constraints

The rules or limitations within a logic puzzle, such as each item having only one match, which help guide the solver's deductions.

Last One Standing Rule

The principle that if all options in a row or column of a grid puzzle have been eliminated except one, that remaining option must be the correct match.

Grid Puzzles: The Basics

What they are

Grid puzzles are a popular format where you're given several categories of items (e.g., friends, pets, colors) and clues to figure out how they match up.
