Important Terms (1/2)

Impossible

An event that cannot happen; there is zero chance.

Unlikely

An event that could happen, but probably won't.

Likely

An event that will probably happen, but is not guaranteed.

Certain

An event that will definitely happen; there is no doubt.

Possible

Describes an event that *could* happen; there is some chance, however small.

Important Terms (2/2)

Probable

Describes an event that *likely will* happen; the chances are good.

Anecdote

A single story or personal experience, often vivid and memorable, but not representative of general truth.

Base Rate

How common something is in the overall population; the "background" probability before any other information is known.

The Probability Scale

Impossible

This means something literally cannot occur. There is no chance whatsoever.

Unlikely

This describes events that could happen but are not expected to. The odds are against it.

Likely

This applies to events that are expected to happen, though there's still a small chance they won't. The odds are in favor of it.

Certain

This describes an event that will definitely happen, with no possibility of failure.

Possible vs. Probable

Possible

An event is possible if there is *any* chance it could occur. It signifies that it's not impossible.

Probable

An event is probable if it is *likely* to occur. The chances are good.

Lack of Representativeness

One person's experience does not reflect what is true for the broader population.
