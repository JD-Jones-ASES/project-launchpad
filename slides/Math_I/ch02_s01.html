Key Concepts (1/2)

Need for Integers

Integers are necessary to represent quantities that are less than zero, such as temperatures below freezing, financial losses, or depths below sea level.

Definition of Integers

The set of integers includes all whole numbers ($0, 1, 2, 3, ...$) and their opposites ($-1, -2, -3, ...$). This set extends infinitely in both positive and negative directions.

Important Terms

Integer

The set of whole numbers and their opposites. This includes ..., -4, -3, -2, -1, 0, 1, 2, 3, 4, ...

Positive Integers

Integers that are greater than zero (e.g., 1, 2, 3). They are usually written without a sign.

Negative Integers

Integers that are less than zero (e.g., -1, -2, -3). They are always written with a negative sign.

Number Line

A straight line on which every point matches a number. It is used to visualize and compare numbers.

Negative Integers

These are integers less than zero, such as -1, -2, -3, and so on. A negative sign (-) must always be written in front of a negative integer.

Zero

The number 0 is unique. It is the integer that separates the positive integers from the negative integers, but it is neither positive nor negative itself.
