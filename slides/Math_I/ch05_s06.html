Understanding Rational Numbers

Fractions

Like $\frac{1}{2}$, $\frac{3}{4}$, etc.

Terminating Decimals

Decimals that end, like $0.5$, $0.25$, $0.75$. These can be written as fractions (e.g., $0.5 = \frac{1}{2}$).

Repeating Decimals

Decimals that have a pattern of digits that repeats infinitely, like $0.333...$ or $0.\overline{3}$ (which is $\frac{1}{3}$).

Integers

Whole numbers, including zero and negative whole numbers, like $-3, 0, 5$. These can be written as fractions (e.g., $5 = \frac{5}{1}$).

Negatives of all the above

Such as $-\frac{1}{2}$, $-0.75$, $-0.\overline{3}$, $-5$.

Why Decimals are Preferred

• You can easily line up decimal points.

• Comparing digits by place value (tenths, hundredths, thousandths, etc.) becomes straightforward.

Summary of Comparing and Ordering Rational Numbers

• Always convert all numbers to the same form (decimals are usually best) before comparing.

• For positive rational numbers, compare them digit by digit after aligning decimal points.

• For negative rational numbers, the number with the greater absolute value is the lesser number (it's farther left on the number line).

• Visualizing numbers on a number line can be very helpful, especially when dealing with both positive and negative rational numbers.

Key Concepts (2/2)

Compare $5/8$ and $0.62$. Use $<$, $>$, or $=$.

$5/8 = 0.625$. Since $0.625 > 0.620$, then $5/8 > 0.62$.

Order the following rational numbers from least to greatest: $3/4$, $0.7$, $4/5$.

Convert to decimals: $3/4 = 0.75$, $0.7 = 0.70$, $4/5 = 0.80$. Comparing gives $0.70 < 0.75 < 0.80$, so the order is $0.7 < 3/4 < 4/5$.

Compare $-0.4$ and $-3/8$. Use $<$, $>$, or $=$.

Convert $-3/8$ to a decimal: $-3/8 = -0.375$. On the number line, $-0.4$ is to the left of $-0.375$. Therefore, $-0.4 < -3/8$.

Explain the relationship between a negative number's absolute value and its actual value when comparing.

For negative numbers, the number with the larger absolute value is actually the lesser number. For example, $|-0.5| > |-0.3|$, but $-0.5 < -0.3$.

Store A sells juice for $3/4$ of a dollar per bottle. Store B sells the same juice for $0.72 per bottle. Which store has the lower price?

Convert $3/4$ to a decimal: $3/4 = 0.75$. Compare $0.75$ and $0.72$. Since $0.72 < 0.75$, Store B has the lower price.

Can an integer like $-7$ be considered a rational number? Explain why or why not.

Yes, an integer like $-7$ is a rational number because it can be written as a quotient of two integers: $-7/1$.
