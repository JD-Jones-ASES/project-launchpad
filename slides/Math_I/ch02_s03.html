Adding and Subtracting Integers

Study Slides

Key Concepts

Adding Integers with the Same Sign

When adding two integers that are both positive or both negative, you add their absolute values. The result will carry the shared sign (positive if both were positive, negative if both were negative).

Adding Integers with Different Signs

When adding two integers, one positive and one negative, you subtract the smaller absolute value from the larger absolute value. The result will take the sign of the integer that had the larger abs...

Adding Zero Property

Adding zero to any integer leaves the integer unchanged ($a + 0 = a$).

Subtracting Integers

To subtract an integer, you add its opposite. This transforms the operation $a - b$ into $a + (-b)$. This is a crucial rule for simplifying subtraction problems.

Two Negatives in a Row

A special case in subtraction where $a - (-b)$ simplifies to $a + b$. Subtracting a negative number is equivalent to adding a positive number.

Summary of Adding and Subtracting Integers

Same sign addition

Add absolute values, keep the shared sign.

Different sign addition

Subtract absolute values, use the sign of the number with the larger absolute value.

Subtraction

Add the opposite ($a - b = a + (-b)$).

Two negatives in a row

$a - (-b) = a + b$.
