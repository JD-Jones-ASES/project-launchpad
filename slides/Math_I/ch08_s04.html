Solving One-Step Equations (Addition and Subtraction) Study Guide

Study Slides

Key Concepts (1/2)

Equation

A mathematical statement asserting that two expressions are equal. It always contains an equal sign ($=$).

Solution of an Equation

A specific value for the variable that, when substituted into the equation, makes the statement true. Solving an equation means finding this value.

Inverse Operations

Operations that "undo" each other. Addition and subtraction are inverse operations, meaning one can reverse the effect of the other.

Addition Property of Equality

This property states that if you add the same number to both sides of a true equation, the equation remains true. (If $a = b$, then $a + c = b + c$).

Subtraction Property of Equality

This property states that if you subtract the same number from both sides of a true equation, the equation remains true. (If $a = b$, then $a - c = b - c$).

Key Concepts (2/2)

Isolating the Variable

The goal when solving an equation is to get the variable by itself on one side of the equation. This is achieved by applying inverse operations.

Checking Solutions

After finding a potential solution, it's crucial to substitute it back into the original equation to verify that it makes the equation true.

Important Terms

Equation

A mathematical sentence that uses an equal sign ($=$) to show that two expressions have the same value. Example: $x + 5 = 12$.

Solution of an Equation

A value of the variable that makes the equation true. For $x + 5 = 12$, the solution is $x = 7$ because $7 + 5 = 12$.

Inverse Operations

Operations that "undo" each other. For example, addition is the inverse of subtraction, and subtraction is the inverse of addition.

Variable

A symbol, usually a letter (like $x$, $n$, $y$, $b$, etc.), used to represent an unknown number or quantity in a mathematical expression or equation.

Isolate the Variable

The process of performing inverse operations on both sides of an equation until the variable is alone on one side, thus revealing its value.

Key Concepts (1/2)

What is an equation?

An equation is a mathematical sentence that uses an equal sign (=) to show that two expressions have the same value.

Give an example of an equation involving a variable.

$x + 5 = 12$

What is a solution of an equation?

A solution of an equation is a value of the variable that makes the equation true.

For the equation $x + 5 = 12$, what is its solution?

$x = 7$, because $7 + 5 = 12$.

What does it mean to 'solve an equation'?

Solving an equation means finding the value of the variable that makes the equation true.

What are inverse operations?

Inverse operations are operations that 'undo' each other.

Key Concepts (2/2)

What operation is the inverse of addition?

Subtraction.

What operation is the inverse of subtraction?

Addition.

State the Addition Property of Equality.

If $a = b$, then $a + c = b + c$. You can add the same number to both sides of an equation.

State the Subtraction Property of Equality.

If $a = b$, then $a - c = b - c$. You can subtract the same number from both sides of an equation.

What is the key principle to remember when performing an operation on one side of an equation?

Whatever you do to one side of an equation, you must do to the other side to maintain equality.

How do you solve an equation of the form $x + a = b$?

Subtract $a$ from both sides ($x = b - a$) to isolate the variable.
