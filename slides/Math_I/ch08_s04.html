Solving One-Step Equations (Addition and Subtraction) Study Guide

Study Slides

Key Concepts (1/2)

Equation

A mathematical statement asserting that two expressions are equal, identified by an equal sign (=).

Solution of an Equation

The specific value of the variable that makes the equation a true statement. Finding this value is the goal of solving an equation.

Inverse Operations

Operations that "undo" each other. Addition and subtraction are inverse operations, meaning one can cancel out the effect of the other.

Addition Property of Equality

States that if you add the same number to both sides of a true equation, the equation remains true. (If $a = b$, then $a + c = b + c$).

Subtraction Property of Equality

States that if you subtract the same number from both sides of a true equation, the equation remains true. (If $a = b$, then $a - c = b - c$).

Key Concepts (2/2)

Isolating the Variable

The process of manipulating an equation to get the variable by itself on one side of the equal sign. This is achieved using inverse operations.

Checking Solutions

Always substitute your found solution back into the *original* equation to verify that it makes the equation true.

The Power of Inverse Operations

Addition and Subtraction are inverse operations.

If you add a number, you can undo it by subtracting that same number. If you subtract a number, you can undo it by adding that same number.

Multiplication and Division are also inverse operations

(covered in a later section).

Maintaining Balance: Properties of Equality

Addition Property of Equality

If $a = b$, then $a + c = b + c$. This means if you have an equation and you add the same number $c$ to both sides, the equation remains true.

Subtraction Property of Equality

If $a = b$, then $a - c = b - c$. Similarly, if you subtract the same number $c$ from both sides, the equation remains true.

Summary for Solving One-Step Equations (Addition/Subtraction)

For $x + a = b$

Subtract $a$ from both sides to find $x = b - a$.

For $x - a = b$

Add $a$ to both sides to find $x = b + a$.

Always check your solution

by plugging it back into the original equation to ensure it yields a true statement.
