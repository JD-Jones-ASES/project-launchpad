Data Displays

Study Slides

Key Concepts (1/2)

Data Displays Aid Understanding

Data displays are visual tools that organize and present data, making complex information easier to read, understand, and interpret.

Four Main Types of Displays

The section focuses on four fundamental types: Bar Graphs, Circle Graphs (Pie Charts), Line Graphs, and Dot Plots.

Purpose-Driven Choice

Each data display is best suited for presenting a specific *type* of data or answering a particular *kind* of question. Choosing the right display is crucial for effective communication.

Bar Graphs for Category Comparison

Use bar graphs to compare distinct amounts across different categories.

Circle Graphs for Parts of a Whole

Use circle graphs to show how different parts contribute to a total, typically expressed as percentages. The sections must sum to 100% and not overlap.

Key Concepts (2/2)

Line Graphs for Trends Over Time

Use line graphs to illustrate how data changes or trends over a continuous period.

Dot Plots for Frequency of Individual Values

Use dot plots to show the frequency of individual data values on a number line, revealing data distribution.

Beware of Misleading Graphs

Be cautious when reading bar graphs where the vertical axis does not start at zero, as this can exaggerate differences and misrepresent the data.

Important Terms (1/2)

Data

Information collected about people, events, or things.

Data Display

A visual way to organize and present data so that it is easier to read and understand.

Bar Graph

A data display that uses horizontal or vertical bars to compare amounts across different categories. The length or height of each bar represents the value for that category.

Circle Graph (Pie Chart)

A data display that is a circle divided into sections, where each section represents a part of the whole, and all sections together equal 100% of the data.

Line Graph

A data display that uses points connected by line segments to show how data changes over time, typically with time on the horizontal axis.

Important Terms (2/2)

Dot Plot

A data display that uses a number line with dots stacked above each value to show how often that value occurs in a data set (its frequency).

Range

The difference between the highest and lowest values in a data set. (Learned from Dot Plot example).

What are Data Displays?

is simply information gathered from various sources.

Choosing the Right Display: A Summary

Bar Graph

To compare amounts across different, distinct categories.

Circle Graph

To show how parts contribute to a whole (percentages of a total).

Line Graph

To show change or trends over time.

Dot Plot

To display individual data values and their frequency or distribution on a number line.
