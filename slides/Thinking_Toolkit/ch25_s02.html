Russell's Paradox — Study Guide

Study Slides

Key Concepts (1/2)

Naive Set Theory

The intuitive view that any clearly defined property generates a legitimate set — the foundation that Russell's Paradox destroyed

Russell's Paradox

The contradiction arising from defining the set R of all sets that do not contain themselves

Axiom of Unrestricted Comprehension

The naive principle that for any property P, a set of all objects satisfying P exists — proven inconsistent by Russell

Axiomatic Set Theory

The rigorous replacement for naive set theory, using carefully restricted axioms to avoid paradox

Zermelo-Fraenkel Set Theory (ZF/ZFC)

The standard axiomatic foundation for modern mathematics, developed to resolve Russell's Paradox

Key Concepts (2/2)

Self-Membership

The property of a set containing itself as one of its own elements — the concept at the heart of the paradox

Frege's Grundgesetze

The foundational work whose second volume was devastated when Russell communicated the paradox just before publication

Important Terms (1/2)

Russell's Paradox

Naive set theory

The pre-paradox assumption that any well-defined property defines a legitimate set — the Axiom of Unrestricted Comprehension

Axiom of Unrestricted Comprehension

The axiom stating that for any property P(x), the collection {x : P(x)} is a set; proven inconsistent by Russell

Zermelo-Fraenkel set theory (ZF/ZFC)

The axiomatic system replacing naive set theory, which avoids Russell's Paradox by restricting set formation to subsets of existing sets

Axiom of Separation

A ZF axiom permitting formation of subsets of existing sets using a property, but not creation of arbitrary new sets from scratch

Important Terms (2/2)

Gottlob Frege

German logician whose life's work building arithmetic on logical foundations was shattered when Russell communicated the paradox to him in 1902

Bertrand Russell

British philosopher and logician (1872–1970) who discovered the paradox in 1901; later co-authored Principia Mathematica with Whitehead; Nobel laureate in literature

Explain the contradiction if the set R (of all sets that do not contain themselves) does contain itself.

If R contains itself, then by its definition (as a set of sets that do not contain themselves), it must not contain itself. This is a contradiction.
