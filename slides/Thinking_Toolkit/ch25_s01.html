The Liar Paradox — Study Guide

Study Slides

Key Concepts (1/2)

Self-Reference

A statement that refers to itself, creating circular logical dependencies that resist stable truth-value assignment

The Liar Sentence

"This sentence is false" — the core paradox that cannot be consistently assigned a truth value

Logical Oscillation

When a statement's truth value flips endlessly between true and false without settling on either

Tarski's Hierarchy

The proposed solution that a language cannot contain its own truth predicate — truth must be defined in a higher metalanguage

Paraconsistent Logic

Logical systems designed to tolerate localized contradictions without the system collapsing entirely

Key Concepts (2/2)

Dialethism

The philosophical position that some contradictions are genuinely true, championed by Graham Priest

Ex Falso Quodlibet

The classical logic principle that a single contradiction allows proving anything at all

Important Terms (1/2)

Liar Paradox

The logical puzzle arising from the sentence "This sentence is false," which cannot be consistently assigned either truth value

Epimenides Paradox

A weaker variant in which the Cretan philosopher Epimenides said "All Cretans are liars" — not strictly paradoxical because he could simply be lying while some other Cretans tell the truth

Eubulides of Miletus

4th-century BC logician credited with formulating the Liar Paradox; attributed to him and discussed for over 2,300 years

Alfred Tarski

Polish logician who proposed in the 1930s that a language cannot consistently contain its own truth predicate, requiring a metalanguage hierarchy

Metalanguage

A language used to discuss another language (the object language); in Tarski's framework, truth predicates belong to metalanguages, not object languages

Important Terms (2/2)

Paraconsistent logic

Logical systems that block ex falso quodlibet, allowing localized contradictions to coexist with otherwise sound reasoning

Dialethism

The view associated with Graham Priest that some sentences are genuinely both true and false simultaneously
