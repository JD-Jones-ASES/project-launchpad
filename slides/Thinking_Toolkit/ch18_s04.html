The Antidote: Reference Class Forecasting (The Outside View)

Shift in Question

Instead of asking, "How long will *this specific project* take, given its unique features?", the approach shifts to, "How long did *similar projects* actually take?"

Base Rate Adjustment

Estimates are then anchored to this historical base rate of similar projects, and only adjusted for factors that genuinely distinguish the current project from the reference class.
