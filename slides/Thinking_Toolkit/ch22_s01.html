Feedback Loops — Reinforcing and Balancing

Study Slides

Key Concepts

Reinforcing (positive) feedback loop

— A process where output circles back to intensify the input, producing exponential growth or exponential collapse; "positive" means amplifying, not good

Balancing (negative) feedback loop

— A process where output circles back to counteract the input, producing stability, equilibrium, or oscillation around a set point

Feedback

— The fundamental mechanism by which the output of a process loops back to influence its own input

Dominance

— At any given moment, either reinforcing or balancing loops may dominate a system, determining whether it accelerates, stabilizes, or oscillates

Echo chamber

— A real-world reinforcing loop where algorithmic content delivery amplifies engagement in one direction without a built-in balancing mechanism

Important Terms

Systems thinking

An approach to analysis that focuses on the way system components interrelate and work over time within a larger context

Cybernetics

The formal study of feedback and control systems, originating with Norbert Wiener's 1948 work; the intellectual origin of feedback loop analysis

Set point

The target value a balancing loop maintains (e.g., the temperature a thermostat aims for)

Exponential growth

Growth produced by a reinforcing loop where each gain produces proportionally more gain

Equilibrium

The stable state maintained by a balancing loop, which the system returns to after disturbances
