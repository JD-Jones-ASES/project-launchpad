Key Concepts

Commands vs. Polite Requests

Three Main Structures for Polite Requests

Context Dictates Formality

The choice between `¿Puedes?` and `¿Podrías?` (or a direct command) depends heavily on the relationship with the person being addressed and the social context.

Stem-Changing Verbs and Commands

A key reminder that if a verb stem-changes in the *él/ella* present tense form, its *tú* command will also stem-change. This rule simplifies remembering command forms for these verbs.

Useful Verbs for Requests

Familiarization with verbs like *abrir* (to open), *cerrar* (to close), and *venir* (to come) in their command and request forms is emphasized for common household and classroom interactions.

Key Structures for Making Polite Requests

`¿Puedes + infinitive...?` (Can you...?)

`¿Podrías + infinitive...?` (Could you...?)

`Por favor + tú command` (or `tú command + por favor`)

More Useful Verbs for Commands and Requests

Verb (Infinitive). tú Command

Cerrar (to close). ¡Cierra!

Venir (to come). ¡Ven!

Practical Application: Classroom Rules

Conversation Analysis: ¿Me ayudas, por favor?

"Tomás, ¿puedes pasar los colores?"

Camila uses `¿Puedes...?` for a casual request to a classmate.

"¿Me ayudas con las instrucciones?"

Tomás also uses `¿Puedes...?` (implied `¿Puedes ayudarme...?`) for a peer-to-peer request.

"Rodrigo, ¡cierra la puerta, por favor!"
