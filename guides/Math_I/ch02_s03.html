Study Guide: Adding and Subtracting Integers

Overview

This section introduces the fundamental rules for adding and subtracting integers, which include both positive and negative whole numbers. It provides clear, step-by-step methods for handling various combinations of signs, ensuring you can confidently solve problems like $(-7) + 3$ or $4 - (-5)$. Mastering these rules is crucial for all future mathematical operations involving integers.

Key Concepts

  • Adding Integers with the Same Sign:
    • Rule: Add their absolute values. The result keeps the shared sign.
    • Example: $5 + 8 = 13$; $(-4) + (-6) = -10$.
  • Adding Integers with Different Signs:
    • Rule: Subtract the smaller absolute value from the larger absolute value. The result takes the sign of the number that has the larger absolute value.
    • Example: $7 + (-3) = 4$ (because $|7| > |-3|$ and $7$ is positive); $(-9) + 2 = -7$ (because $|-9| > |2|$ and $-9$ is negative).
  • Additive Identity Property (Adding Zero):
    • Rule: Adding zero to any integer leaves the integer unchanged: $a + 0 = a$.
  • Subtracting Integers:
    • Rule: To subtract an integer, you add its opposite: $a - b = a + (-b)$.
    • Example: $6 - 9 = 6 + (-9) = -3$.
  • Subtracting a Negative (Two Negatives in a Row):
    • Rule: When you see two negative signs directly next to each other in a subtraction problem, they become a positive: $a - (-b) = a + b$.
    • Example: $5 - (-3) = 5 + 3 = 8$. This can be thought of as "subtracting a debt is the same as gaining money."
  • Number-Line Model: A visual tool to understand integer addition, where you start at the first number and move right for positive additions and left for negative additions.

Important Terms

  • Integers: The set of whole numbers and their opposites. This includes positive whole numbers ($1, 2, 3, \ldots$), negative whole numbers ($\ldots, -3, -2, -1$), and zero ($0$).
  • Absolute Value: The distance a number is from zero on the number line. It is always a non-negative value. For example, the absolute value of $-5$ is $5$, written as $|-5| = 5$. The absolute value of $5$ is also $5$, written as $|5| = 5$.
  • Opposite (or Additive Inverse): For any integer $a$, its opposite is $-a$. When an integer and its opposite are added together, the sum is zero (e.g., $5 + (-5) = 0$).

Detailed Notes

Introduction to Adding and Subtracting Integers

While you're familiar with adding and subtracting positive numbers, working with negative numbers requires specific rules. This section provides those rules for adding and subtracting any combination of positive and negative integers.

Adding Integers

  1. Integers with the Same Sign:

    • When both numbers are positive, you just add them as usual and the sum is positive (e.g., $5 + 8 = 13$).
    • When both numbers are negative, you first add their absolute values. Then, the sum will always be negative. For instance, to add $(-4) + (-6)$:
      • Find the absolute values: $|-4| = 4$ and $|-6| = 6$.
      • Add the absolute values: $4 + 6 = 10$.
      • Since both original numbers were negative, the result is negative: $-10$.

  2. Integers with Different Signs:

    • When one number is positive and the other is negative, the process is slightly different.
    • First, find the absolute value of each number.
    • Subtract the smaller absolute value from the larger absolute value.
    • The result will take the sign of the number that had the larger absolute value.
    • Example 1: $7 + (-3)$
      • Absolute values: $|7| = 7$, $|-3| = 3$.
      • Subtract smaller from larger: $7 - 3 = 4$.
      • The number with the larger absolute value is $7$ (which is positive), so the result is positive: $4$.
    • Example 2: $(-9) + 2$
      • Absolute values: $|-9| = 9$, $|2| = 2$.
      • Subtract smaller from larger: $9 - 2 = 7$.
      • The number with the larger absolute value is $-9$ (which is negative), so the result is negative: $-7$.

  3. Adding Zero:

    • Adding zero to any integer does not change its value. This is known as the Additive Identity Property. For any integer $a$, $a + 0 = a$.

Worked Examples for Adding Integers:

  • $(-8) + (-5)$: Both negative (same sign). Add absolute values ($8+5=13$). Keep the negative sign. Result: $\textbf{-13}$.
  • $(-12) + 7$: Different signs. Subtract absolute values ($12-7=5$). The number with the larger absolute value is $-12$ (negative). Result: $\textbf{-5}$.
  • $9 + (-15)$: Different signs. Subtract absolute values ($15-9=6$). The number with the larger absolute value is $-15$ (negative). Result: $\textbf{-6}$.

Subtracting Integers

The core rule for subtraction is to transform it into an addition problem: "To subtract an integer, add its opposite." This means that $a - b$ becomes $a + (-b)$.

  • Example 1: $3 - 10$

    • Change to addition of the opposite: $3 + (-10)$.
    • Now apply addition rules (different signs): Subtract $10 - 3 = 7$. The number with the larger absolute value is $-10$ (negative).
    • Result: $\textbf{-7}$.

  • Example 2: $-4 - 6$

    • Change to addition of the opposite: $-4 + (-6)$.
    • Now apply addition rules (same sign): Add absolute values $4+6=10$. Keep the negative sign.
    • Result: $\textbf{-10}$.

  • Caution: Two Negative Signs in a Row:

    • A very important rule: When you subtract a negative number, it's equivalent to adding a positive number.
    • $a - (-b) = a + b$.
    • Think of it like this: "Subtracting a debt" (a negative quantity) means your overall amount of money increases (you "gain" money).
    • Example: $5 - (-3) = 5 + 3 = 8$.

  • Example 3: $-7 - (-2)$

    • Notice the two negative signs in a row. They become a positive: $-7 + 2$.
    • Now apply addition rules (different signs): Subtract $7 - 2 = 5$. The number with the larger absolute value is $-7$ (negative).
    • Result: $\textbf{-5}$.

Number-Line Model

A number line can help visualize addition of integers:

  • Start at the first integer.
  • If you are adding a positive number, move to the right.
  • If you are adding a negative number, move to the left.
  • Example: To show $(-2) + 5$:
    1. Start at $-2$ on the number line.
    2. Add $5$ (a positive number), so move $5$ units to the right.
    3. You land on $3$. Therefore, $(-2) + 5 = 3$.

Real-World Applications

Integers are used to represent quantities like money (deposits/withdrawals, debt), temperature changes, altitude, and more.

  • Checking Account Example: Mia's account has a balance of $-$35. She deposits $50.

    • This is an addition problem: $-35 + 50$.
    • Using different signs addition rules: $50 - 35 = 15$. Since $50$ has the larger absolute value and is positive, the result is positive.
    • Her new balance is $\textbf{15}$.

  • Temperature Example: The temperature is $-12$C and falls $8$ more degrees.

    • "Falls $8$ more degrees" means subtracting $8$ or adding $-8$: $-12 - 8$ or $-12 + (-8)$.
    • Using same signs addition rules: Add absolute values $12 + 8 = 20$. Keep the negative sign.
    • The new temperature is $\textbf{-20}$C.

Adding and Subtracting Integers Summary (Key Takeaways)

  • Same sign: Add absolute values, keep the sign.
  • Different signs: Subtract absolute values, use the sign of the number with the larger absolute value.
  • Subtract: Add the opposite ($a - b = a + (-b)$).
  • Two negatives in a row: Become a positive ($a - (-b) = a + b$).

  2. How does the concept of "adding the opposite" simplify subtraction of integers? Why is it useful to convert subtraction problems into addition problems?
  3. Describe a real-life scenario where you might encounter the operation $a - (-b) = a + b$. Explain why "subtracting a negative" makes sense as "adding a positive" in your scenario.
