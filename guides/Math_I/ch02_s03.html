Study Guide: Adding and Subtracting Integers

Overview

This section provides clear, actionable rules for adding and subtracting any integers, including positive and negative numbers. You will learn specific methods for combining integers based on whether they share the same sign or have different signs, and a fundamental rule for transforming subtraction problems into addition. Practical examples, including number line models and real-world scenarios, are used to illustrate these core mathematical operations.

Key Concepts

  • Adding Integers with the Same Sign: When adding two integers that are both positive or both negative, you add their absolute values. The result will carry the shared sign (positive if both were positive, negative if both were negative).
  • Adding Integers with Different Signs: When adding two integers, one positive and one negative, you subtract the smaller absolute value from the larger absolute value. The result will take the sign of the integer that had the larger absolute value.
  • Adding Zero Property: Adding zero to any integer leaves the integer unchanged ($a + 0 = a$).
  • Subtracting Integers: To subtract an integer, you add its opposite. This transforms the operation $a - b$ into $a + (-b)$. This is a crucial rule for simplifying subtraction problems.
  • Two Negatives in a Row: A special case in subtraction where $a - (-b)$ simplifies to $a + b$. Subtracting a negative number is equivalent to adding a positive number.

Important Terms

  • Integer: A whole number (not a fraction or decimal) that can be positive, negative, or zero. Examples include ..., -3, -2, -1, 0, 1, 2, 3, ...
  • Absolute Value: The distance of a number from zero on the number line. Absolute value is always a non-negative value. It is denoted by vertical bars around the number, e.g., $|-5| = 5$ and $|5| = 5$.
  • Opposite (of an integer): The number that is the same distance from zero on the number line but on the opposite side. For example, the opposite of $5$ is $-5$, and the opposite of $-3$ is $3$.

Detailed Notes

Introduction to Adding and Subtracting Integers

While you're familiar with adding and subtracting positive numbers (like $5+8$), this section extends these operations to include negative numbers. Understanding how to handle signs is crucial for accurate calculations with integers.

Adding Integers

There are two main rules for adding integers, depending on their signs:

  1. If two integers have the same sign:

    • Rule: Add their absolute values.
    • Result Sign: The sum will have the same sign as the original integers.
    • Examples:
      • $5 + 8 = 13$ (Both are positive. Add $5+8=13$. The result is positive.)
      • $(-4) + (-6) = -10$ (Both are negative. Add absolute values: $|-4| + |-6| = 4 + 6 = 10$. Since both were negative, the result is negative.)

  2. If two integers have different signs:

    • Rule: Subtract the smaller absolute value from the larger absolute value.
    • Result Sign: The sum will have the sign of the integer with the larger absolute value.
    • Examples:
      • $7 + (-3) = 4$ (Absolute values are $7$ and $3$. Subtract $7-3=4$. Since $7$ has a larger absolute value and is positive, the result is positive.)
      • $(-9) + 2 = -7$ (Absolute values are $9$ and $2$. Subtract $9-2=7$. Since $-9$ has a larger absolute value and is negative, the result is negative.)
  • Adding Zero Property: Remember that adding zero to any integer does not change its value: $a + 0 = a$.

Example: Adding Integers Let's find each sum using the rules:

  • $(-8) + (-5)$:
    • Both negative (same sign): Add absolute values: $8 + 5 = 13$.
    • Keep the negative sign: $(-8) + (-5) = -13$.
  • $(-12) + 7$:
    • Different signs: Subtract absolute values: $12 - 7 = 5$.
    • Sign of larger absolute value: The integer with the larger absolute value is $-12$ (absolute value is $12$), which is negative. So, the result is $-5$.
  • $9 + (-15)$:
    • Different signs: Subtract absolute values: $15 - 9 = 6$.
    • Sign of larger absolute value: The integer with the larger absolute value is $-15$ (absolute value is $15$), which is negative. So, the result is $-6$.

Subtracting Integers

The key to subtracting integers is to transform the subtraction problem into an addition problem:

  • Rule: To subtract an integer, add its opposite.

    • This means that $a - b = a + (-b)$.

  • Example: $6 - 9$

    • Change to addition of the opposite: $6 + (-9)$.
    • Now, use the addition rule for different signs: Subtract absolute values ($9-6=3$). The number with the larger absolute value is $9$ (from $-9$), which is negative. So, the result is $-3$.

  • Caution: Two Negative Signs in a Row!

    • When you see a subtraction of a negative number, like $a - (-b)$, it simplifies to $a + b$.
    • Think of it as "subtracting a debt is the same as gaining money."
    • Example: $5 - (-3) = 5 + 3 = 8$.

Example: Subtracting Integers Let's find each difference by changing subtraction to addition of the opposite:

  • $3 - 10$:
    • Change to $3 + (-10)$.
    • Now, apply addition rules for different signs: $10 - 3 = 7$. The larger absolute value is $10$ (from $-10$), which is negative. So, the result is $-7$.
  • $-4 - 6$:
    • Change to $-4 + (-6)$.
    • Now, apply addition rules for same signs: $4 + 6 = 10$. Both were negative, so the result is $-10$.
  • $-7 - (-2)$:
    • Change to $-7 + 2$. (Two negatives become a positive!)
    • Now, apply addition rules for different signs: $7 - 2 = 5$. The larger absolute value is $7$ (from $-7$), which is negative. So, the result is $-5$.

Number-Line Model

A number line can help visualize addition of integers:

  • To add $A + B$: Start at the first number ($A$) on the number line.
    • If $B$ is positive, move $B$ units to the right.
    • If $B$ is negative, move $B$ units to the left.
  • Example: Show $(-2) + 5$
    • Start at $-2$.
    • Adding $5$ means move $5$ units to the right.
    • You land on $3$. So, $(-2) + 5 = 3$.

Real-World Application: Checking Account

Integer operations are common in finance:

  • Problem: Mia's checking account balance is $-$35$ (overdrawn). She deposits $$50$. What is her new balance?
  • Setup: This is an addition problem: $-35 + 50$.
  • Solve: Using the rule for adding integers with different signs:
    • Subtract absolute values: $50 - 35 = 15$.
    • The number with the larger absolute value ($50$) is positive, so the result is positive.
    • $-35 + 50 = 15$.
  • Answer: Mia's new balance is $$15$.

Summary of Adding and Subtracting Integers

  • Same sign addition: Add absolute values, keep the shared sign.
  • Different sign addition: Subtract absolute values, use the sign of the number with the larger absolute value.
  • Subtraction: Add the opposite ($a - b = a + (-b)$).
  • Two negatives in a row: $a - (-b) = a + b$.

Review Questions

  1. Explain in your own words the difference between adding integers with the same sign and adding integers with different signs. Provide an example for each case, showing the steps involved.
  2. What is the fundamental rule for subtracting integers? How does this rule simplify the process of subtraction, and why is it useful to convert subtraction problems into addition problems?
  3. Describe the special rule for when two negative signs appear in a row (e.g., $a - (-b)$). Can you think of a real-world scenario, other than a checking account, where "subtracting a negative" could represent an increase?
  4. Using a number line, demonstrate how to solve the problem $(-6) + 4$. Clearly explain your starting point, direction, and final position.
  5. The temperature in a city was $-5^\circ$C at dawn. By noon, it had risen $12^\circ$C. In the evening, it fell $7^\circ$C. Write a single mathematical expression to represent these changes and calculate the final temperature.
