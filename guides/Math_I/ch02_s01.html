Integers and the Number Line Study Guide

Overview

This section introduces integers, expanding beyond whole numbers to include values less than zero. It explains how to represent these numbers and visualize them using a number line, which is a fundamental tool for understanding their order and magnitude. The chapter also covers how to compare, order, and apply integers to real-world situations.

Key Concepts

  • Integers: The set of whole numbers (0, 1, 2, 3...) and their opposites (-1, -2, -3...). They extend infinitely in both positive and negative directions.
  • Positive Integers: Numbers greater than zero (1, 2, 3, ...). A positive sign (+) is usually not written.
  • Negative Integers: Numbers less than zero (-1, -2, -3, ...). They are always written with a negative sign (-).
  • Zero: Zero is neither positive nor negative. It acts as the central point on the number line.
  • Number Line: A straight line where every point corresponds to a unique number. It visually organizes integers, with positive numbers to the right of zero and negative numbers to the left.
  • Comparing Integers: On a number line, numbers increase in value from left to right. Therefore, the number farther to the right is greater, and the number farther to the left is less.
  • Ordering Integers: To order integers from least to greatest, visualize or plot them on a number line and read them from left to right.
  • Real-World Representation: Integers are used to describe quantities that involve values above/below a reference point, gains/losses, or increases/decreases. "Below," "loss," or "drop" typically indicate negative integers, while "above," "gain," or "increase" indicate positive integers.

Important Terms

  • Integer: The set of whole numbers and their opposites. This set includes ..., -3, -2, -1, 0, 1, 2, 3, ...
  • Positive Integers: Integers that are greater than zero, such as 1, 2, 3, and so on.
  • Negative Integers: Integers that are less than zero, such as -1, -2, -3, and so on.
  • Number Line: A visual representation of numbers as points on a straight line, used to compare and order numbers.

Detailed Notes

Introduction to Integers

You are familiar with counting numbers (1, 2, 3, ...) and the number zero. However, many real-world situations require numbers that represent values less than zero. For example, temperatures can drop below 0 degrees, or a business might experience a financial loss. To describe these situations, we use integers.

The set of integers includes all the positive whole numbers, all the negative whole numbers, and zero. Integers extend infinitely in both directions, meaning there is no largest or smallest integer.

Positive, Negative, and Zero

  • Positive Integers: These are the integers greater than zero (1, 2, 3, ...). When writing positive integers, the plus sign (+) is usually omitted. For example, +5 is simply written as 5.
  • Negative Integers: These are the integers less than zero (-1, -2, -3, ...). Negative integers are always written with a negative sign (-) in front of the number. For example, -4 is read as "negative four." It's important to remember that the negative sign is part of the number itself, not an operation like subtraction when it stands alone.
  • Zero: The integer 0 is unique because it is neither positive nor negative. It serves as the reference point between positive and negative numbers.

The Number Line

A number line is a powerful visual tool for understanding integers. It's a straight line where:

  • Every point corresponds to a unique number.
  • Zero is typically marked in the center.
  • Positive integers are located to the right of zero.
  • Negative integers are located to the left of zero.
  • The numbers on a number line increase in value as you move from left to right.

Plotting Integers on a Number Line

To plot integers, draw a number line, mark zero and a few other integers (like -5 to 5) for scale. Then, locate each specified integer and place a dot at its corresponding position.

Example: Plot -3, 1, and 4 on a number line. You would draw a line, mark 0, then move three units to the left for -3, one unit to the right for 1, and four units to the right for 4, placing a dot at each.

Comparing Integers

The number line makes comparing integers straightforward:

  • The number farther to the right is greater.
  • The number farther to the left is less.

We use the symbols:

  • < (less than): a < b means 'a is to the left of b'
  • > (greater than): a > b means 'a is to the right of b'

Examples:

  • Compare -5 and 2: Since -5 is to the left of 2 on the number line, -5 < 2.
  • Compare 3 and -7: Since 3 is to the right of -7, 3 > -7.
  • Compare -4 and -1: Since -4 is to the left of -1, -4 < -1.

Ordering Integers

To order a set of integers from least to greatest, you can imagine them placed on a number line. Read the numbers from left to right on the number line to get them in ascending order. For ordering from greatest to least, read from right to left.

Example: Order the integers 3, -5, 0, -2, 7 from least to greatest. Mentally place them on a number line: -5 is farthest left, then -2, then 0, then 3, then 7. So the order is: -5, -2, 0, 3, 7.

Representing Real-World Quantities

Integers are excellent for describing situations that involve direction or position relative to a zero point:

  • Below/Above Zero:
    • 200 feet below sea level is represented as -200.
    • 15 degrees below zero is represented as -15.
  • Gain/Loss:
    • A gain of 8 yards is represented as +8 (or simply 8).
    • A loss of $50 would be represented as -50.
  • Other Examples: Bank balances (deposit = positive, withdrawal = negative), floors in a building (above ground = positive, basement = negative), time changes (ahead = positive, behind = negative).

The key is to identify the reference point (often zero) and determine if the quantity indicates a value in the "negative" direction (below, loss, decrease) or the "positive" direction (above, gain, increase).

Review Questions

  1. What are integers, and why are they necessary to describe certain situations that whole numbers alone cannot? Provide two real-world examples where integers would be used.
  2. Explain how a number line helps in understanding the relationship between positive and negative integers. How does the position of a number on the line indicate its value?
  3. Given two integers, -9 and -2, explain in your own words how to determine which one is greater using the concept of a number line.
  4. A scuba diver descends 30 feet below sea level, and a hot air balloon is 150 feet above sea level. Write an integer for each situation and explain what zero represents in this context.
  5. Order the following integers from greatest to least and justify your sequence: 6, -12, 0, 3, -1, -8.
