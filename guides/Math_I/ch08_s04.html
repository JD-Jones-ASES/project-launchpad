Solving One-Step Equations (Addition and Subtraction) Study Guide

Overview

This section introduces the fundamental concepts of equations and their solutions. It focuses on solving one-step linear equations that involve addition or subtraction using the principle of inverse operations and the properties of equality, ensuring that the equation remains balanced while finding the unknown variable.

Key Concepts

  • Equation: A mathematical statement asserting that two expressions are equal. It always contains an equal sign ($=$).
  • Solution of an Equation: A specific value for the variable that, when substituted into the equation, makes the statement true. Solving an equation means finding this value.
  • Inverse Operations: Operations that "undo" each other. Addition and subtraction are inverse operations, meaning one can reverse the effect of the other.
  • Addition Property of Equality: This property states that if you add the same number to both sides of a true equation, the equation remains true. (If $a = b$, then $a + c = b + c$).
  • Subtraction Property of Equality: This property states that if you subtract the same number from both sides of a true equation, the equation remains true. (If $a = b$, then $a - c = b - c$).
  • Isolating the Variable: The goal when solving an equation is to get the variable by itself on one side of the equation. This is achieved by applying inverse operations.
  • Checking Solutions: After finding a potential solution, it's crucial to substitute it back into the original equation to verify that it makes the equation true.

Important Terms

  • Equation: A mathematical sentence that uses an equal sign ($=$) to show that two expressions have the same value. Example: $x + 5 = 12$.
  • Solution of an Equation: A value of the variable that makes the equation true. For $x + 5 = 12$, the solution is $x = 7$ because $7 + 5 = 12$.
  • Inverse Operations: Operations that "undo" each other. For example, addition is the inverse of subtraction, and subtraction is the inverse of addition.
  • Variable: A symbol, usually a letter (like $x$, $n$, $y$, $b$, etc.), used to represent an unknown number or quantity in a mathematical expression or equation.
  • Isolate the Variable: The process of performing inverse operations on both sides of an equation until the variable is alone on one side, thus revealing its value.

Detailed Notes

What is an Equation and Its Solution?

An equation is a fundamental concept in algebra that establishes equality between two mathematical expressions. For example, $x + 5 = 12$ is an equation. The "solution" to an equation is the specific value of the variable that makes the equation true. In our example, if $x=7$, then $7 + 5 = 12$, which is a true statement, so $x=7$ is the solution.

The Role of Inverse Operations

To find the solution to an equation, we use inverse operations. Inverse operations are pairs of operations that cancel each other out.

  • Addition and Subtraction are inverse operations.
  • Multiplication and Division are also inverse operations (though not covered in this section).

Properties of Equality

To solve an equation, we must keep it balanced. This means whatever operation you perform on one side of the equation, you must perform the exact same operation on the other side. This principle is formalized by the properties of equality:

  1. Addition Property of Equality: If you have an equation $a = b$, you can add the same number ($c$) to both sides, and the equality will still hold true: $a + c = b + c$.
  2. Subtraction Property of Equality: Similarly, if you have an equation $a = b$, you can subtract the same number ($c$) from both sides, and the equality will remain true: $a - c = b - c$.

Solving One-Step Equations

The goal is to "isolate" the variable – to get it by itself on one side of the equation.

Type 1: Equations with Addition ($x + a = b$)

If a number is being added to the variable, use the inverse operation: subtraction.

  • Example: Solve $x + 5 = 12$.
    • To isolate $x$, we need to undo the "+ 5". The inverse operation is subtracting 5.
    • Subtract 5 from both sides of the equation: $x + 5 - 5 = 12 - 5$ $x = 7$
    • Check: Substitute $x = 7$ back into the original equation: $7 + 5 = 12$. This is true, so $x = 7$ is the correct solution.

Type 2: Equations with Subtraction ($n - a = b$)

If a number is being subtracted from the variable, use the inverse operation: addition.

  • Example: Solve $n - 8 = 16$.
    • To isolate $n$, we need to undo the "- 8". The inverse operation is adding 8.
    • Add 8 to both sides of the equation: $n - 8 + 8 = 16 + 8$ $n = 24$
    • Check: Substitute $n = 24$ back into the original equation: $24 - 8 = 16$. This is true, so $n = 24$ is the correct solution.

Equations with Fractions and Decimals

The same principles apply when dealing with fractions or decimals.

  • Example 1: Solve $c + 13 = 1$.
    • Subtract 13 from both sides: $c = 1 - 13$.
    • $c = -12$.
    • Check: $-12 + 13 = 1$. (Note: The provided text example has an error, $1-13 \neq 23$. Correct calculation is $1-13 = -12$. I've corrected this in the detailed notes.)
  • Example 2: Solve $w - 2.25 = 3$.
    • Add 2.25 to both sides: $w = 3 + 2.25$.
    • $w = 5.25$.
    • Check: $5.25 - 2.25 = 3$.

Solving Word Problems

Algebraic equations are powerful tools for solving real-world problems.

  • Example (Savings Goal): Jamal has saved some money and then deposited $125 more. He now has $430 in his account. How much did he have before the deposit?
    1. Read: Understand the problem. Jamal added money and now has a total.
    2. Identify: What are you trying to find? Let $b$ = amount before the deposit.
    3. Write: Formulate the equation based on the information. "Amount before" + "deposit" = "total". So, $b + 125 = 430$.
    4. Solve: Use inverse operations. Subtract $125$ from both sides: $b = 430 - 125$ $b = 305$
    5. Check: Substitute the solution back into the original equation: $305 + 125 = 430$. This is true.
    • Answer: Jamal had $305 before the deposit.

Key Reminders

  • Always do the same thing to both sides of the equation. This maintains the balance and equality.
  • Use the inverse operation to "undo" what's being done to the variable.
  • Check your answer by substituting it back into the original equation.

Review Questions

  1. Define "equation" and "solution of an equation." How are inverse operations and the properties of equality used to find a solution?
  2. Explain the difference between the Addition Property of Equality and the Subtraction Property of Equality. Provide a simple example for each.
  3. Solve the equation $y + 17 = 42$. Show your steps and check your solution.
  4. Solve the equation $m - 6.5 = 11.3$. Show your steps and check your solution.
  5. Sarah spent $28 on a new book, and now has $54 left in her wallet. How much money did she have before buying the book? Write an equation to represent the situation and solve it.
