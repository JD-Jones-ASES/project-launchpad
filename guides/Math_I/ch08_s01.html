Study Guide: Variables and Algebraic Expressions

Overview

This section introduces the fundamental building blocks of algebra: variables and algebraic expressions. It explains how letters are used to represent unknown or changing values, and how these variables combine with numbers and operations to form expressions. You'll learn the key vocabulary for describing parts of an expression, understand the difference between expressions and equations, and practice translating real-world scenarios into algebraic expressions.

Key Concepts

  • Variables: Letters or symbols that stand for unknown or changeable numbers. They are crucial for representing quantities whose values can vary.
  • Algebraic Expressions: Combinations of numbers, variables, and mathematical operations (like addition, subtraction, multiplication, and division) that do not include an equal sign.
  • Terms: The individual components within an algebraic expression that are separated by addition (+) or subtraction (-) signs. A term can be a number, a variable, or a product of numbers and variables.
  • Coefficients: The numerical factor that is multiplied by a variable in a term. If a variable appears alone (e.g., x), its coefficient is understood to be 1.
  • Constants: Terms in an algebraic expression that consist only of a number and do not contain any variables. Their value remains fixed.
  • Expression vs. Equation: An expression is a mathematical phrase without an equal sign, while an equation is a mathematical sentence that includes an equal sign, indicating that two expressions are equivalent.
  • Translating Words to Algebra: The ability to convert verbal descriptions of situations into algebraic expressions is a core skill in problem-solving.

Important Terms

  • Variable: A letter or symbol (e.g., $x, y, n, t$) that represents an unknown or changeable numerical value.
  • Algebraic Expression: A mathematical phrase formed by combining numbers, variables, and operation symbols (e.g., $3x + 5$, $7y$).
  • Term: A single number, a single variable, or a product of numbers and variables within an expression. Terms are separated by addition or subtraction signs.
  • Coefficient: The numerical factor that directly multiplies a variable in a term (e.g., in $5x$, the coefficient is $5$).
  • Constant: A term in an algebraic expression that consists only of a number and does not contain a variable (e.g., in $5x + 3$, the constant is $3$).
  • Expression: A mathematical phrase that does not contain an equal sign (e.g., $2x + 7$).
  • Equation: A mathematical statement that shows two expressions are equal by using an equal sign (e.g., $2x + 7 = 15$).

Detailed Notes

From Arithmetic to Algebra

In arithmetic, you primarily work with specific numbers. Algebra extends this by introducing letters, called variables, to represent numbers whose values are unknown or can change. This allows for more general problem-solving and modeling.

What is a Variable?

  • A variable is a placeholder for a number.
  • It's called "variable" because the value it represents can vary or change depending on the situation.
  • Common variables used are $x, y, n, t$, but any letter or symbol can be used.

Understanding Algebraic Expressions

  • An algebraic expression is built by combining numbers, variables, and mathematical operations (+, -, $\times$, $\div$).
  • Examples: $3x + 5$, $n4 - 2$, $7y$.
  • Notice that algebraic expressions do not have an equal sign.

Components of an Algebraic Expression

Let's break down the parts of an expression:

  1. Terms:

    • These are the individual pieces of an expression, separated by addition (+) or subtraction (-) signs.
    • A term can be:
      • A single number (e.g., $3$).
      • A single variable (e.g., $x$).
      • A number multiplied by one or more variables (e.g., $5x$, $2xy^2$).
    • Example: In the expression $5x + 3$:
      • The terms are $5x$ and $3$.

  2. Coefficients:

    • The numerical factor that is multiplied by a variable in a term.
    • Example: In the term $5x$, the coefficient is $5$.
    • Important Note: If a variable appears alone, like $x$ or $y$, its coefficient is understood to be $1$ (e.g., $x$ is the same as $1x$).

  3. Constants:

    • A term that consists only of a number and does not have any variables attached to it.
    • Its value is "constant" because it never changes.
    • Example: In the expression $5x + 3$, the constant is $3$.

Example: Identifying Parts of an Expression For the expression $4n + 9$:

  • The terms are $4n$ and $9$.
  • The coefficient of the variable term ($4n$) is $4$.
  • The constant is $9$.

Expressions vs. Equations

This is a critical distinction:

  • An expression is like a phrase; it does not contain an equal sign.
    • Example: $2x + 7$
  • An equation is like a complete sentence; it always contains an equal sign, showing that two expressions are equivalent.
    • Example: $2x + 7 = 15$

Example: Expression or Equation?

  • $3y - 8$: This is an expression because it has no equal sign.
  • $3y - 8 = 10$: This is an equation because it contains an equal sign.

Checkpoint: Is $7 + 2x$ an expression or an equation?

  • Answer: It is an expression (no equal sign).

Writing Algebraic Expressions from Word Problems

A key skill in algebra is translating verbal descriptions into mathematical expressions.

Example: Items Remaining A store has $250$ items. If $r$ defective items are returned, write an expression for the number of items remaining.

  • Solution: Start with the initial amount and subtract the returned amount: $250 - r$.

Example: Earning Money Maria earns $2 for every hour she helps at home, plus a $3 weekly allowance. Write an expression for her total weekly earnings if she works $h$ hours.

  1. Read: Understand the components: $2 per hour, $3 flat allowance.
  2. Identify: Let $h$ represent the number of hours. The hourly earnings are $2$ times $h$ ($2h$). The allowance is a fixed amount ($3$).
  3. Write: Combine the parts: $2h + 3$.
  4. Check: If $h=5$, total = $2(5) + 3 = 10 + 3 = 13. This makes sense ($10 for hours, $3 allowance).

Checkpoint: Write an expression for the cost of $n$ notebooks at $1.50 each plus a $2 binder.

  • Answer: $1.50n + 2$.

Important Conventions and Cautions

  • Order of Multiplication: When a number is multiplied by a variable, the number (coefficient) is conventionally written first. Write $5x$, not $x5$. Both have the same value, but $5x$ is standard.
  • Multiplication vs. Addition: A common mistake is to confuse multiplication with addition. Placing a number directly next to a variable (e.g., $5x$) means "5 times $x$," not "5 plus $x$." If you mean addition, you must use the plus sign ($5 + x$).

Review Questions

  1. Explain, in your own words, the primary difference between a variable and a constant. Provide an algebraic expression that clearly demonstrates both.
  2. What is an algebraic expression, and how is it fundamentally different from an equation? Give an example of each and briefly explain why your examples fit the definitions.
  3. For the expression $15 - 7y + 2x - 8$:
    • Identify all the terms.
    • State the coefficient for each variable term.
    • Identify any constants.
  4. A cell phone plan costs a monthly fee of $30 plus $0.15 for every text message sent. If a user sends $t$ text messages in a month, write an algebraic expression that represents the total monthly cost of the plan.
  5. The textbook highlights two important conventions/cautions when writing algebraic expressions involving multiplication. Describe both of these and explain why they are important for clarity and avoiding common mistakes.
