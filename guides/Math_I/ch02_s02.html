Chapter 2, Section 2: Absolute Value and Opposites

Overview

This section introduces two fundamental concepts for working with integers: absolute value and opposites. Absolute value describes a number's distance from zero on a number line, always resulting in a non-negative value. Opposites are numbers that are the same distance from zero but lie on opposite sides of the number line. Understanding these concepts is crucial for later mathematical operations and real-world applications involving magnitude and direction.

Key Concepts

  • Absolute Value: The distance of a number from zero on the number line. It is always a non-negative value (zero or positive). The absolute value of a number a is denoted as |a|.
    • Example: |-5| = 5 because -5 is 5 units away from zero. |3| = 3 because 3 is 3 units away from zero.
  • Opposites: Two numbers are opposites if they are the same distance from zero but are located on opposite sides of the number line. The opposite of a number a is written as -a.
    • Example: The opposite of 7 is -7. The opposite of -4 is 4. The opposite of 0 is 0.
  • Absolute Value is Never Negative: Since absolute value represents distance, its value can never be negative. It is always greater than or equal to zero (|a| ≥ 0).
  • Distinguishing Between |a| and -|a|: It's critical to understand that a negative sign outside the absolute value bars is applied after the absolute value is calculated.
    • Example: |-6| = 6 (distance of -6 from zero is 6). But -|6| = -(6) = -6 (absolute value of 6 is 6, then apply the negative sign).

Important Terms

  • Opposite: For any number a, its opposite, denoted -a, is the number on the number line that is the same distance from zero as a but on the opposite side.
  • Absolute Value: For any number a, its absolute value, denoted |a|, is its distance from zero on the number line. It is always a non-negative quantity.

Detailed Notes

Introduction to Absolute Value and Opposites

The number line is a fundamental tool for visualizing numbers. When we consider how "far" a number is from zero, we are looking at its magnitude or size, regardless of whether it's positive or negative. This concept is called absolute value. For instance, both -5 and 5 are 5 units away from zero.

Closely related are opposites. Two numbers are opposites if they have the same distance from zero but are on different sides of the number line. For example, 7 and -7 are opposites.

Understanding Opposites

  • Definition: Two numbers are opposites if they are equally distant from zero but in opposite directions.
  • Notation: The opposite of a number a is denoted by -a.
    • If a is positive, -a is negative (e.g., opposite of 7 is -7).
    • If a is negative, -a is positive (e.g., opposite of -4 is -(-4) = 4).
    • The opposite of 0 is 0 itself.
  • Key Idea: A number and its opposite essentially "cancel each other out" if added together, resulting in zero (e.g., 7 + (-7) = 0).

Understanding Absolute Value

  • Definition: The absolute value of a number is its distance from zero on the number line.
  • Notation: We write the absolute value of a as |a|.
  • Examples:
    • |-5| = 5 (The distance from -5 to 0 is 5 units.)
    • |3| = 3 (The distance from 3 to 0 is 3 units.)
    • |0| = 0 (The distance from 0 to 0 is 0 units.)

Key Property: Absolute Value is Never Negative

Because absolute value represents distance, it cannot be a negative value. Distance is always zero or a positive quantity.

  • Symbolically: For every integer a, |a| ≥ 0.

Caution: Distinguishing |a| from -|a|

This is a common point of confusion. The position of the negative sign matters greatly:

  • |a| means "the distance of a from zero." The result is always non-negative.
  • -|a| means "the opposite of the distance of a from zero." Here, you first find the absolute value of a, and then apply the negative sign to that result.
  • Example:
    • |-6| = 6 (The absolute value of -6 is 6.)
    • -|6| = -(6) = -6 (The absolute value of 6 is 6, then make it negative.)
    • –|-3| = -(3) = -3 (First, |-3|=3, then apply the negative sign to get -3.)

Applying Absolute Value and Opposites

1. Finding Opposites:

  • To find the opposite of a positive number, simply place a negative sign in front of it (e.g., opposite of 12 is -12).
  • To find the opposite of a negative number, remove the negative sign (e.g., opposite of -9 is 9). This is equivalent to -(-9) = 9.
  • The opposite of 0 is 0.

2. Evaluating Absolute Value Expressions:

  • |-14| = 14 (The distance of -14 from zero is 14.)
  • |8| = 8 (The distance of 8 from zero is 8.)
  • |5 - 5| = |0| = 0 (Simplify inside the bars first, then take absolute value.)

3. Comparing Numbers Using Absolute Value: When comparing |a| and |b|, you are comparing their magnitudes (their distances from zero).

  • Example: Is |-9| > |6|? |-9| = 9 and |6| = 6. Since 9 > 6, then |-9| > |6|.
  • Example: Is |-4| = |4|? |-4| = 4 and |4| = 4. Since 4 = 4, then |-4| = |4|.

4. Real-World Application: Temperature Difference Absolute value is useful for finding the magnitude of a change or the distance between two points.

  • Problem: Temperature rose from -8°F to 15°F. How many degrees did it rise?
  • Solution: We want the distance between 15 and -8. This can be calculated as |15 - (-8)|.
    • 15 - (-8) = 15 + 8 = 23.
    • |23| = 23.
  • The temperature rose 23 degrees. (You can visualize this as 8 degrees to get from -8 to 0, plus 15 degrees to get from 0 to 15, for a total of 8 + 15 = 23 degrees.)
  • General Formula for Distance: The distance between two points a and b on a number line is |a - b|.

Summary of Absolute Value and Opposites

  • The opposite of a is -a. A number and its opposite are the same distance from zero.
  • |a| is the distance from a to zero on the number line.
  • Absolute value is always non-negative (≥ 0).
  • -|a| means to apply the negative sign after taking the absolute value.

Review Questions

  3. Evaluate the following expressions: a. |-25| b. |12 - 12| c. -|17| d. -(-|8|) e. |-(-30)|
