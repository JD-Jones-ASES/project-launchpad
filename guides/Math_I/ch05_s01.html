Study Guide: Decimal Place Value and Comparing Decimals

Overview

This section introduces decimals as an extension of the place value system to numbers less than one. It covers how to understand the value of each digit in a decimal number, write decimals in various forms (standard, expanded, and word), and effectively compare and order decimal numbers using a systematic approach that includes annexing zeros.

Key Concepts

  • Decimal Place Value System: Decimals utilize the same base-ten structure as whole numbers, extending to the right of the decimal point. Each place to the right represents a fractional value that is one-tenth of the value of the place to its left (e.g., tenths, hundredths, thousandths).
  • Three Forms of a Decimal:
    • Standard Form: The typical way we write numbers using digits and a decimal point (e.g., 4.037).
    • Expanded Form: Breaking down a number to show the value of each digit (e.g., 4 + 0.0 + 0.03 + 0.007).
    • Word Form: Writing the number out in words, using "and" to represent the decimal point (e.g., four and thirty-seven thousandths).
  • Comparing Decimals: To accurately compare decimals, it's crucial to align their decimal points, annex zeros to ensure both numbers have the same number of decimal places, and then compare the digits from left to right, starting with the largest place value.
  • Annexing Zeros: Adding one or more zeros to the end of a decimal number (to the right of the last non-zero digit) does not change its value. This technique is vital for comparing decimals effectively by making them have the same number of decimal places.

Important Terms

  • Decimal: A number that uses a decimal point to represent values less than one. The digits to the right of the decimal point represent fractional parts.
  • Decimal Point: The symbol used to separate the whole-number part from the fractional (decimal) part of a number.
  • Tenths: The first place value to the right of the decimal point, representing $1/10$ or $0.1$.
  • Hundredths: The second place value to the right of the decimal point, representing $1/100$ or $0.01$.
  • Thousandths: The third place value to the right of the decimal point, representing $1/1000$ or $0.001$.
  • Standard Form: The common way of writing a decimal number using digits and a decimal point.
  • Expanded Form: A way to write a decimal by showing the sum of the value of each digit.
  • Word Form: A way to write a decimal using words, where "and" signifies the decimal point.
  • Annex Zeros: The process of writing extra zeros at the end of a decimal number without changing its value, often done to help with comparison or arithmetic.

Detailed Notes

Understanding Decimal Place Value

Just like whole numbers, decimals use a place value system where each position of a digit matters. However, decimals extend this system to represent parts of a whole, or values less than one.

  • The Decimal Point: This crucial symbol separates the whole number part (to its left) from the fractional or decimal part (to its right).
  • Place Values to the Right:
    • The first digit to the right of the decimal point is in the tenths place ($0.1$ or $1/10$).
    • The second digit to the right is in the hundredths place ($0.01$ or $1/100$).
    • The third digit to the right is in the thousandths place ($0.001$ or $1/1000$).
  • Pattern: Each place value to the right of the decimal point is one-tenth the value of the place immediately to its left.
  • Naming Convention: Notice that decimal place values end in "-ths" (e.g., tenths, hundredths), which distinguishes them from whole number places (e.g., tens, hundreds).

Example: In the number $12.583$:

  • $1$ is in the tens place.
  • $2$ is in the ones place.
  • $5$ is in the tenths place.
  • $8$ is in the hundredths place.
  • $3$ is in the thousandths place.

Three Forms of a Decimal

Decimals can be expressed in different ways to highlight their structure and value.

  1. Standard Form: This is the most common way to write a decimal number using digits.
    • Example: $4.037$
  2. Expanded Form: This form shows the sum of the value of each digit. You write each non-zero digit multiplied by its place value, or simply write out the decimal value for each digit.
    • Example: For $4.037$, the expanded form is $4 + 0.0 + 0.03 + 0.007$.
    • Example: For $0.269$, the expanded form is $0.2 + 0.06 + 0.009$.
  3. Word Form: This form writes the decimal out in words.
    • Rule: Read the whole number part (if any), say "and" for the decimal point, then read the digits to the right of the decimal point as if they were a whole number, and finally state the place value of the last digit.
    • Example: For $4.037$, the word form is "four and thirty-seven thousandths."
    • Example: For $0.269$, the word form is "two hundred sixty-nine thousandths." (There is no whole number part, so "and" is not used here).

Checkpoint Example: Write $3.04$ in expanded form and word form.

  • Expanded form: $3 + 0.04$
  • Word form: three and four hundredths

Comparing Decimals

Comparing decimals helps us determine which number is greater, less, or if they are equal. Follow these steps for accurate comparison:

  1. Line Up Decimal Points: Write the numbers vertically, ensuring the decimal points are directly aligned. This helps match corresponding place values.
  2. Annex Zeros: Add zeros to the end of the decimal with fewer digits to the right of the decimal point until both numbers have the same number of decimal places. This step is crucial for an "apples-to-apples" comparison.
    • What is Annexing Zeros? It means adding zeros to the rightmost end of a decimal number without changing its value. For instance, $3.5$ is the same as $3.50$ or $3.500$.
  3. Compare Digits from Left to Right: Starting from the largest place value (the leftmost digit), compare the digits in each corresponding position.
    • If the digits are the same, move to the next place value to the right.
    • The first place where the digits differ determines which number is larger or smaller. The number with the larger digit in that position is the greater number.

Example: Comparing $0.45$ and $0.437$

  1. Line up:
    0.45
0.437
  2. Annex zeros: Make both have three decimal places.
    0.450
0.437
  3. Compare left to right:
    • Ones place: $0 = 0$ (same)
    • Tenths place: $4 = 4$ (same)
    • Hundredths place: $5 > 3$.
      • Since $5$ (in $0.450$) is greater than $3$ (in $0.437$), $0.450$ is greater than $0.437$.

Therefore, $0.45 > 0.437$.

Example: Comparing Measurements ($2.75$ cm and $2.7$ cm)

  1. Align:
    2.75
2.7
  2. Annex zero:
    2.75
2.70
  3. Compare:
    • Ones: $2 = 2$
    • Tenths: $7 = 7$
    • Hundredths: $5 > 0$
      • Since $5 > 0$, $2.75$ is greater than $2.70$.

Thus, $2.75$ cm is longer than $2.7$ cm.

Caution: Always remember to line up the decimal points and annex zeros before comparing. Comparing digits in different place values (e.g., comparing the '5' in $0.45$ to the '3' in $0.437$ directly without annexing) can lead to incorrect conclusions.

Checkpoint Example: Order from least to greatest: $0.6$, $0.59$, $0.605$.

  1. Align and Annex Zeros (to thousandths place, as $0.605$ has three decimal places):
    0.600
0.590
0.605
  2. Compare:
    • Start with the tenths place: $0.590$ has $5$ in the tenths place, while $0.600$ and $0.605$ have $6$. So, $0.590$ is the smallest.
    • Now compare $0.600$ and $0.605$.
      • Tenths: $6=6$
      • Hundredths: $0=0$
      • Thousandths: $0 < 5$. So, $0.600$ is smaller than $0.605$.

Answer: $0.59 < 0.6 < 0.605$
