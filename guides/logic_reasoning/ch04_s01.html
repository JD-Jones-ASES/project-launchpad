Chapter 4, Section 1: Basic Connectives: AND, OR, NOT, AND IF THEN - Study Guide

Overview

This section introduces the foundational concepts of propositional logic, which focuses on how simple statements connect to form more complex ones. It defines what a proposition is—a sentence that can be definitively true or false—and then explores the four basic logical connectives: AND, OR, NOT, and IF THEN, explaining their meaning and how they combine propositions.

Key Concepts

  • Proposition: A declarative sentence that is either absolutely true or absolutely false, but cannot be both or neither. This forms the basic building block of propositional logic.
  • Logical Connective: A word or phrase that joins two or more propositions (or modifies one) to create a new, larger proposition.
  • AND (Conjunction): A connective that links two propositions; the combined statement is only true if both individual propositions are true. It uses the symbol $\land$.
  • OR (Disjunction): A connective that links two propositions; the combined statement is true if at least one of the individual propositions is true. In logic, "OR" is always inclusive, meaning it is true even if both propositions are true. It uses the symbol $\lor$.
  • NOT (Negation): A connective that changes the truth value of a single proposition. If the original proposition is true, its negation is false, and vice versa. It uses the symbol $\lnot$.
  • IF THEN (Conditional): A connective that expresses a relationship where one proposition (the antecedent) implies another (the consequent). The statement promises that if the "if" part is true, the "then" part will also be true. It uses the symbol $\to$.
  • Antecedent: The "if" part of an IF THEN statement; the condition.
  • Consequent: The "then" part of an IF THEN statement; the result that follows if the antecedent is true.

Important Terms

Proposition : A sentence that is either true or false, but not both, and not neither. Commands, questions, and exclamations are not propositions.

Logical Connective : A word or phrase (like "and," "or," "not," "if then") that combines simple propositions to form more complex ones.

AND (Conjunction) : A logical connective that results in a true statement only when both propositions it connects are true. Represented by the symbol $\land$.

OR (Disjunction) : A logical connective that results in a true statement if at least one of the propositions it connects is true. In logic, it is always inclusive (true if one or both are true). Represented by the symbol $\lor$.

NOT (Negation) : A logical connective that reverses the truth value of a single proposition. If the original proposition is true, its negation is false, and vice versa. Represented by the symbol $\lnot$.

IF THEN (Conditional) : A logical connective that expresses a cause-and-effect or conditional relationship between two propositions. Represented by the symbol $\to$.

Antecedent : The first part of an IF THEN statement, following "if," which states the condition.

Consequent : The second part of an IF THEN statement, following "then," which states the result.

Detailed Notes

Introduction to Propositional Logic

  • Shift in Focus: Chapters 1-3 examined arguments as a whole. Propositional logic (Chapters 4-7) "zooms in" to analyze how individual sentences are connected.
  • Core Idea: It's the study of how simple statements are combined using specific words like "and," "or," "not," and "if then."

What Is a Proposition?

  • Definition: A proposition is a statement that has a definite truth value. It must be able to be labeled as true or false.
  • Exclusions: Sentences that are commands ("Close the door!"), questions ("What time is lunch?"), or exclamations ("Wow, that was amazing!") are not propositions because they cannot be assigned a truth value.
  • Test: If you can reasonably determine whether a sentence is true or false, it's a proposition.

The Four Connectives

Logical connectives are words or phrases that join propositions to form new, larger propositions.

  1. AND (Conjunction)

    • Meaning: The combined statement using "AND" is true only if both of the propositions it connects are true. If even one part is false, the entire "AND" statement is false.
    • Symbol: $\land$ (looks like an upside-down 'V').
    • Example: "I have a pencil $\land$ I have a notebook" is only true if you possess both items.

  2. OR (Disjunction)

    • Meaning: The combined statement using "OR" is true if at least one of the propositions it connects is true.
    • Symbol: $\lor$ (a regular 'V').
    • Inclusive OR: In logic, "OR" is always inclusive. This means if both parts of an "OR" statement are true, the entire statement is still considered true. (e.g., "You can have cake $\lor$ pie" is true even if you get both). This differs from some everyday uses of "or" where it might imply "one or the other, but not both."
    • Example: "We'll play outside $\lor$ we'll watch a movie" is true if we play outside, or if we watch a movie, or if we do both.

  3. NOT (Negation)

    • Meaning: "NOT" changes the truth value of a single proposition to its opposite. If the original statement is true, its negation is false, and vice versa.
    • Symbol: $\lnot$ (a little dash with a tail).
    • Operation: Unlike the other connectives, "NOT" operates on just one proposition.
    • Example: If $P$ is "It is raining," then $\lnot P$ means "It is not raining."

  4. IF THEN (Conditional)

    • Meaning: This connective states that one proposition (the condition) leads to another proposition (the result). It creates a conditional relationship.
    • Symbol: $\to$ (an arrow pointing right). $P \to Q$ means "If $P$, then $Q$."
    • Parts:
      • Antecedent: The proposition immediately following "if" (the condition).
      • Consequent: The proposition immediately following "then" (the result).
    • Example: "If you study, then you'll do well on the test." Here, "you study" is the antecedent, and "you'll do well on the test" is the consequent.
    • Important Note: The conditional statement only tells you what happens when the "if" part is true. It doesn't necessarily mean anything specific if the "if" part is false. (This will be explored in more detail in Section 4.2).

Putting It All Together (Symbolic Representation)

  • Propositions can be represented by letters (e.g., $P$, $Q$).
  • This allows for concise representation of complex statements:
    • $P \land Q$: "It is sunny AND we will go to the park."
    • $P \lor Q$: "It is sunny OR we will go to the park."
    • $\lnot P$: "It is NOT sunny."
    • $P \to Q$: "IF it is sunny, THEN we will go to the park."
  • The primary goal is understanding the meaning of each connective, even if the symbols aren't memorized immediately.

Real-World Application

  • Logic Gates in Computers: The logical operations AND, OR, and NOT are fundamental to how computers and other electronic devices work. Billions of tiny "logic gates" perform these operations to process information. Understanding these connectives means understanding the basic language of computing.

Review Questions

  1. Explain what a "proposition" is in your own words, and provide two examples of sentences that are propositions and two examples of sentences that are not propositions.
  2. Describe the difference between how "OR" is used in everyday English (sometimes exclusively) and its meaning as an inclusive logical connective. Give an example where the logical "OR" would be true but an everyday "either/or" might not be.
  3. Imagine you have two true propositions, P and Q, and one false proposition, R. For each of the following, determine if the combined statement would be True or False, and explain why:
    • a) $P \land Q$
    • b) $P \lor R$
    • c) $\lnot R$
    • d) $P \to R$
  4. Write one sentence for each of the four connectives (AND, OR, NOT, IF THEN) using different scenarios from those provided in the text. Clearly identify which connective you are using.
  5. Why is it important to have a precise, unambiguous definition for logical connectives like AND, OR, NOT, and IF THEN, especially when thinking about fields like computer science or philosophy?
