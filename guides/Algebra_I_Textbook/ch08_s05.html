Study Guide: Graphing Quadratic Functions

This study guide provides a comprehensive overview of graphing quadratic functions based on the standard form $f(x) = ax^2 + bx + c$. It covers essential properties, step-by-step graphing procedures, and conceptual applications.

Key Concepts and Properties

1. The Parabola

A quadratic function creates a U-shaped curve known as a parabola. This curve is characterized by its symmetry and a specific point where the graph changes direction, known as the vertex.

2. Direction of Opening

The leading coefficient $a$ determines how the parabola opens and whether the vertex represents a high or low point on the graph:

Value of $a$ Direction of Opening Vertex Type
$a > 0$ (Positive) Opens Upward Minimum (Lowest Point)
$a < 0$ (Negative) Opens Downward Maximum (Highest Point)

3. Essential Formulas

  • Axis of Symmetry: The vertical line that divides the parabola into two symmetric halves. It is calculated using the formula:
    $x = \frac{-b}{2a}$
  • Vertex Coordinates: The vertex is found by using the axis of symmetry as the $x$-coordinate and evaluating the function at that value:
    $(\frac{-b}{2a}, f(\frac{-b}{2a}))$
  • Vertex Form: An alternative way to write a quadratic function is $f(x) = a(x - h)^2 + k$, where $(h, k)$ is the vertex.

4. Domain and Range

  • Domain: The domain for every quadratic function is always all real numbers, expressed as $(-\infty, \infty)$.
  • Range: The range depends on the $y$-coordinate of the vertex ($y_v$) and the direction of opening:
    • If $a > 0$, the range is $[y_v, \infty)$.
    • If $a < 0$, the range is $(-\infty, y_v]$.

Step-by-Step Graphing Procedure

To graph a quadratic function $f(x) = ax^2 + bx + c$, follow these six steps:

  1. Find the Axis of Symmetry: Use $x = \frac{-b}{2a}$.
  2. Locate the Vertex: Evaluate the function at the axis of symmetry value.
  3. Determine the $y$-intercept: Set $x = 0$; the intercept is $(0, c)$.
  4. Find the $x$-intercepts: Solve the equation $ax^2 + bx + c = 0$. Note that real $x$-intercepts may not always exist.
  5. Identify Additional Points: Use the axis of symmetry to find mirror images of existing points (such as the $y$-intercept).
  6. Sketch the Curve: Plot the points and draw the parabola.

Short-Answer Practice Quiz

1. Given the function $f(x) = -x^2 + 2x + 8$, what is the equation for the axis of symmetry? Answer: $x = 1$. (Calculated via $\frac{-2}{2(-1)}$).

2. How do you determine the $y$-intercept of a quadratic function in standard form? Answer: Set $x = 0$, which results in $f(0) = c$. The $y$-intercept is always the point $(0, c)$.

3. In the function $f(x) = 2x^2 - 4x + 5$, why are there no real $x$-intercepts? Answer: When solving for the intercepts, the discriminant ($b^2 - 4ac$) is negative ($16 - 40 = -24$). Since the value is less than zero, the parabola does not cross the $x$-axis.

4. If a parabola has a vertex at $(2, 5)$ and opens upward, what is its range? Answer: $[5, \infty)$.

5. What is the vertex of the function provided in the vertex form $f(x) = 3(x - 2)^2 + 5$? Answer: $(2, 5)$.

Essay Prompts for Deeper Exploration

1. The Role of the Leading Coefficient $a$ Explain how the leading coefficient $a$ serves as the primary indicator for the shape and orientation of a parabola. Discuss how its sign influences the existence of a maximum or minimum value and how this relates to the real-world concept of a range.

2. Symmetry in Graphing Describe the significance of the axis of symmetry in the graphing process. How does finding the axis of symmetry and one point (such as the $y$-intercept) allow a mathematician to efficiently plot the rest of the parabola without solving for every individual $x$-value?

3. The Relationship Between Intercepts and the Vertex Using the examples of $f(x) = x^2 - 4x + 3$ (which has two $x$-intercepts) and $f(x) = 2x^2 - 4x + 5$ (which has none), analyze how the position of the vertex relative to the $x$-axis, combined with the direction of opening, determines whether a quadratic equation will have real solutions.

Glossary of Important Terms

  • Axis of Symmetry: A vertical line ($x = \frac{-b}{2a}$) about which a parabola is symmetric.
  • Domain: The set of all possible input values (all real numbers for quadratic functions).
  • Parabola: The U-shaped curve that represents the graph of a quadratic function.
  • Range: The set of all possible output values, determined by the vertex and the direction the parabola opens.
  • Standard Form: The expression of a quadratic function as $f(x) = ax^2 + bx + c$.
  • Vertex: The highest or lowest point of a parabola; the point where the curve changes direction.
  • Vertex Form: The expression of a quadratic function as $f(x) = a(x - h)^2 + k$, where $(h, k)$ explicitly identifies the vertex.
  • $x$-intercept: The point(s) where the parabola crosses the $x$-axis, found by solving $f(x) = 0$.
  • $y$-intercept: The point where the parabola crosses the $y$-axis, found by evaluating $f(0)$.
