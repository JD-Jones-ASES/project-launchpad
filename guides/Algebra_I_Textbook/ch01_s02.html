Study Guide: The Real Number System

This study guide provides a comprehensive overview of the real number system, its organization, and the fundamental tools used to measure and compare numerical values.

Core Concepts

The Hierarchy of Real Numbers

The real numbers are organized into a series of nested sets. Understanding these relationships is essential for navigating algebra. Every real number corresponds to exactly one point on the number line.

Set Name Description Examples
Natural Numbers Also known as counting numbers. $1, 2, 3, 4, 5, \dots$
Whole Numbers Natural numbers plus the number zero. $0, 1, 2, 3, \dots$
Integers All whole numbers and their opposites (negative versions). $\dots, -3, -2, -1, 0, 1, 2, 3, \dots$
Rational Numbers Numbers that can be written as a fraction $\frac{a}{b}$ (where $a$ and $b$ are integers and $b \neq 0$). Their decimal forms either terminate or repeat. $0.75$ (terminates), $0.333\dots$ (repeats)
Irrational Numbers Real numbers that cannot be expressed as a fraction of integers. Their decimal forms never terminate and never repeat. $\sqrt{2} \approx 1.41421$, $\pi \approx 3.14159$
Real Numbers The complete set consisting of all rational and irrational numbers. All numbers listed above.

Subset Relationships

The number sets are categorized by their inclusion within one another:

  • Every Natural Number is a Whole Number.
  • Every Whole Number is an Integer.
  • Every Integer is a Rational Number.
  • Every Rational Number is a Real Number.
  • Irrational Numbers are Real Numbers but do not belong to any of the other categories.

The Number Line and Absolute Value

The number line is a visual representation where every point represents a unique real number.

  • Opposites: The opposite of any number $a$ is $-a$. Opposites are the same distance from $0$ on the number line but lie on opposite sides.
  • Absolute Value: Represented as $|a|$, this is the distance from a number to $0$ on the number line. Because it measures distance, absolute value is always non-negative.
    • If $a \geq 0$, then $|a| = a$.
    • If $a < 0$, then $|a| = -a$.
  • Ordering: When comparing two numbers on a number line, the number located further to the right is the greater value.

Short-Answer Practice Questions

  1. Classification: To which number sets does the number $0$ belong?
  2. Rational vs. Irrational: If a decimal continues forever without a repeating pattern, is it rational or irrational?
  3. Opposites: What is the opposite of $-5$?
  4. Absolute Value: Evaluate $|-12|$.
  5. Distance: Evaluate $|0|$.
  6. Definitions: What defines a number as "Natural"?
  7. Comparison: Between $-2$ and $-5$, which number is greater according to its position on the number line?
  8. Calculation: Evaluate the absolute value of $-7$.
  9. Conversion: When ordering the numbers $-2$ and $\frac{3}{4}$, what is a helpful first step to compare them easily?
  10. Set Inclusion: Is every integer also a natural number? Explain why or why not.

Essay Prompts for Deeper Exploration

  1. The Geometry of Real Numbers: Discuss the relationship between the set of real numbers and the number line. Why is it significant that every real number corresponds to exactly one point, and how does this visual model assist in comparing rational and irrational values?
  2. Distinguishing the Sets: Explain the structural differences between rational and irrational numbers. In your explanation, detail the requirements for a number to be expressed as a fraction and how those requirements dictate the behavior of the number's decimal expansion.
  3. The Concept of Absolute Value: Define absolute value in terms of distance. Explore why the absolute value of a negative number must result in a positive value, using the definition of distance from zero on a number line to support your reasoning.

Glossary of Key Terms

  • Absolute Value: The non-negative distance of a real number from zero on the number line, denoted by $|a|$.
  • Integers: The set of numbers including all whole numbers and their opposites.
  • Irrational Numbers: Real numbers that cannot be written as a fraction of integers and have non-terminating, non-repeating decimal forms.
  • Natural Numbers: The set of counting numbers starting from 1.
  • Number Line: A visual model where every point corresponds to exactly one real number.
  • Opposite: A number that is the same distance from zero as a given number $a$, but on the opposite side, denoted as $-a$.
  • Rational Numbers: Any number that can be expressed as a fraction $\frac{a}{b}$ where $a$ and $b$ are integers and $b \neq 0$; these numbers have terminating or repeating decimals.
  • Real Numbers: The total set of all rational and irrational numbers.
  • Whole Numbers: The set of natural numbers plus zero.
