# Study Guide: Solving Quadratic Equations by Square Roots

This study guide focuses on solving quadratic equations using the Square Root Property. This method is specifically applicable when a quadratic equation lacks a linear term (the $bx$ term) or when the equation can be structured to have a perfect square on one side.

---

## Key Concepts and Procedures

### 1. The Square Root Property
The Square Root Property is the foundational rule for solving equations where a squared variable or expression is equal to a constant.

*   **The Rule:** If $x^2 = k$, then $x = \sqrt{k}$ or $x = -\sqrt{k}$. This is commonly written as $x = \pm\sqrt{k}$.
*   **Solutions based on the value of $k$:**
    *   **If $k > 0$:** The equation has two real solutions.
    *   **If $k < 0$:** The equation has no real solution (as no real number squared results in a negative value).
    *   **The $\pm$ Symbol:** It is critical to include both the positive and negative roots. For instance, in the equation $x^2 = 9$, the solutions are $3$ and $-3$. Providing only the positive root is considered incomplete.

### 2. Simplified Radical Form
When solving quadratics by square roots, the final answer must often be written in simplified radical form. A radical expression is simplified only when it meets the following three criteria:
1.  **No perfect square factors:** Other than 1, no perfect square factors remain under the radical (e.g., $\sqrt{12}$ must be simplified to $2\sqrt{3}$ because 4 is a perfect square factor).
2.  **No fractions under the radical:** The radicand cannot be a fraction.
3.  **No radicals in the denominator:** A radical cannot remain in the bottom part of a fraction.

### 3. Solving $(ax + b)^2 = k$
The Square Root Property can be extended to expressions where a binomial is squared. To solve these:
1.  Apply the Square Root Property to both sides to eliminate the square.
2.  Set up two separate linear equations (one for the positive root and one for the negative root).
3.  Solve each linear equation for $x$.

**Example:**
To solve $(2x - 1)^2 = 25$:
*   $2x - 1 = \pm\sqrt{25}$
*   $2x - 1 = 5$ AND $2x - 1 = -5$
*   $2x = 6 \implies x = 3$
*   $2x = -4 \implies x = -2$

## Short-Answer Practice Quiz

**Instructions:** Solve the following equations using the Square Root Property. Ensure all radicals are in simplified form.

1.  **Solve for $x$:** $x^2 = 49$
2.  **Solve for $x$:** $x^2 = 12$
3.  **Solve for $x$:** $x^2 = -4$
4.  **Solve for $x$:** $(x - 4)^2 = 36$
5.  **Solve for $x$:** $(x + 3)^2 = 20$

### Answer Key
1.  $x = \pm 7$
2.  $x = \pm 2\sqrt{3}$ (Since $\sqrt{12} = \sqrt{4 \cdot 3}$)
3.  No real solution (Since $-4 < 0$)
4.  $x = 10$ and $x = -2$ (From $x - 4 = 6$ and $x - 4 = -6$)
5.  $x = -3 \pm 2\sqrt{5}$ (From $x + 3 = \pm\sqrt{20}$, where $\sqrt{20} = 2\sqrt{5}$)

## Glossary of Important Terms

| Term | Definition |
| :--- | :--- |
| **Linear Term** | The $bx$ term in a quadratic equation; its absence allows for direct solving via square roots. |
| **Square Root Property** | A rule stating that if $x^2 = k$, then $x$ is equal to the positive or negative square root of $k$. |
| **$\pm$ (Plus-Minus)** | A mathematical symbol indicating that a value can be both positive and negative; essential for identifying all solutions to a quadratic. |
| **Simplified Radical Form** | A radical expression containing no perfect square factors, no internal fractions, and no radicals in the denominator. |
| **Real Solution** | A solution that exists within the set of real numbers; equations with $x^2 = \text{negative number}$ do not have these. |
| **Imaginary Numbers** | A concept introduced in advanced algebra (Algebra II) that allows for the solving of square roots of negative numbers. |
| **Radicand** | The value located inside or under the square root symbol. |
