# Study Guide: Inequalities and Their Graphs

This study guide provides a comprehensive overview of linear inequalities, their representation on number lines, and various notation methods used to describe solution sets. It is based on the fundamental principles of algebra used to compare mathematical expressions.

---

## Key Concepts and Foundations

### Understanding Inequalities
While equations represent statements where two expressions are equal, inequalities are used to compare expressions where quantities may not be equal. These comparisons are essential for representing real-world constraints, such as temperature thresholds or financial targets.

An **inequality** is a mathematical sentence comparing two expressions using one of four primary symbols:
*   $<$ (less than)
*   $>$ (greater than)
*   $\leq$ (less than or equal to)
*   $\geq$ (greater than or equal to)

### The Solution Set
Unlike a standard linear equation which may have a single solution, an inequality often has a **solution set**. This is defined as the set of all values of the variable that make the inequality statement true.

### Graphing on a Number Line
To visualize the solution set, regions are shaded on a number line. The boundary point—the value where the inequality begins—is marked using specific conventions to indicate whether that value is included in the solution:

| Symbol | Meaning | Circle Type | Shading Direction |
| :--- | :--- | :--- | :--- |
| $<$ | Less than | Open ( $\circ$ ) | Left |
| $>$ | Greater than | Open ( $\circ$ ) | Right |
| $\leq$ | Less than or equal to | Closed ( $\bullet$ ) | Left |
| $\geq$ | Greater than or equal to | Closed ( $\bullet$ ) | Right |

**Examples of Graphing:**
*   **$x > 2$**: Place an open circle at $2$ and shade to the right because $x$ is strictly greater than $2$.
*   **$x \leq 4$**: Place a closed circle at $4$ and shade to the left because $x$ is less than or equal to $4$.

## Notation Methods

There are two compact mathematical conventions for expressing solution sets: Set-Builder Notation and Interval Notation.

### Set-Builder Notation
Set-builder notation describes the solution set by stating a specific condition.
*   **Format:** $\{x \mid x > 2\}$
*   **Reading:** "The set of all $x$ such that $x$ is greater than $2$."

### Interval Notation
Interval notation uses parentheses and brackets to describe the span of the solution set on the number line.
*   **Parentheses ( or )**: Used when the endpoint is **not included** (corresponds to $<$ or $>$).
*   **Brackets [ or ]**: Used when the endpoint is **included** (corresponds to $\leq$ or $\geq$).
*   **Infinity ($\infty$) and Negative Infinity ($-\infty$)**: These symbols represent values that are not real numbers and are **always** paired with a parenthesis.

**Comparison Table:**

| Inequality | Set-Builder Notation | Interval Notation |
| :--- | :--- | :--- |
| $x > 2$ | $\{x \mid x > 2\}$ | $(2, \infty)$ |
| $x \geq -3$ | $\{x \mid x \geq -3\}$ | $[-3, \infty)$ |
| $x \leq -1$ | $\{x \mid x \leq -1\}$ | $(-\infty, -1]$ |

## Glossary of Key Terms

*   **Closed Circle:** A mark on a number line graph used to indicate that the boundary value is included in the solution set (used with $\leq$ or $\geq$).
*   **Inequality:** A mathematical sentence that compares two expressions using symbols such as $<, >, \leq,$ or $\geq$.
*   **Interval Notation:** A method of describing a solution set using parentheses and brackets to indicate included and excluded endpoints.
*   **Open Circle:** A mark on a number line graph used to indicate that the boundary value is not included in the solution set (used with $<$ or $>$).
*   **Set-Builder Notation:** A shorthand way to write a solution set by stating the condition that all variables in the set must meet (e.g., $\{x \mid x > 2\}$).
*   **Solution Set:** The collection of all values that make a given inequality true.
