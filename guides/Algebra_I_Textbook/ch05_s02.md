# Study Guide: Solving Systems by Substitution

This study guide focuses on the substitution method, an algebraic technique used to find exact solutions for systems of two equations. While graphing provides a visual representation of a system, algebraic methods are often preferred for their precision.

## Core Concepts of Substitution

### The Substitution Method Process
To solve a system of two equations by substitution, follow these five steps:

1.  **Isolate a Variable:** Solve one of the equations for one variable. It is most efficient to choose the variable that is easiest to isolate.
2.  **Substitute the Expression:** Take the expression found in Step 1 and substitute it into the *other* equation.
3.  **Solve for the First Variable:** Solve the resulting equation, which now contains only one variable.
4.  **Find the Second Variable:** Substitute the value obtained in Step 3 back into the equation from Step 1 to solve for the remaining variable.
5.  **Check the Solution:** Verify the resulting $(x, y)$ coordinate pair by plugging it into both of the original equations.

### Precision and Best Practices
*   **Exactness:** Algebraic techniques like substitution produce exact solutions, whereas reading coordinates from a graph can be imprecise.
*   **The Use of Parentheses:** When substituting an expression into an equation, it is critical to replace the variable with the entire expression contained within parentheses. For example, if substituting $4y + 1$ for $x$ in the term $2x$, the result should be $2(4y + 1)$. Failing to use parentheses, such as writing $2 \cdot 4y + 1$, changes the mathematical meaning and leads to incorrect results.

## Categorizing Solutions

When solving a system algebraically, the resulting mathematical statement determines the nature of the solution.

| Result of Substitution | Mathematical Categorization | Geometric Interpretation | System Outcome |
| :--- | :--- | :--- | :--- |
| Variable equals a specific value (e.g., $x = 3$) | Standard Solution | The lines intersect at a single point. | One exact solution $(x, y)$. |
| A false statement without variables (e.g., $6 = 10$) | **Contradiction** | The lines are parallel and never intersect. | No solution. |
| A statement that is always true (e.g., $18 = 18$) | **Identity** | The two equations describe the same line. | Infinitely many solutions. |

## Short-Answer Practice Quiz

**1. Why might a student choose the substitution method over graphing to solve a system of equations?**
*Answer:* Graphing can be imprecise when reading coordinates, while the substitution method provides exact algebraic solutions.

**2. In the first step of substitution, which variable should you typically choose to isolate?**
*Answer:* You should choose the variable that is easiest to isolate.

**3. What is the common error to avoid when substituting an expression like $3x - 5$ into another equation?**
*Answer:* Failing to use parentheses. The expression must be treated as a whole to ensure the entire value is correctly multiplied or combined.

**4. If the variables in a system cancel out and you are left with the statement $10 = 10$, what does this indicate about the system?**
*Answer:* This is an identity, meaning the equations describe the same line and there are infinitely many solutions.

**5. What does a contradiction (such as $0 = 4$) tell you about the relationship between the two lines in a system?**
*Answer:* A contradiction indicates that the lines are parallel and the system has no solution.

**6. Solve the following system using substitution:**
$x = 4y + 1$
$3x - 2y = 13$
*Answer:* $(5, 1)$. (Substitute $4y + 1$ for $x$ to get $3(4y + 1) - 2y = 13$, which simplifies to $10y = 10$, so $y = 1$ and $x = 5$).

## Glossary of Key Terms

*   **Contradiction:** A mathematical statement that is false and contains no variables (e.g., $6 = 10$). In a system of equations, it indicates there is no solution because the lines are parallel.
*   **Identity:** A mathematical statement that is always true (e.g., $18 = 18$). In a system of equations, it indicates that the two equations describe the same line, resulting in infinitely many solutions.
*   **Substitution Method:** An algebraic technique used to solve systems of equations by isolating one variable and replacing it in the second equation to reduce the system to a single variable.
*   **System of Equations:** A set of two or more equations with the same variables that are solved simultaneously to find common solutions.
