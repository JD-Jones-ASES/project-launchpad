Study Guide: Relations and Functions

This study guide focuses on the fundamental mathematical concepts of relations and functions, exploring their definitions, how to identify them, and how to use function notation to evaluate expressions.

Key Concepts Review

1. Understanding Relations

A relation is any set of ordered pairs. These relationships can be expressed in several formats, including:

  • Lists of ordered pairs
  • Tables
  • Graphs
  • Equations

2. Domain and Range

Every relation consists of two sets of values:

  • Domain: The set of all first coordinates, or $x$-values.
  • Range: The set of all second coordinates, or $y$-values.

When identifying the domain and range from a list of ordered pairs, each value should be listed only once, even if it appears multiple times in the coordinates.

3. Defining a Function

A function is a specific type of relation. For a relation to qualify as a function, every element in the domain must be paired with exactly one element in the range.

Criteria for Functions:

  • No $x$-value can be repeated with a different $y$-value.
  • One input must result in exactly one output.
  • Note: Multiple different $x$-values can point to the same $y$-value (e.g., $(0, 1), (1, 1), (2, 1)$ is a function).

4. The Vertical Line Test

The Vertical Line Test is a visual method used to determine if a graph represents a function.

  • Rule: A graph is a function if and only if every vertical line intersects the graph at most once.
  • Failure: If a vertical line crosses the graph at two or more points, a single $x$-value is paired with more than one $y$-value, meaning the relation is not a function.

5. Function Notation

Functions are often named with letters such as $f, g,$ or $h$. The notation $f(x)$ is read as "$f$ of $x$."

  • Meaning: $f(x)$ represents the output of the function $f$ when the input is $x$.
  • Caution: $f(x)$ does not mean "$f$ times $x$." It is a notation for evaluation, not multiplication.

Short-Answer Practice Quiz

1. Identify Domain and Range: Given the relation ${(-3, 2), (0, 5), (1, -1), (4, 5)}$, what is the domain and what is the range?

2. Identifying Functions (Ordered Pairs): Is the relation ${(2, 4), (3, -1), (2, 6)}$ a function? Explain why or why not.

3. Identifying Functions (Ordered Pairs): Is the relation ${(0, 1), (1, 1), (2, 1)}$ a function? Explain why or why not.

4. Evaluating Functions: If $f(x) = 3x - 2$, what is the value of $f(5)$?

5. Evaluating Functions: If $g(x) = x^2 + 4x$, what is the value of $g(3)$?

6. Algebraic Evaluation: If $f(x) = 3x - 2$, find the expression for $f(a + 1)$.

7. Graphical Analysis: According to the Vertical Line Test, what does it mean if a vertical line intersects a graph at three different points?

Answer Key

  1. Domain: ${-3, 0, 1, 4}$; Range: ${-1, 2, 5}$.
  2. No. The $x$-value $2$ is paired with two different outputs ($4$ and $6$).
  3. Yes. Each $x$-value appears only once; every input has exactly one output.
  4. 13. ($3(5) - 2 = 15 - 2 = 13$).
  5. 21. ($3^2 + 4(3) = 9 + 12 = 21$).
  6. $3a + 1$. ($3(a + 1) - 2 = 3a + 3 - 2 = 3a + 1$).
  7. The relation represented by the graph is not a function because one $x$-value is paired with more than one $y$-value.

Essay Prompts for Deeper Exploration

1. The Distinction Between Relations and Functions Explain the specific criteria that elevate a general relation to the status of a function. Why can a function have repeating $y$-values but not repeating $x$-values? Use examples of ordered pairs to support your explanation.

2. Interpreting the Vertical Line Test Describe the logical connection between the definition of a function (one input per output) and the Vertical Line Test. Why is the intersection of a vertical line at more than one point a definitive indicator that a graph is not a function?

3. The Purpose and Pitfalls of Function Notation Discuss the significance of the notation $f(x)$ in mathematics. Address the common misconception regarding multiplication and explain how to correctly interpret the instruction to evaluate $f(x)$ for a given input.

Glossary of Terms

Term Definition
Domain The set of all first coordinates ($x$-values) in a relation.
Function A relation in which every element of the domain is paired with exactly one element of the range.
Function Notation A method of naming a function (e.g., $f(x)$) where $f(x)$ represents the output for the input $x$.
Range The set of all second coordinates ($y$-values) in a relation.
Relation Any set of ordered pairs, which can be described by a list, table, graph, or equation.
Vertical Line Test A property stating that a graph represents a function if and only if every vertical line intersects the graph at most once.
