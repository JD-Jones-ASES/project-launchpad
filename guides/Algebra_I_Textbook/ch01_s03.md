# Study Guide: Properties of Real Numbers

The real numbers are governed by several fundamental properties that serve as the foundation for all algebraic manipulation. These properties allow for the simplification of expressions and the systematic solving of equations. By identifying these properties by name, one can justify each step taken during mathematical work.

## Core Properties of Real Numbers

The following table outlines the essential properties applicable to all real numbers ($a$, $b$, and $c$).

| Property | Addition | Multiplication |
| :--- | :--- | :--- |
| **Commutative Property** | $a + b = b + a$ | $ab = ba$ |
| **Associative Property** | $(a + b) + c = a + (b + c)$ | $(ab)c = a(bc)$ |
| **Identity Property** | $a + 0 = a$ (Identity: $0$) | $a \cdot 1 = a$ (Identity: $1$) |
| **Inverse Property** | $a + (-a) = 0$ (Inverse: $-a$) | $a \cdot \frac{1}{a} = 1, a \neq 0$ (Inverse: $\frac{1}{a}$) |

### The Distributive Property
The Distributive Property applies to all real numbers $a$, $b$, and $c$ across addition and subtraction:
*   **Addition:** $a(b + c) = ab + ac$
*   **Subtraction:** $a(b - c) = ab - ac$

## Practical Applications and Limitations

### Simplifying Algebraic Expressions
Properties are used to combine like terms and group numbers for easier calculation. For example, to simplify $3(2x + 5) - 4x$:
1.  Apply the **Distributive Property**: $6x + 15 - 4x$
2.  **Group like terms**: $(6x - 4x) + 15$
3.  **Combine like terms**: $2x + 15$

### Non-Commutative Operations
It is critical to note that subtraction and division do **not** obey the Commutative Property. Changing the order of the numbers in these operations changes the result:
*   **Subtraction:** $8 - 3 = 5$, whereas $3 - 8 = -5$.
*   **Division:** $12 \div 4 = 3$, whereas $4 \div 12 = \frac{1}{3}$.

## Short-Answer Practice Quiz

**Instruction:** Name the property illustrated by each statement or provide the requested simplification.

1.  **Statement:** $9 + 0 = 9$
2.  **Statement:** $4(x + 7) = 4x + 28$
3.  **Statement:** $(2 \cdot 6) \cdot 3 = 2 \cdot (6 \cdot 3)$
4.  **Statement:** $5 + 3 = 3 + 5$
5.  **Simplification:** What is the result of $5(n - 3) + 2n$?

### Answer Key
1.  *Additive Identity Property*
2.  *Distributive Property*
3.  *Associative Property of Multiplication*
4.  *Commutative Property of Addition*
5.  *7n - 15* (Calculation: $5n - 15 + 2n = 7n - 15$)

## Glossary of Important Terms

*   **Additive Identity:** The number $0$, which, when added to any real number, results in the original number.
*   **Additive Inverse:** Also known as the **opposite**; the number $-a$ which, when added to $a$, equals $0$.
*   **Associative Property:** A rule stating that the way in which numbers are grouped in addition or multiplication does not change their sum or product.
*   **Commutative Property:** A rule stating that the order in which numbers are added or multiplied does not change their sum or product.
*   **Distributive Property:** A property that allows a multiplier outside of parentheses to be distributed to each term within the parentheses.
*   **Multiplicative Identity:** The number $1$, which, when multiplied by any real number, results in the original number.
*   **Multiplicative Inverse:** Also known as the **reciprocal**; the number $\frac{1}{a}$ (where $a \neq 0$) which, when multiplied by $a$, equals $1$.
*   **Opposite:** The additive inverse of a number.
*   **Reciprocal:** The multiplicative inverse of a number.
