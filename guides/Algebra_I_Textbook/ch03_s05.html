Study Guide: Absolute Value Inequalities

This study guide provides a comprehensive overview of solving absolute value inequalities, translating them into compound inequalities, and representing solutions in interval notation.

Key Concepts and Procedures

1. Conceptual Foundation

An absolute value inequality measures the distance a value is from zero on a number line. To solve these inequalities, they must be translated into compound inequalities.

  • "Less Than" ($\mathbf{<, \leq}$): Represents numbers that are within a specific distance from zero.
  • "Greater Than" ($\mathbf{>, \geq}$): Represents numbers that are further than a specific distance from zero.

2. Solving "Less Than" Inequalities ($|ax + b| < c$)

When $c > 0$, the inequality asks for values less than $c$ units from zero.

  • Translation: Rewrite as an "and" compound inequality: $-c < ax + b < c$.
  • Special Case ($c < 0$): Since an absolute value is always $\geq 0$, if $c$ is negative or zero, the inequality $|ax + b| < c$ has no solution.
  • Memory Aid: "Less thAND" (results in an "and" inequality).

3. Solving "Greater Than" Inequalities ($|ax + b| > c$)

When $c > 0$, the inequality asks for values more than $c$ units from zero.

  • Translation: Rewrite as an "or" compound inequality: $ax + b < -c$ or $ax + b > c$.
  • Special Case ($c < 0$): Since an absolute value is always $\geq 0$, the inequality $|ax + b| > c$ is true for all real numbers if $c$ is negative.
  • Memory Aid: "GreatOR" (results in an "or" inequality).

4. The Isolation Requirement

Before translating an absolute value inequality into a compound inequality, the absolute value expression must be isolated on one side of the inequality symbol.

Example of Isolation: To solve $3|x - 2| + 1 > 10$:

  1. Subtract 1 from both sides: $3|x - 2| > 9$.
  2. Divide by 3: $|x - 2| > 3$.
  3. Now translate into an "or" inequality.

Short-Answer Practice Questions

  1. True or False: The inequality $|x| < -2$ has no solution. Justify your answer.
  2. Translation: Rewrite $|2x - 3| < 7$ as a compound inequality without absolute value bars.
  3. Operation: What is the first step in solving the inequality $3|x - 2| + 1 > 10$?
  4. Interval Notation: Solve $|3x + 6| \leq 9$ and provide the solution in interval notation.
  5. Application: If $|x| > 5$, what does this ask regarding the distance of $x$ from zero?
  6. Translation: Rewrite $|x + 4| \geq 3$ as an "or" compound inequality.
  7. Special Case: What is the solution set for $|ax + b| > c$ if $c$ is a negative number?

Essay Prompts for Deeper Exploration

  1. The Logic of Distance: Explain the conceptual difference between $|x| < 5$ and $|x| > 5$. In your explanation, describe how the distance from zero dictates whether the solution is a single contained interval or two separate rays.
  2. The Necessity of Isolation: Discuss why it is critical to isolate the absolute value expression before attempting to rewrite it as a compound inequality. Use the expression $3|x - 2| + 1 > 10$ to demonstrate how the inequality changes during the isolation process.
  3. Comparative Analysis of Compound Inequalities: Compare and contrast "and" vs. "or" compound inequalities in the context of absolute value. How does the choice of inequality symbol ($<$ vs. $>$) fundamentally alter the structure of the resulting solution?

Glossary of Key Terms

  • Absolute Value: A mathematical expression representing the distance of a number from zero, which is always non-negative ($\geq 0$).
  • "And" Compound Inequality: A mathematical statement where the solution must satisfy two conditions simultaneously, resulting in a single interval between two values.
  • GreatOR: A memory aid indicating that "greater than" absolute value inequalities translate into "or" compound inequalities.
  • Interval Notation: A method of writing solution sets using brackets $[ ]$ for inclusive values ($\leq, \geq$) and parentheses $( )$ for exclusive values $(<, >)$.
  • Isolation: The process of using algebraic operations to ensure the absolute value expression is the only term on one side of the inequality.
  • Less thAND: A memory aid indicating that "less than" absolute value inequalities translate into "and" compound inequalities.
  • "Or" Compound Inequality: A mathematical statement where the solution satisfies at least one of two conditions, often resulting in two separate rays pointing in opposite directions.
