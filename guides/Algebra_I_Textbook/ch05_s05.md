# Study Guide: Systems of Linear Inequalities

This study guide provides a comprehensive overview of systems of linear inequalities, detailing the methods for graphing, identifying solution regions, and determining if specific points satisfy a system.

---

## Key Concepts and Procedures

### Defining Systems of Linear Inequalities
A system of linear inequalities involves two or more inequalities in two variables. A **solution of a system of inequalities** is an ordered pair $(x, y)$ that satisfies every inequality within the system. Geometrically, the set of all solutions forms a **solution region**, which represents the intersection or overlap of the half-planes defined by the individual inequalities.

### Graphing a Single Linear Inequality
To visualize the solution for an inequality such as $y < mx + b$, follow these steps:

1.  **Graph the Boundary Line:** Use the equation $y = mx + b$ to establish the boundary.
    *   **Solid Line:** Used for $\le$ or $\ge$ symbols to indicate the boundary is included in the solution.
    *   **Dashed Line:** Used for $<$ or $>$ symbols to indicate the boundary is not included.
2.  **Select a Test Point:** Choose a point that does not lie on the boundary line. The origin $(0, 0)$ is commonly used for convenience.
3.  **Substitute and Evaluate:** Plug the coordinates of the test point into the inequality.
    *   If the result is **true**, shade the half-plane that contains the test point.
    *   If the result is **false**, shade the half-plane on the opposite side of the line.

### Graphing a System of Inequalities
When graphing a system, the goal is to identify the region where the shading for all individual inequalities overlaps.

*   **Identifying the Vertex:** The vertex of a solution region is found by calculating the intersection of the boundary lines. This is done by setting the equations of the boundaries equal to each other and solving for $x$ and $y$.
*   **Inclusion of the Vertex:** A vertex is only included in the solution region if it lies on the intersection of two solid lines. If one or both of the boundary lines are dashed, the vertex is not included.

## Short-Answer Practice Questions

**1. What determines whether a boundary line should be drawn as solid or dashed?**
A solid line is used when the inequality includes "or equal to" ($\le$ or $\ge$). A dashed line is used for strict inequalities ($<$ or $>$).

**2. How is the solution region of a system defined?**
The solution region is the intersection or overlap of the half-planes created by each individual inequality in the system.

**3. If the point (0, 0) is tested in the inequality $y \le 2x + 1$ and the result is true, which region should be shaded?**
The half-plane containing the origin $(0, 0)$ should be shaded. In this specific case, that is the region below the solid line $y = 2x + 1$.

**4. When does a system of linear inequalities have no solution?**
A system has no solution when there is no overlapping region between the individual inequalities. An example of this is two parallel strips that never intersect.

**5. How do you mathematically determine if a specific ordered pair is a solution to a system?**
Substitute the $x$ and $y$ values of the ordered pair into every inequality in the system. If the point makes every inequality true, it is a solution.

## Glossary of Important Terms

| Term | Definition |
| :--- | :--- |
| **Boundary Line** | The line defined by the equation version of an inequality (e.g., $y = mx + b$) that separates the coordinate plane into two half-planes. |
| **Dashed Line** | A visual representation of a boundary line indicating that points on the line are not included in the solution set (used for $<$ or $>$). |
| **Half-Plane** | The region on one side of a boundary line in a coordinate plane. |
| **Intersection** | The area where two or more shaded regions overlap, representing the common solutions to a system. |
| **Solution of a System of Inequalities** | An ordered pair $(x, y)$ that satisfies every individual inequality within a given system. |
| **Solution Region** | The set of all ordered pairs that satisfy all inequalities in a system; visually represented as the overlapping shaded area. |
| **Solid Line** | A visual representation of a boundary line indicating that points on the line are included in the solution set (used for $\le$ or $\ge$). |
| **Vertex** | The point where the boundary lines of a system of inequalities intersect. |
