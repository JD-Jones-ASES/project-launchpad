Study Guide: More Exotic Functions

Overview

This section introduces several advanced function families beyond basic polynomials, rationals, exponentials, logarithms, and radicals. It explores piecewise functions, which use different rules over specified domain intervals; step functions, characterized by constant values that "jump" at certain points; cube root functions, defined for all real numbers; and bell curves (Gaussian functions), which are crucial in statistics and science. Understanding these functions enables modeling a wider array of real-world phenomena.

Key Concepts

  • Piecewise Functions: Functions defined by multiple sub-functions, each applied to a specific interval of the domain. They are essential for modeling situations with varying conditions, such as tax brackets or tiered pricing.
  • Continuity at Breakpoints: For a piecewise function to be continuous at a point where its definition changes, the function must be defined there, and the values approached from both the left and right sides of that point must be equal to the function's value at that point. Otherwise, a jump discontinuity occurs.
  • Absolute Value as Piecewise: The absolute value function, $|x|$, is inherently a piecewise function, defined as $x$ for $x \ge 0$ and $-x$ for $x < 0$. Any absolute value expression can be rewritten in this form.
  • Step Functions: A type of piecewise function where each piece is a constant value, leading to a graph of horizontal segments that "jump" vertically at specific points. The floor and ceiling functions are prime examples.
  • Floor and Ceiling Functions: The floor function, $\lfloor x \rfloor$, returns the greatest integer less than or equal to $x$. The ceiling function, $\lceil x \rceil$, returns the least integer greater than or equal to $x$. These are used for "rounding down" or "rounding up" scenarios.
  • Cube Root Functions: Functions of the form $f(x) = \sqrt[3]{x}$. Unlike square roots, they are defined for all real numbers and their graphs are symmetric about the origin (odd functions), forming an S-shape.
  • Transformations of Cube Root Functions: Similar to other parent functions, $\sqrt[3]{x}$ can be transformed by shifting, stretching, compressing, and reflecting to form $g(x) = a\sqrt[3]{x - h} + k$. The point $(h, k)$ represents the inflection point or center of the S-curve.
  • Gaussian Function (Bell Curve): A function of the form $f(x) = ae^{-(x-h)^2/(2\sigma^2)}$. It describes a symmetric, bell-shaped curve with a peak at $x=h$, height $a$, and width controlled by $\sigma$ (the spread). It's fundamental to probability and statistics (normal distribution).
  • Features of a Bell Curve: The peak is at the center of symmetry ($x=h$), the maximum value is $a$, and the curve approaches a horizontal asymptote at $y=0$. Inflection points occur where the curve changes concavity, located at $x = h \pm \sigma$.

Important Terms

  • Piecewise Function : A function defined by two or more expressions, each applied to a specified interval of the domain.
  • Breakpoint : A value of $x$ in a piecewise function where the definition (the rule/formula) of the function changes.
  • Jump Discontinuity : A type of discontinuity in a piecewise function where the graph "jumps" from one value to another at a breakpoint, meaning the left-hand and right-hand limits at that point are not equal.
  • Floor Function ($\lfloor x \rfloor$) (Greatest Integer Function) : A function that returns the greatest integer less than or equal to $x$.
  • Ceiling Function ($\lceil x \rceil$) : A function that returns the least integer greater than or equal to $x$.
  • Cube Root Function : A function of the form $f(x) = \sqrt[3]{x}$, which finds the number that, when cubed, equals $x$. It has a domain and range of all real numbers.
  • Inflection Point : A point on a curve where the curvature changes direction (from concave up to concave down, or vice versa). For cube root functions, it's the center of the S-curve; for bell curves, it's where the steepest part of the curve transitions.
  • Gaussian Function (Bell Curve) : A symmetric, bell-shaped function, commonly written as $f(x) = ae^{-(x-h)^2/(2\sigma^2)}$, used to model data distributions, especially the normal distribution in statistics.
  • Normal Distribution : A common continuous probability distribution in statistics, characterized by its bell-shaped curve (Gaussian function).

Detailed Notes

Introduction to Exotic Functions

While familiar function families (polynomials, rationals, exponentials, logarithms, and radicals) cover many applications, certain situations require more specialized functions. This section introduces piecewise functions, step functions, cube root functions, and bell curves, which appear frequently in mathematics and real-world modeling.

Piecewise Functions

  • Definition and Notation: A piecewise function applies different algebraic rules (expressions) to different parts of its domain.
    • General form: $$ f(x) = \begin{cases} f_1(x) & \text{if } x \in I_1 \ f_2(x) & \text{if } x \in I_2 \ \dots \ f_n(x) & \text{if } x \in I_n \end{cases} $$ where $I_k$ are non-overlapping intervals covering the entire domain.
  • Evaluation: To find $f(a)$, identify which interval $a$ falls into, then use only the corresponding formula.
    • Example: For $f(x) = \begin{cases} x^2 - 1 & \text{if } x < 0 \ 3 & \text{if } x = 0 \ 2x + 1 & \text{if } x > 0 \end{cases}$:
      • $f(-3)$: uses $x^2 - 1$, so $f(-3) = (-3)^2 - 1 = 8$.
      • $f(0)$: uses $3$, so $f(0) = 3$.
      • $f(4)$: uses $2x + 1$, so $f(4) = 2(4) + 1 = 9$.
  • Graphing:
    • Graph each piece on its designated interval.
    • Use a solid dot ($\bullet$) for endpoints included in the interval ($\le, \ge$).
    • Use an open dot ($\circ$) for endpoints excluded from the interval ($<, >$).
    • Observation: At breakpoints, the graph might have a jump if the values from different pieces don't meet.
  • Checking Continuity at Breakpoints: A piecewise function is continuous at a breakpoint $x=c$ if:
    1. $f(c)$ is defined.
    2. The value of the left piece at $x=c$ equals the value of the right piece at $x=c$.
    3. This common value also equals $f(c)$.
    • Informally, the graph connects without jumps or holes.
    • Formal Check: $\lim_{x \to c^-} f(x) = \lim_{x \to c^+} f(x) = f(c)$. If values differ, it's a jump discontinuity.
    • Example 1 (Continuous): $f(x) = \begin{cases} 2x + 1 & x \le 3 \ x + 4 & x > 3 \end{cases}$ At $x=3$, left piece $2(3)+1 = 7$; right piece $3+4=7$. Since $f(3)=7$, it's continuous.
    • Example 2 (Discontinuous): $g(x) = \begin{cases} x^2 - 2 & x < 1 \ 3x & x \ge 1 \end{cases}$ At $x=1$, left piece $1^2-2 = -1$; right piece $3(1)=3$. Since $-1 \ne 3$, there's a jump discontinuity.
    • Finding 'k' for Continuity: To make $f(x) = \begin{cases} kx + 1 & \text{if } x < 2 \ 3x - 1 & \text{if } x \ge 2 \end{cases}$ continuous at $x=2$, set the left and right values equal: $k(2)+1 = 3(2)-1 \Rightarrow 2k+1 = 5 \Rightarrow 2k=4 \Rightarrow k=2$.
  • Absolute Value as a Piecewise Function:
    • $|x| = \begin{cases} x & x \ge 0 \ -x & x < 0 \end{cases}$
    • To rewrite $|ax+b|$ as piecewise, find where $ax+b=0$.
    • Example: $|2x - 6|$ becomes $\begin{cases} 2x - 6 & x \ge 3 \ -(2x - 6) = -2x + 6 & x < 3 \end{cases}$.
  • Real-World Models: Piecewise functions are perfect for modeling tiered pricing, tax brackets, and other situations where rules change based on thresholds.
    • Cell Phone Plan: Flat fee up to a limit, then additional cost per unit beyond the limit.
    • Income Tax: Different tax rates apply to different income brackets. The tax on lower brackets is accumulated and added to the tax on the current bracket's income.

Step Functions

  • Definition: A piecewise-constant function whose graph consists of horizontal line segments and "jumps" at specific values.
  • Floor and Ceiling Functions:
    • Floor function ($\lfloor x \rfloor$): Returns the greatest integer less than or equal to $x$. (e.g., $\lfloor 3.7 \rfloor = 3$, $\lfloor -2.1 \rfloor = -3$, $\lfloor 5 \rfloor = 5$). Also called the greatest integer function.
    • Ceiling function ($\lceil x \rceil$): Returns the least integer greater than or equal to $x$. (e.g., $\lceil 3.7 \rceil = 4$, $\lceil -2.1 \rceil = -2$, $\lceil 5 \rceil = 5$).
    • Applications: "rounding down" (floor) like calculating complete years from months, or "rounding up" (ceiling) like determining the number of boxes needed for items.
  • Real-World Model (Postage): Cost is often constant for a weight range, then increases for the next range (e.g., first ounce is $X, each additional ounce or fraction is $Y). This uses the ceiling function (implicitly or explicitly).
    • Example: $C(w) = \begin{cases} 5 & 0 < w \le 1 \ 5 + 2 \lceil w - 1 \rceil & w > 1 \end{cases}$. For $C(3.4)$, $\lceil 3.4 - 1 \rceil = \lceil 2.4 \rceil = 3$, so $C(3.4) = 5 + 2(3) = $11$.

Cube Root Functions

  • Parent Function $f(x) = \sqrt[3]{x}$:
    • Domain: $(-\infty, \infty)$ (all real numbers) - unlike square roots, cube roots are defined for negative numbers.
    • Range: $(-\infty, \infty)$ (all real numbers).
    • Intercept: $(0, 0)$.
    • Symmetry: Odd function ($f(-x) = -f(x)$), symmetric about the origin.
    • Shape: An "S-curve" that increases through the origin, steep near $x=0$ and flattening as $|x|$ increases.
  • Transformations: $g(x) = a\sqrt[3]{x - h} + k$
    • $h$: Horizontal shift (right if $h>0$, left if $h<0$). The central point shifts from $(0,0)$ to $(h,0)$.
    • $k$: Vertical shift (up if $k>0$, down if $k<0$). The central point shifts from $(h,0)$ to $(h,k)$.
    • $a$: Vertical stretch/compression. If $|a|>1$, stretch; if $0<|a|<1$, compress. If $a<0$, reflection across the x-axis.
    • The point $(h, k)$ is the inflection point of the transformed cube root function.
    • Example: $g(x) = -2\sqrt[3]{x + 1} + 3$ implies a shift left by 1, vertical stretch by 2, reflection over the x-axis, and shift up by 3. The inflection point is $(-1, 3)$.

Bell Curves and Other Functions

  • Gaussian Function (Bell Curve):
    • Basic form: $f(x) = e^{-x^2}$.
    • General form: $f(x) = ae^{-(x-h)^2/(2\sigma^2)}$
      • $h$: Horizontal shift, determines the location of the peak.
      • $a$: Vertical stretch, determines the height of the peak.
      • $\sigma$ (sigma): "Spread" or standard deviation, controls the width of the curve. A larger $\sigma$ means a wider, flatter curve.
    • This function is foundational for the normal distribution in statistics.
  • Key Features of a Bell Curve:
    • Symmetry: Symmetric about the vertical line $x=h$.
    • Maximum: Occurs at $x=h$, with a maximum value of $a$.
    • Asymptote: Approaches $y=0$ as $x \to \pm\infty$ (horizontal asymptote).
    • Domain: $(-\infty, \infty)$.
    • Range: $(0, a]$ (or $[0, a]$ if it can reach zero, but generally it approaches it).
    • Inflection Points: Occur at $x = h \pm \sigma$, where the curve changes from concave down (curving like an upside-down U) to concave up (curving like a right-side-up U).
  • Identifying Features (Example): For $f(x) = 5e^{-(x-2)^2/8}$:
    • Compare to $ae^{-(x-h)^2/(2\sigma^2)}$:
      • Peak ($h$): $x=2$.
      • Height ($a$): $5$.
      • $2\sigma^2 = 8 \Rightarrow \sigma^2 = 4 \Rightarrow \sigma = 2$.
      • Inflection points ($h \pm \sigma$): $2 \pm 2$, so at $x=0$ and $x=4$.
  • Role of Technology: Graphing calculators and software are invaluable for analyzing these functions, especially for finding zeros, extrema, and inflection points when algebraic methods are complex. The focus is on understanding shape and interpreting features in context.

Review Questions

  1. Describe a real-world scenario (different from the textbook examples) that could be modeled by a piecewise function. Write a general form for the function, defining your variables and intervals.
  2. Given a piecewise function, explain the steps you would take to graph it accurately, specifically focusing on how to represent endpoints at breakpoints.
  3. Explain the key differences between the floor function $\lfloor x \rfloor$ and the ceiling function $\lceil x \rceil$. Provide an example of a situation where one would be more appropriate than the other.
  4. Compare and contrast the domain, range, and symmetry of the square root function $f(x) = \sqrt{x}$ with the cube root function $g(x) = \sqrt[3]{x}$. How do these differences impact their graphs?
  5. A bell curve function is given by $f(x) = 10e^{-(x-5)^2/18}$. Identify the location and height of its peak, and calculate the approximate locations of its inflection points. What does the value of $18$ tell you about the shape of this particular bell curve?
