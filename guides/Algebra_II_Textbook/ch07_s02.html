Study Guide: Exponential Functions

Overview

This section introduces exponential functions, a type of function where the variable appears in the exponent, leading to rapid growth or decay. We'll explore their fundamental properties, how to graph them using transformations, the significance of the natural base e, and a practical application in calculating compound interest.

Key Concepts

  • Definition of an Exponential Function: A function of the form $f(x) = b^x$, where the base $b$ is a positive constant not equal to 1 ($b > 0, b \ne 1$), and the exponent $x$ can be any real number.
  • Key Properties: For $f(x) = b^x$:
    • Domain: All real numbers $(-\infty, \infty)$.
    • Range: All positive real numbers $(0, \infty)$.
    • Y-intercept: Always $(0, 1)$, since $b^0 = 1$.
    • Horizontal Asymptote: The x-axis, $y = 0$.
    • Growth/Decay: If $b > 1$, the function is increasing (exponential growth). If $0 < b < 1$, the function is decreasing (exponential decay).
    • One-to-One: Passes the Horizontal Line Test, meaning each y-value corresponds to only one x-value.
  • Transformations: Like other functions, exponential functions can be shifted vertically or horizontally, stretched, or reflected. For $g(x) = b^{x-h} + k$, $h$ causes a horizontal shift and $k$ causes a vertical shift.
  • The Natural Base e: An important irrational number approximately equal to 2.71828. It arises naturally in many growth and decay processes and is the base for the natural exponential function, $f(x) = e^x$.
  • Compound Interest: An application of exponential functions where interest is earned not only on the initial principal but also on accumulated interest. The formula is $A = P(1 + r/n)^{nt}$, where $A$ is the final amount, $P$ is the principal, $r$ is the annual interest rate (as a decimal), $n$ is the number of times interest is compounded per year, and $t$ is the time in years.

Important Terms

  • Exponential Function: A function where the independent variable is the exponent, typically written as $f(x) = b^x$ for a positive base $b \ne 1$.
  • Base ($b$): The constant value that is raised to the power of the variable $x$ in an exponential function.
  • Exponent ($x$): The variable in an exponential function, indicating the power to which the base is raised.
  • Exponential Growth: Occurs when the base $b > 1$, causing the function's output to increase at an accelerating rate.
  • Exponential Decay: Occurs when the base $0 < b < 1$, causing the function's output to decrease at an accelerating rate towards zero.
  • Natural Base e: An irrational mathematical constant approximately 2.71828, fundamental in calculus and various scientific applications.
  • Natural Exponential Function: The exponential function with base $e$, written as $f(x) = e^x$.
  • Compound Interest: Interest calculated on the initial principal and also on all accumulated interest from previous periods.
  • Principal ($P$): The initial amount of money invested or borrowed.
  • Annual Interest Rate ($r$): The stated interest rate per year, expressed as a decimal in formulas.
  • Compounding Frequency ($n$): The number of times interest is calculated and added to the principal within one year.
  • Time ($t$): The duration, in years, for which the money is invested or borrowed.
  • Amount ($A$): The final value of an investment or loan after a certain period, including both principal and accumulated interest.
  • Horizontal Asymptote: A horizontal line that the graph of a function approaches as the input (x-value) tends towards positive or negative infinity.

Detailed Notes

Introduction to Exponential Functions

Unlike polynomial functions like $x^3$ where the variable is the base, an exponential function places the variable in the exponent, as in $3^x$. This seemingly minor difference results in dramatically different behavior, characterized by extremely rapid growth or decay.

The general form of an exponential function is $f(x) = b^x$, where:

  • The base $b$ must be a positive number ($b > 0$).
  • The base $b$ cannot be equal to 1 ($b \ne 1$). If $b=1$, then $f(x) = 1^x = 1$, which is a constant function, not exponential.
  • The exponent $x$ can be any real number.

Key Properties of $f(x) = b^x$

Understanding these properties is crucial for graphing and analyzing exponential functions:

  • Domain: The set of all possible $x$-values is $(-\infty, \infty)$. You can raise any positive number (except 1) to any real power.
  • Range: The set of all possible $y$-values is $(0, \infty)$. The output of $b^x$ will always be positive, but it will never actually reach zero.
  • Y-intercept: For any base $b$, the $y$-intercept is always $(0, 1)$. This is because $f(0) = b^0 = 1$.
  • Horizontal Asymptote: The line $y = 0$ (the x-axis) is a horizontal asymptote. As $x$ approaches $-\infty$ (for growth functions) or $\infty$ (for decay functions), the function's value gets closer and closer to 0 but never touches it.
  • Growth vs. Decay:
    • If $b > 1$ (e.g., $2^x, 3^x$), the function exhibits exponential growth, meaning it is always increasing.
    • If $0 < b < 1$ (e.g., $(1/2)^x, (0.5)^x$), the function exhibits exponential decay, meaning it is always decreasing.
    • Note that $(1/b)^x = b^{-x}$, which means the graph of $y = (1/b)^x$ is a reflection of $y = b^x$ across the $y$-axis.
  • One-to-One: Exponential functions pass the Horizontal Line Test, meaning for every unique $y$-value, there is only one unique $x$-value. This property is important for inverse functions (logarithms).

Example: Evaluating an Exponential Function Let $f(x) = 3^x$.

  • $f(0) = 3^0 = 1$
  • $f(2) = 3^2 = 9$
  • $f(-1) = 3^{-1} = \frac{1}{3}$
  • $f(1/2) = 3^{1/2} = \sqrt{3} \approx 1.732$

Graphing Exponential Functions with Transformations

Transformations allow us to graph more complex exponential functions based on a parent function $f(x) = b^x$. Consider a transformed function $g(x) = b^{x-h} + k$.

  • Horizontal Shift: The term $x-h$ shifts the graph horizontally. If $h > 0$, it shifts right by $h$ units. If $h < 0$, it shifts left by $|h|$ units.
  • Vertical Shift: The term $+k$ shifts the graph vertically. If $k > 0$, it shifts up by $k$ units. If $k < 0$, it shifts down by $|k|$ units.
  • Impact on Asymptote: A vertical shift by $k$ units will also shift the horizontal asymptote from $y=0$ to $y=k$.

Example: Graphing Transformations To obtain the graph of $g(x) = 2^{x-3} + 1$ from $f(x) = 2^x$:

  1. The $x-3$ in the exponent indicates a shift right 3 units.
  2. The $+1$ outside the exponent indicates a shift up 1 unit.

This means:

  • The horizontal asymptote shifts from $y=0$ to $y=1$.
  • The Domain remains $(-\infty, \infty)$.
  • The Range shifts from $(0, \infty)$ to $(1, \infty)$.
  • The y-intercept for $g(x)$ would be $g(0) = 2^{0-3} + 1 = 2^{-3} + 1 = \frac{1}{8} + 1 = \frac{9}{8}$.

The Natural Base e

The natural base $e$ is a fundamental mathematical constant, an irrational number approximately equal to $2.71828$. It naturally arises in processes involving continuous growth, such as continuously compounded interest or population growth. Mathematically, $e$ is defined as the limit of $(1 + 1/n)^n$ as $n$ approaches infinity. The exponential function with base $e$ is called the natural exponential function, $f(x) = e^x$. This function is widely used in science, engineering, and finance.

Example: Evaluating the Natural Exponential Function Using a calculator (typically denoted by $e^x$ or $EXP(x)$):

  • $e^2 \approx 7.3891$
  • $e^{-0.5} \approx 0.6065$
  • $e^{1.2} \approx 3.3201$

Compound Interest

One of the most common real-world applications of exponential functions is in calculating compound interest. When interest is compounded, it is added to the principal, and subsequent interest calculations are based on this new, larger principal.

The Compound Interest Formula is: $A = P(1 + r/n)^{nt}$

Where:

  • $A$ = the amount in the account after $t$ years (final amount)
  • $P$ = the principal amount (initial investment)
  • $r$ = the annual interest rate (expressed as a decimal, e.g., 6% = 0.06)
  • $n$ = the number of times interest is compounded per year
    • Annually: $n=1$
    • Semi-annually: $n=2$
    • Quarterly: $n=4$
    • Monthly: $n=12$
    • Daily: $n=365$
  • $t$ = the time (in years) the money is invested or borrowed for

Example: Calculating Compound Interest Suppose $5,000 is invested at 6% annual interest, compounded monthly, for 10 years. Here, $P = 5000$, $r = 0.06$, $n = 12$, and $t = 10$.

$A = 5000(1 + 0.06/12)^{12 \cdot 10}$ $A = 5000(1 + 0.005)^{120}$ $A = 5000(1.005)^{120}$ $A \approx 5000(1.8193967)$ $A \approx $9,096.98$

Checkpoint Example: If $3,000 is invested at 4% annual interest compounded quarterly for 5 years: $P = 3000$, $r = 0.04$, $n = 4$, $t = 5$. $A = 3000(1 + 0.04/4)^{4 \cdot 5}$ $A = 3000(1 + 0.01)^{20}$ $A = 3000(1.01)^{20}$ $A \approx 3000(1.22019)$ $A \approx $3,661.03$

As the compounding frequency ($n$) increases significantly, the growth approaches what is called continuous compounding, which uses the formula $A = Pe^{rt}$ (a topic covered in a later section).

Review Questions

  1. What is the defining characteristic of an exponential function, and how does its behavior (in terms of growth or decay rate) fundamentally differ from a linear or polynomial function?
  2. List and explain the four main properties of the graph of a basic exponential function, $f(x) = b^x$. How does changing the value of the base $b$ (e.g., from $b>1$ to $0<b<1$) affect these properties, particularly in terms of growth/decay and reflection?
  3. Describe the process of transforming the graph of a parent exponential function $f(x) = b^x$ to obtain the graph of $g(x) = -b^{x-h} + k$. How do these transformations impact the domain, range, and horizontal asymptote of the original function?
  4. Explain the significance of the natural base $e$. What is its approximate value, and why is the natural exponential function $f(x) = e^x$ so important in various fields?
  5. Break down the Compound Interest Formula $A = P(1 + r/n)^{nt}$. Identify each variable and explain its role in determining the final amount. How does the compounding frequency, $n$, influence the total amount of interest earned over time?
