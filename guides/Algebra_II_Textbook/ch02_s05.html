Study Guide: Systems of Linear Equations

Overview

This section introduces systems of linear equations, which involve solving two or more linear equations simultaneously. It covers the graphical interpretation of solutions, the two primary algebraic methods for solving systems (Substitution and Elimination), and how to classify systems based on the number of solutions (one, none, or infinitely many). Finally, it demonstrates how to apply these methods to solve real-world problems.

Key Concepts

  • System of Linear Equations: A set of two or more linear equations in two or more variables that are to be solved simultaneously.
  • Solution to a System: An ordered pair $(x, y)$ that satisfies all equations in the system. Graphically, this represents the point(s) where the lines intersect.
  • Substitution Method: An algebraic technique to solve systems by solving one equation for one variable and substituting that expression into the other equation.
  • Elimination (Addition) Method: An algebraic technique to solve systems by multiplying equations by constants to create opposite coefficients for one variable, then adding the equations to eliminate that variable.
  • Classifying Systems: Systems can be classified by their number of solutions:
    • Consistent and Independent: One unique solution (intersecting lines).
    • Inconsistent: No solution (parallel lines). Algebraically results in a false statement (e.g., $0 = 5$).
    • Consistent and Dependent: Infinitely many solutions (the same line). Algebraically results in a true statement (e.g., $0 = 0$).
  • Applications: Systems of linear equations are powerful tools for modeling and solving real-world problems by translating conditions into equations.

Important Terms

  • System of Linear Equations: A collection of two or more linear equations considered together. For example, $a_1 x + b_1 y = c_1$ and $a_2 x + b_2 y = c_2$.
  • Solution to a System: An ordered pair $(x, y)$ that makes all equations in the system true.
  • Substitution Method: An algebraic method for solving systems where one variable is isolated in one equation and its expression is substituted into the other equation.
  • Elimination (Addition) Method: An algebraic method for solving systems where equations are manipulated (multiplied by constants) and then added together to eliminate one variable.
  • Consistent System: A system of equations that has at least one solution.
  • Inconsistent System: A system of equations that has no solution.
  • Dependent System: A system of equations where the equations are multiples of each other, resulting in infinitely many solutions.
  • Independent System: A system of equations where the equations are distinct and not multiples of each other (can be consistent or inconsistent).

Detailed Notes

Introduction to Systems of Linear Equations

A single linear equation in two variables, like $y = 2x + 1$, has infinitely many solutions, forming a line on a graph. A system of linear equations arises when we need two (or more) such equations to be satisfied simultaneously. For a system of two linear equations in two unknowns: $$ \begin{cases} a_1 x + b_1 y = c_1 \ a_2 x + b_2 y = c_2 \end{cases} $$ A solution is an ordered pair $(x, y)$ that makes both equations true. Graphically, each equation represents a line, and a solution is the point(s) where these lines intersect. There are three possible geometric outcomes for two lines in a plane: they intersect at a single point, they are parallel and never intersect, or they are the exact same line.

The Substitution Method

The Substitution Method is an effective way to solve systems, especially when one equation is already solved for a variable or can be easily solved for one.

Steps:

  1. Solve for one variable: Choose one equation and solve it for either $x$ or $y$. (Look for equations where a variable has a coefficient of 1 or -1).
  2. Substitute: Substitute the expression from Step 1 into the other equation. This will result in a single equation with only one variable.
  3. Solve the single-variable equation: Solve the equation obtained in Step 2 for its variable.
  4. Back-substitute: Substitute the value found in Step 3 back into the expression from Step 1 (or either of the original equations) to find the value of the second variable.
  5. Check: Verify your ordered pair solution $(x, y)$ by substituting both values into both original equations.

Example: Solving by Substitution Solve the system: $$ \begin{cases} y = 2x - 3 \ 3x + y = 7 \end{cases} $$

  1. The first equation is already solved for $y$: $y = 2x - 3$.
  2. Substitute $y = 2x - 3$ into the second equation: $3x + (2x - 3) = 7$.
  3. Solve for $x$: $5x - 3 = 7$ $5x = 10$ $x = 2$
  4. Back-substitute $x = 2$ into $y = 2x - 3$: $y = 2(2) - 3 = 4 - 3 = 1$.
  5. Check: Equation 1: $1 = 2(2) - 3 \Rightarrow 1 = 4 - 3 \Rightarrow 1 = 1$ (True) Equation 2: $3(2) + 1 = 7 \Rightarrow 6 + 1 = 7 \Rightarrow 7 = 7$ (True) The solution is $(2, 1)$.

The Elimination (Addition) Method

The Elimination Method is particularly useful when variables in both equations have coefficients that are easy to make opposites.

Steps:

  1. Prepare for elimination: If necessary, multiply one or both equations by non-zero constants so that the coefficients of one variable (either $x$ or $y$) become opposites (e.g., $3x$ and $-3x$, or $2y$ and $-2y$).
  2. Add equations: Add the modified equations together. This will eliminate one variable, resulting in a single equation with one variable.
  3. Solve the single-variable equation: Solve the equation obtained in Step 2.
  4. Substitute back: Substitute the value found in Step 3 into either of the original equations to find the value of the second variable.
  5. Check: Verify your ordered pair solution $(x, y)$ by substituting both values into both original equations.

Example: Solving by Elimination Solve the system: $$ \begin{cases} 3x - 2y = 11 \ 5x + 4y = 1 \end{cases} $$

  1. We want to eliminate $y$. Multiply the first equation by $2$ to make the $y$-coefficients opposites ($-4y$ and $+4y$): Original: $\begin{cases} 3x - 2y = 11 \ 5x + 4y = 1 \end{cases}$ Multiply first by 2: $\begin{cases} 2(3x - 2y) = 2(11) \ 5x + 4y = 1 \end{cases} \Rightarrow \begin{cases} 6x - 4y = 22 \ 5x + 4y = 1 \end{cases}$
  2. Add the equations: $(6x - 4y) + (5x + 4y) = 22 + 1$ $11x = 23$
  3. Solve for $x$: $x = \frac{23}{11}$.
  4. Back-substitute $x = \frac{23}{11}$ into the first original equation ($3x - 2y = 11$): $3\left(\frac{23}{11}\right) - 2y = 11$ $\frac{69}{11} - 2y = 11$ $\frac{69}{11} - 2y = \frac{121}{11}$ (Convert 11 to a fraction with denominator 11) $-2y = \frac{121}{11} - \frac{69}{11}$ $-2y = \frac{52}{11}$ $y = \frac{52}{11} \div (-2)$ $y = \frac{52}{11} \times \left(-\frac{1}{2}\right)$ $y = -\frac{26}{11}$
  5. Check: Equation 1: $3\left(\frac{23}{11}\right) - 2\left(-\frac{26}{11}\right) = \frac{69}{11} + \frac{52}{11} = \frac{121}{11} = 11$ (True) Equation 2: $5\left(\frac{23}{11}\right) + 4\left(-\frac{26}{11}\right) = \frac{115}{11} - \frac{104}{11} = \frac{11}{11} = 1$ (True) The solution is $\left(\frac{23}{11}, -\frac{26}{11}\right)$.

Important Caution: When multiplying an entire equation by a constant in the elimination method, every term on both sides of the equation must be multiplied. A common error is to forget to multiply the constant term on the right-hand side or to drop a negative sign. Always double-check this step. For example, multiplying $3x - 2y = 11$ by $-2$ gives $-6x + 4y = -22$, not $-6x + 4y = 22$.

Classifying Systems

The algebraic process of solving a system will reveal one of three possible outcomes, which correspond to the three graphical cases:

  1. Consistent and Independent:

    • Graphical Representation: The lines intersect at exactly one point.
    • Algebraic Signal: You will arrive at a unique value for $x$ and a unique value for $y$. This is the "normal" case when you successfully find a single solution.
    • Number of Solutions: One unique solution.

  2. Inconsistent:

    • Graphical Representation: The lines are parallel and never intersect (they have the same slope but different $y$-intercepts).
    • Algebraic Signal: When solving, you will eliminate both variables and end up with a false equation (e.g., $0 = 5$, or $3 = -2$). This contradiction means there is no solution that satisfies both equations simultaneously.
    • Number of Solutions: No solution.

    Example: Inconsistent System Classify the system: $$ \begin{cases} 2x - y = 4 \ 4x - 2y = 9 \end{cases} $$ Multiply the first equation by $-2$: $$ \begin{cases} -4x + 2y = -8 \ 4x - 2y = 9 \end{cases} $$ Add the equations: $(-4x + 2y) + (4x - 2y) = -8 + 9 \Rightarrow 0 = 1$. This is a false statement. Conclusion: The system is inconsistent and has no solution. The lines are parallel.

  3. Consistent and Dependent:

    • Graphical Representation: The equations describe the same line (one equation is a multiple of the other).
    • Algebraic Signal: When solving, you will eliminate both variables and end up with a true equation (e.g., $0 = 0$, or $7 = 7$). This indicates that the equations are equivalent, and any solution to one is a solution to the other.
    • Number of Solutions: Infinitely many solutions. The solution set is typically expressed in terms of one of the original equations, e.g., ${(x, y) \mid x + 2y = 6}$.

    Example: Dependent System Classify the system: $$ \begin{cases} x + 2y = 6 \ 3x + 6y = 18 \end{cases} $$ Multiply the first equation by $-3$: $$ \begin{cases} -3x - 6y = -18 \ 3x + 6y = 18 \end{cases} $$ Add the equations: $(-3x - 6y) + (3x + 6y) = -18 + 18 \Rightarrow 0 = 0$. This is an always true statement. Conclusion: The system is consistent and dependent, with infinitely many solutions. The solution set is ${(x, y) \mid x + 2y = 6}$.

Applications of Systems

Systems of linear equations are widely used to solve word problems by translating given information into algebraic relationships.

General Approach for Word Problems:

  1. Define Variables: Clearly state what each variable represents.
  2. Set Up Equations: Use the information given in the problem to write two or more linear equations. Each piece of independent information usually forms an equation.
  3. Solve the System: Use either the Substitution or Elimination Method to find the values of the variables.
  4. State the Answer: Write your answer in the context of the problem, using appropriate units.
  5. Check: Make sure your solution makes sense in the original word problem.

Example: Theater Ticket Sales A theater sells adult tickets for $9 and child tickets for $5. For one performance, 200 tickets were sold totaling $1,400. How many adult and child tickets were sold?

  1. Define variables: Let $a$ = number of adult tickets Let $c$ = number of child tickets

  2. Set up the system:

    • Total tickets sold: $a + c = 200$
    • Total revenue: $9a + 5c = 1400$ The system is: $$ \begin{cases} a + c = 200 \ 9a + 5c = 1400 \end{cases} $$

  3. Solve by elimination: Multiply the first equation by $-5$ to eliminate $c$: $$ \begin{cases} -5a - 5c = -1000 \ 9a + 5c = 1400 \end{cases} $$ Add the equations: $(-5a - 5c) + (9a + 5c) = -1000 + 1400$ $4a = 400$ $a = 100$ Back-substitute $a = 100$ into the first equation ($a + c = 200$): $100 + c = 200$ $c = 100$

  4. State the answer: 100 adult tickets and 100 child tickets were sold.

  5. Check: Total tickets: $100 + 100 = 200$ (Correct) Total revenue: $9(100) + 5(100) = 900 + 500 = 1400$ (Correct)

Note on Units: When setting up equations from word problems, ensure that each equation has consistent units. For instance, an equation should not mix "number of tickets" with "total dollars" directly. Separate equations for "total quantity" and "total value" are common.

Review Questions

  1. Explain what a "system of linear equations" is and what a "solution to the system" represents both algebraically and graphically.
  2. Describe the steps involved in solving a system of linear equations using the Substitution Method. In what scenarios might this method be more efficient than the Elimination Method?
  3. Describe the steps involved in solving a system of linear equations using the Elimination Method. What is a crucial point to remember when multiplying an equation by a constant in this method?
  4. Explain the three classifications of systems of linear equations (consistent/independent, inconsistent, consistent/dependent). For each type, describe its graphical appearance and the algebraic signal you would observe when attempting to solve it.
  5. A candy store sells two types of chocolate, dark and milk. Dark chocolate costs $20 per pound, and milk chocolate costs $15 per pound. If a customer buys a total of 3 pounds of chocolate for $50, set up a system of linear equations to represent this situation. Which method (Substitution or Elimination) would you choose to solve it and why? (You do not need to solve the system).
