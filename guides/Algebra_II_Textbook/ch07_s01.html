Study Guide: Inverse Functions

Overview

This section introduces the fundamental concept of inverse functions, which essentially "undo" the action of an original function. It establishes that only "one-to-one" functions, where each output corresponds to a unique input, can have an inverse. The study guide details how to identify one-to-one functions, algebraically find an inverse, understand the swap of domain and range, and verify if two functions are indeed inverses of each other.

Key Concepts

  • One-to-One Function: A function where every element of the range corresponds to exactly one element of the domain. In simpler terms, different inputs always produce different outputs ($f(a) = f(b) \implies a = b$).
  • Horizontal Line Test: A graphical method to determine if a function is one-to-one. If any horizontal line intersects the graph of a function at most once, the function is one-to-one.
  • Inverse Function ($f^{-1}$): For a one-to-one function $f$, its inverse function $f^{-1}$ reverses the mapping. If $f(x) = y$, then $f^{-1}(y) = x$. Crucially, applying a function and its inverse in sequence brings you back to the original input: $f^{-1}(f(x)) = x$ and $f(f^{-1}(y)) = y$.
  • Domain and Range Swap: The domain of a function $f$ becomes the range of its inverse $f^{-1}$, and the range of $f$ becomes the domain of $f^{-1}$.
  • Algebraic Method for Finding an Inverse: A systematic procedure involving replacing $f(x)$ with $y$, swapping $x$ and $y$, solving for the new $y$, and then replacing $y$ with $f^{-1}(x)$.
  • Verification of Inverses: Two functions $f$ and $g$ are inverses of each other if and only if their compositions result in the identity function: $f(g(x)) = x$ AND $g(f(x)) = x$.

Important Terms

  • One-to-One Function: A function $f$ for which $f(a) = f(b)$ implies that $a = b$. This means distinct inputs always map to distinct outputs.
  • Horizontal Line Test: A graphical test to determine if a function is one-to-one. A function passes the test if no horizontal line intersects its graph at more than one point.
  • Inverse Function ($f^{-1}$): The function that reverses the action of a given one-to-one function $f$. It takes outputs of $f$ back to their original inputs. The notation $f^{-1}(x)$ is not the same as $1/f(x)$.
  • Domain: The set of all possible input values (independent variable) for which a function is defined.
  • Range: The set of all possible output values (dependent variable) that a function can produce.
  • Composition of Functions: An operation that combines two functions such that the output of one function becomes the input of the other. Denoted as $(f \circ g)(x) = f(g(x))$.

Detailed Notes

The Concept of Inverse Functions

Many processes in the real world are reversible, like converting units (Celsius to Fahrenheit and back) or encrypting and decrypting a message. Mathematically, an inverse function allows us to "undo" the action of an original function.

One-to-One Functions

Before a function can have an inverse, it must be one-to-one.

  • Definition: A function $f$ is one-to-one if for any two distinct inputs, $a$ and $b$, their outputs are also distinct. Formally, $f(a) = f(b)$ implies $a = b$.
  • Graphical Test (Horizontal Line Test): To check if a function is one-to-one from its graph, draw horizontal lines across it. If any horizontal line intersects the graph at more than one point, the function is not one-to-one. If every horizontal line intersects the graph at most once, then the function is one-to-one.
  • Why one-to-one is necessary: If a function were not one-to-one, multiple inputs would lead to the same output. Its inverse would then have to map a single output back to multiple inputs, which would violate the definition of a function (one input can only have one output).

Definition and Properties of an Inverse Function

If $f$ is a one-to-one function with domain $D$ and range $R$, its inverse function, denoted $f^{-1}$, has the following properties:

  • Domain and Range Swap: The domain of $f^{-1}$ is the range of $f$ (D_{f^{-1}} = R_f), and the range of $f^{-1}$ is the domain of $f$ (R_{f^{-1}} = D_f). This is a crucial consequence of "undoing" the function.
  • Composition Property:
    • $f^{-1}(f(x)) = x$ for all $x$ in the domain of $f$.
    • $f(f^{-1}(y)) = y$ for all $y$ in the domain of $f^{-1}$ (which is the range of $f$).
  • Notation Alert: Remember that $f^{-1}(x)$ denotes the inverse function, not the reciprocal $1/f(x)$. These are generally not equal.

Finding the Inverse Algebraically

Here are the steps to find the inverse of a function $f(x)$ when given by a formula:

  1. Replace $f(x)$ with $y$: This makes the equation easier to manipulate.
  2. Swap $x$ and $y$: This is the core step that expresses the inverse relationship, as $x$ (input) becomes $y$ (output) and vice-versa for the inverse.
  3. Solve for $y$: Isolate $y$ on one side of the equation. This new $y$ represents the output of the inverse function.
  4. Replace $y$ with $f^{-1}(x)$: This gives you the formula for the inverse function.
  5. State the domain of $f^{-1}$: Remember that the domain of $f^{-1}$ is the range of the original function $f$. It's important to specify any restrictions.

Examples:

  • Linear Function: $f(x) = 3x - 5$

    1. $y = 3x - 5$
    2. $x = 3y - 5$
    3. $x + 5 = 3y \implies y = \frac{x+5}{3}$
    4. $f^{-1}(x) = \frac{x+5}{3}$
    • Domain/Range: Both $f(x)$ and $f^{-1}(x)$ have a domain and range of all real numbers ($\mathbb{R}$).

  • Rational Function: $g(x) = \frac{2x+1}{x-3}$, where $x \neq 3$.

    1. $y = \frac{2x+1}{x-3}$
    2. $x = \frac{2y+1}{y-3}$
    3. $x(y-3) = 2y+1 \implies xy - 3x = 2y+1 \implies xy - 2y = 3x+1 \implies y(x-2) = 3x+1 \implies y = \frac{3x+1}{x-2}$
    4. $g^{-1}(x) = \frac{3x+1}{x-2}$
    • Domain/Range: The original function $g(x)$ has domain $x \neq 3$ and range $y \neq 2$ (since $y = 2$ is a horizontal asymptote). Thus, $g^{-1}(x)$ has domain $x \neq 2$ and range $y \neq 3$. Notice how the excluded values (from the domain and range) swap.

  • Square Root Function: $h(x) = \sqrt{x-1}$, where $x \ge 1$.

    1. $y = \sqrt{x-1}$
    2. $x = \sqrt{y-1}$
    3. $x^2 = y-1$ (Note: Since $\sqrt{y-1}$ is always non-negative, $x$ must be $\ge 0$ for the inverse. This is the range of $h(x)$).
    4. $y = x^2 + 1$
    5. $h^{-1}(x) = x^2 + 1$, with the domain restriction $x \ge 0$.
    • Domain/Range: $h(x)$ has domain $[1, \infty)$ and range $[0, \infty)$. Therefore, $h^{-1}(x)$ has domain $[0, \infty)$ and range $[1, \infty)$.

Verifying Inverse Functions

To confirm that two functions $f$ and $g$ are inverses of each other, you must check both compositions:

  • $(f \circ g)(x) = f(g(x)) = x$
  • $(g \circ f)(x) = g(f(x)) = x$ Both compositions must simplify to $x$ for all $x$ in their respective domains.

Example: Verify $f(x) = 3x-5$ and $g(x) = \frac{x+5}{3}$ are inverses.

  • $f(g(x)) = f(\frac{x+5}{3}) = 3(\frac{x+5}{3}) - 5 = (x+5) - 5 = x$.
  • $g(f(x)) = g(3x-5) = \frac{(3x-5)+5}{3} = \frac{3x}{3} = x$. Since both compositions equal $x$, they are indeed inverse functions.

Applications

Inverse functions are useful in converting between different systems, decoding information, or understanding reversible physical processes. For instance, the Celsius to Fahrenheit conversion $C(F) = \frac{5}{9}(F-32)$ has an inverse $C^{-1}(x) = \frac{9}{5}x + 32$, which converts Fahrenheit to Celsius.

Review Questions

  1. What is a one-to-one function, and why is this property essential for a function to have an inverse? Explain using both the formal definition and the Horizontal Line Test.
  2. Describe the step-by-step algebraic process for finding the inverse of a function. Apply these steps to find the inverse of $f(x) = 4x+7$.
  3. How do the domain and range of a function relate to the domain and range of its inverse? Provide an example of a function (like $h(x) = \sqrt{x-1}$) where the domain and range swap explicitly involves a restriction.
  4. Explain how to algebraically verify if two functions $f(x)$ and $g(x)$ are inverses of each other. Why is it necessary to check both $f(g(x))$ and $g(f(x))$?
  5. Consider a function that converts your height in inches to centimeters ($C(i) = 2.54i$). Find its inverse function $C^{-1}(c)$ and explain what it represents in the context of height measurement.
