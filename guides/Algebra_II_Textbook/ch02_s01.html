Linear Functions: Chapter 2, Section 1: Function Basics Study Guide

Overview

This section introduces the fundamental concept of a function, defining it as a special type of relation where each input has exactly one output. It covers various ways to identify and work with functions, including using the Vertical Line Test for graphs, function notation for evaluation, determining the domain from a formula, understanding piecewise functions, and identifying one-to-one functions using the Horizontal Line Test.

Key Concepts

  • Relation and Function: A relation is any set of ordered pairs. A function is a specific type of relation where every input (first coordinate) is paired with exactly one output (second coordinate).
  • Domain and Range: For any relation or function, the domain is the set of all possible input values ($x$-coordinates), and the range is the set of all possible output values ($y$-coordinates).
  • Vertical Line Test: A visual tool to determine if a graph represents a function. If any vertical line intersects the graph at more than one point, it is not a function. This is because a single input would then correspond to multiple outputs.
  • Function Notation ($f(x)$): A compact way to represent functions, where $f(x)$ denotes the output of function $f$ for a given input $x$. It is not multiplication, but rather a way to express dependence.
  • Implied Domain: When a function is given by a formula, its domain is assumed to be all real numbers for which the formula produces a real-number output. Common restrictions arise from square roots (radicand $\ge 0$) and fractions (denominator $\neq 0$).
  • Piecewise-Defined Functions: Functions defined by different formulas for different parts of their domain. To evaluate, you must first determine which domain condition the input satisfies.
  • One-to-One Functions: A special type of function where every output corresponds to exactly one input. This means no two different inputs can produce the same output.
  • Horizontal Line Test: A visual tool to determine if a function is one-to-one. If any horizontal line intersects the function's graph at more than one point, the function is not one-to-one.

Important Terms

  • Relation: Any set of ordered pairs $(x, y)$.
  • Domain: The set of all first coordinates (inputs) of a relation or function.
  • Range: The set of all second coordinates (outputs) of a relation or function.
  • Function: A relation in which every element of the domain is paired with exactly one element of the range.
  • Vertical Line Test: A rule stating that a graph represents a function if and only if every vertical line intersects the graph in at most one point.
  • Function Notation: The convention of writing $y = f(x)$ to denote that $y$ is a function of $x$, where $f(x)$ represents the output value.
  • Piecewise Function: A function defined by multiple sub-functions, each applying to a different interval of the main function's domain.
  • One-to-One Function: A function where each output value in the range corresponds to exactly one input value in the domain. No two distinct inputs produce the same output.
  • Horizontal Line Test: A rule stating that a function is one-to-one if and only if every horizontal line intersects its graph in at most one point.

Detailed Notes

1. Understanding Relations and Functions

  • Relation: At its core, a relation is simply a collection of input-output pairs. These pairs are typically written as ordered pairs $(x, y)$.
  • Domain and Range of a Relation:
    • The domain is the set of all $x$-values (first coordinates) present in the ordered pairs.
    • The range is the set of all $y$-values (second coordinates) present in the ordered pairs.
  • Defining a Function: A function is a special type of relation. The crucial restriction is that each input ($x$-value) must correspond to exactly one output ($y$-value).
    • This means you cannot have two different ordered pairs like $(a, y_1)$ and $(a, y_2)$ where $y_1 \neq y_2$. The same input must always lead to the same output.
    • Example (from text):
      • Table A: $(x,y)$ values: $(-2, 5), (0, 3), (1, 5), (4, -1)$.
        • Each $x$-value is unique, so each input has exactly one output. This is a function.
      • Graph contains points: $(-1, 2), (-1, -3), (2, 4)$.
        • The input $x = -1$ is paired with two different outputs ($y=2$ and $y=-3$). This is not a function.

2. The Vertical Line Test

  • Visual Identification: When a relation is graphed on a coordinate plane, the Vertical Line Test provides a quick way to check if it's a function.
  • How it Works: Imagine drawing vertical lines across the graph.
    • If any vertical line intersects the graph at more than one point, then the graph does not represent a function.
    • If every vertical line intersects the graph at most once (i.e., one point or no points), then the graph does represent a function.
  • Why it Works: A vertical line represents a single input value ($x=a$). If this line hits the graph at two different points, say $(a, y_1)$ and $(a, y_2)$, it means the input $a$ produces two different outputs $y_1$ and $y_2$, which violates the definition of a function.

3. Function Notation

  • Writing Functions: When $y$ is a function of $x$, we use the notation $y = f(x)$, read as "$f$ of $x$".
    • $f$ is the name of the function.
    • $x$ is the input variable.
    • $f(x)$ represents the output value (which is $y$) corresponding to the input $x$.
  • Important Note: $f(x)$ does not mean $f$ multiplied by $x$. It indicates the value of the function $f$ at the input $x$.
  • Evaluating Functions: To find the output for a specific input, substitute that input value for every occurrence of $x$ in the function's formula.
    • Example (from text): Let $f(x) = 2x^2 - 3x + 1$.
      • $f(0) = 2(0)^2 - 3(0) + 1 = 1$
      • $f(-2) = 2(-2)^2 - 3(-2) + 1 = 2(4) + 6 + 1 = 8 + 6 + 1 = 15$
      • $f(a+1) = 2(a+1)^2 - 3(a+1) + 1 = 2(a^2 + 2a + 1) - 3a - 3 + 1 = 2a^2 + 4a + 2 - 3a - 3 + 1 = 2a^2 + a$
    • Common Mistake: Remember that $f(a+1) \neq f(a) + f(1)$. Always substitute the entire expression into the function.

4. Domain and Range from a Formula

  • Implied Domain: If a function's domain is not explicitly stated, it is assumed to be all real numbers for which the formula yields a real-number output.
  • Key Restrictions on the Domain:
    1. Square Roots (and other even-indexed radicals): The expression inside the radical (the radicand) must be greater than or equal to zero ($\ge 0$). We cannot take the square root of a negative number and get a real result.
      • Example (from text): $g(x) = \sqrt{x - 4}$.
        • We need $x - 4 \ge 0$, so $x \ge 4$.
        • Domain: $[4, \infty)$
    2. Fractions: The denominator of a fraction cannot be equal to zero. Division by zero is undefined.
      • Example (from text): $h(x) = \frac{1}{x - 3}$.
        • We need $x - 3 \neq 0$, so $x \neq 3$.
        • Domain: $(-\infty, 3) \cup (3, \infty)$

5. Piecewise-Defined Functions

  • Definition: A piecewise function uses different formulas (or "pieces") to define its output, depending on which part of the domain the input falls into.
  • Evaluation: To evaluate a piecewise function for a given input $x$:
    1. Look at the conditions for each piece.
    2. Determine which condition the input $x$ satisfies.
    3. Use only the formula corresponding to that satisfied condition.
  • Example (from text): Let $f(x) = \begin{cases} x^2 & \text{if } x < 0 \ 2x + 1 & \text{if } x \ge 0 \end{cases}$
    • $f(-3)$: Since $-3 < 0$, use $f(x) = x^2 \Rightarrow f(-3) = (-3)^2 = 9$.
    • $f(0)$: Since $0 \ge 0$, use $f(x) = 2x + 1 \Rightarrow f(0) = 2(0) + 1 = 1$.
    • $f(5)$: Since $5 \ge 0$, use $f(x) = 2x + 1 \Rightarrow f(5) = 2(5) + 1 = 11$.

6. One-to-One Functions

  • Definition: A function is one-to-one if distinct inputs always produce distinct outputs. That is, if $f(x_1) = f(x_2)$, then $x_1$ must equal $x_2$. Equivalently, no two different inputs produce the same output.
  • Horizontal Line Test: This test helps determine visually if a function is one-to-one.
    • If any horizontal line intersects the graph of a function at more than one point, then the function is not one-to-one.
    • If every horizontal line intersects the graph at most once, then the function is one-to-one.
  • Significance: One-to-one functions are important because they are the only functions that have inverse functions.
  • Solving for an Input: You might be asked to find the input $x$ that produces a specific output $f(x)$.
    • Example (from text): Given $f(x) = 3x - 1$, find $x$ such that $f(x) = 14$.
      • Set $3x - 1 = 14$.
      • $3x = 15$.
      • $x = 5$.

Review Questions

  1. Explain the fundamental difference between a "relation" and a "function," providing an example of a relation that is not a function.
  2. Describe the purpose of the Vertical Line Test. If a graph fails this test, what does that imply about the inputs and outputs of the relation it represents?
  3. Given the function $h(x) = -x^2 + 5x - 2$, evaluate $h(3)$, $h(-1)$, and $h(b-2)$. Explain why $h(b-2) \neq h(b) - h(2)$.
  4. Identify two common scenarios that restrict the domain of a function given by a formula. For each scenario, provide an example function and find its domain.
  5. What does it mean for a function to be "one-to-one"? How does the Horizontal Line Test help determine if a function possesses this property, and why is this property important in mathematics?
