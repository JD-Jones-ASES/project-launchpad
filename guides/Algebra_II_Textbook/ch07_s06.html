Study Guide: Logarithmic Functions

Overview

This section introduces logarithmic functions, $f(x) = \log_b(x)$, as distinct functions rather than just computational tools. It explores their fundamental properties, including their graphs, domains, ranges, and asymptotes, by highlighting their inverse relationship with exponential functions. The section also covers how standard transformations alter the characteristics of logarithmic graphs.

Key Concepts

  • Definition of a Logarithmic Function: A function of the form $f(x) = \log_b(x)$, where $b > 0$ and $b \neq 1$.
  • Inverse Relationship: A logarithmic function $f(x) = \log_b(x)$ is the inverse of the exponential function $g(x) = b^x$. This means their graphs are reflections of each other across the line $y = x$.
  • Properties of $y = \log_b(x)$:
    • Domain: $(0, \infty)$ – The argument of a logarithm must always be positive.
    • Range: $(-\infty, \infty)$ – All real numbers.
    • x-intercept: $(1, 0)$ – Since $b^0 = 1$, $\log_b(1) = 0$ for any valid base $b$.
    • Vertical Asymptote: $x = 0$ (the y-axis) – The graph approaches but never touches this line.
    • Monotonicity: The function is increasing if $b > 1$ and decreasing if $0 < b < 1$.
    • One-to-one: Each x-value corresponds to exactly one y-value, and vice-versa.
  • Graphing Logarithmic Functions: To graph $y = \log_b(x)$, it's often helpful to use its equivalent exponential form, $x = b^y$, to generate a table of points.
  • Transformations: Logarithmic functions undergo the same transformations as other functions:
    • Vertical Shift: $f(x) + k$ shifts the graph up $k$ units.
    • Horizontal Shift: $f(x - h)$ shifts the graph right $h$ units, moving the vertical asymptote to $x = h$ and changing the domain.
    • Reflection across x-axis: $-f(x)$ reflects the graph vertically.
    • Reflection across y-axis: $f(-x)$ reflects the graph horizontally, affecting the domain and vertical asymptote.
  • Domain Restriction: For any logarithmic function $f(x) = \log_b(g(x))$, its argument $g(x)$ must be strictly positive ($g(x) > 0$). This is the most crucial step when finding the domain.

Important Terms

  • Logarithmic Function: A function $f(x) = \log_b(x)$ that is the inverse of the exponential function $g(x) = b^x$.
  • Base (of a logarithm): The number $b$ in $\log_b(x)$. It must be positive and not equal to 1.
  • Argument (of a logarithm): The expression inside the logarithm, denoted by $x$ in $\log_b(x)$ or $g(x)$ in $\log_b(g(x))$. This expression must always be greater than zero.
  • Inverse Functions: Two functions $f$ and $g$ are inverse functions if $f(g(x)) = x$ and $g(f(x)) = x$. Their graphs are symmetric with respect to the line $y=x$.
  • Vertical Asymptote: A vertical line that the graph of a function approaches but never touches. For $f(x) = \log_b(x)$, the y-axis ($x=0$) is the vertical asymptote.
  • Domain: The set of all possible input values (x-values) for which the function is defined. For logarithmic functions, this is determined by ensuring the argument is positive.
  • Range: The set of all possible output values (y-values) of a function. The range of a basic logarithmic function is all real numbers.
  • x-intercept: The point where the graph of a function crosses the x-axis, meaning the y-value is 0. For basic logarithmic functions, this is $(1,0)$.
  • y-intercept: The point where the graph of a function crosses the y-axis, meaning the x-value is 0. Basic logarithmic functions do not have a y-intercept because $x=0$ is a vertical asymptote.

Detailed Notes

Introduction to Logarithmic Functions

While logarithms are commonly used for computations, this section emphasizes understanding $y = \log_b(x)$ as a distinct function. This involves exploring its graphical representation, its valid input values (domain), its output values (range), and its unique asymptotic behavior.

Definition and Inverse Relationship

A logarithmic function is formally defined as $f(x) = \log_b(x)$, where the base $b$ must be positive ($b > 0$) and not equal to one ($b \neq 1$). The most crucial concept to grasp is that the logarithmic function is the inverse of the exponential function $g(x) = b^x$. This inverse relationship has significant implications for understanding its graph and properties.

The Graph of $y = \log_b(x)$

Because $y = \log_b(x)$ and $y = b^x$ are inverse functions, their graphs are mirror images of each other across the line $y = x$.

Key Properties of $y = \log_b(x)$:

  • Domain: $(0, \infty)$. This means you can only take the logarithm of a positive number.
  • Range: $(-\infty, \infty)$. Logarithmic functions can output any real number.
  • x-intercept: $(1, 0)$. This is because any positive base raised to the power of 0 equals 1 ($b^0 = 1$), so $\log_b(1) = 0$.
  • Vertical Asymptote: $x = 0$ (the y-axis). As $x$ approaches 0 from the right side, the value of $\log_b(x)$ approaches negative infinity (if $b>1$) or positive infinity (if $0<b<1$).
  • Increasing/Decreasing Behavior:
    • If $b > 1$, the function is increasing (the graph rises from left to right).
    • If $0 < b < 1$, the function is decreasing (the graph falls from left to right).
  • One-to-One: Logarithmic functions are one-to-one, meaning each x-value has a unique y-value, and each y-value comes from a unique x-value. There are no horizontal asymptotes.

Example: Graphing $f(x) = \log_3(x)$ To graph $f(x) = \log_3(x)$, we can use its equivalent exponential form, $x = 3^y$. By choosing integer values for $y$, we can easily find corresponding $x$ values:

$x = 3^y$ $y$
$1/9 = 3^{-2}$ $-2$
$1/3 = 3^{-1}$ $-1$
$1 = 3^0$ $0$
$3 = 3^1$ $1$
$9 = 3^2$ $2$

Plotting these points (e.g., $(1/9, -2)$, $(1/3, -1)$, $(1,0)$, $(3,1)$, $(9,2)$) reveals a graph that rises slowly and gets closer and closer to the vertical asymptote $x=0$ without ever touching it.

Transformations of Logarithmic Functions

Just like other function types, logarithmic functions can be transformed:

  • Vertical Shift: $f(x) + k$ moves the graph vertically. $k > 0$ shifts up, $k < 0$ shifts down. The domain and vertical asymptote remain unchanged.
  • Horizontal Shift: $f(x - h)$ moves the graph horizontally. $h > 0$ shifts right, $h < 0$ shifts left. This transformation does affect the vertical asymptote and domain. The new vertical asymptote will be $x = h$, and the new domain will be $(h, \infty)$ (if shifted right) or $(-\infty, h)$ (if reflected across the y-axis and then shifted).
  • Reflection across the x-axis: $-f(x)$ flips the graph upside down. The domain and vertical asymptote are unchanged.
  • Reflection across the y-axis: $f(-x)$ flips the graph left-to-right. This does affect the vertical asymptote and domain. The new domain will be $(-\infty, 0)$ for a basic reflected log.

Example: Transformations of $\log_2(x)$

(a) $g(x) = \log_2(x + 4)$ This represents a shift of $f(x) = \log_2(x)$ 4 units to the left.

  • Domain: The argument $(x+4)$ must be positive: $x+4 > 0 \implies x > -4$. So, the domain is $(-4, \infty)$.
  • Vertical Asymptote: Since the graph shifted left 4 units, the asymptote also shifts: $x = -4$.
  • x-intercept: Set $g(x) = 0$: $\log_2(x+4) = 0$. By definition of logarithm, $x+4 = 2^0 \implies x+4 = 1 \implies x = -3$. The x-intercept is $(-3, 0)$.

(b) $h(x) = -\log_2(x) + 3$ This represents a reflection of $f(x) = \log_2(x)$ across the x-axis, followed by a shift up 3 units.

  • Domain: The argument is $x$, which must be positive: $x > 0$. So, the domain is $(0, \infty)$. (Reflections across the x-axis and vertical shifts do not change the domain or vertical asymptote for $f(x)=\log_b(x)$).
  • Vertical Asymptote: $x = 0$ (the y-axis).
  • x-intercept: Set $h(x) = 0$: $-\log_2(x) + 3 = 0 \implies \log_2(x) = 3$. By definition, $x = 2^3 \implies x = 8$. The x-intercept is $(8, 0)$.

Finding the Domain of a Logarithmic Function

The most critical rule for logarithmic functions is that the argument must always be strictly positive. For any function $f(x) = \log_b(g(x))$, you must solve the inequality $g(x) > 0$ to find its domain.

Example: Finding the domain of $f(x) = \log(4 - x^2)$ The argument is $4 - x^2$. We must have: $4 - x^2 > 0$ $4 > x^2$ This inequality holds when $-2 < x < 2$. So, the domain is $(-2, 2)$.

Identifying Key Features from the Equation

Combining all the concepts, you can determine various features of a logarithmic function from its equation.

Example: For $f(x) = \log_5(x - 2) - 1$, state the domain, range, vertical asymptote, x-intercept, and y-intercept (if any).

  • Domain: The argument $(x-2)$ must be positive: $x-2 > 0 \implies x > 2$. Domain: $(2, \infty)$.
  • Range: The range of all basic logarithmic functions, even with shifts, is $(-\infty, \infty)$.
  • Vertical Asymptote: The horizontal shift means the asymptote moves from $x=0$ to $x=2$. So, the vertical asymptote is $x = 2$.
  • x-intercept: Set $f(x) = 0$: $\log_5(x - 2) - 1 = 0$ $\log_5(x - 2) = 1$ Using the definition of a logarithm, $x - 2 = 5^1$ $x - 2 = 5 \implies x = 7$. The x-intercept is $(7, 0)$.
  • y-intercept: To find the y-intercept, set $x = 0$. However, $x=0$ is not in the domain $(2, \infty)$. Therefore, there is no y-intercept.

Checkpoint: For $g(x) = \ln(x + 1) + 2$, find the domain, vertical asymptote, and x-intercept. (Solution: Domain $(-1, \infty)$; asymptote $x = -1$; $x$-intercept: $x = e^{-2} - 1 \approx -0.865$.)

Review Questions

  1. Explain the inverse relationship between exponential functions and logarithmic functions. How does this relationship visually manifest when their graphs are drawn on the same coordinate plane?
  2. What is the single most important restriction when determining the domain of a logarithmic function? Illustrate this with an example by finding the domain of $f(x) = \log_2(5 - x)$.
  3. Describe how a horizontal shift affects the vertical asymptote and the domain of a logarithmic function. Provide an example comparing the vertical asymptote and domain of $f(x) = \log(x)$ and $g(x) = \log(x+3)$.
  4. Consider the function $f(x) = \log_b(x)$. How do the increasing/decreasing behavior and the end behavior of the graph change when $b > 1$ versus when $0 < b < 1$?
  5. Given the function $h(x) = -\log_4(x - 1) + 2$, identify all the transformations from the basic function $f(x) = \log_4(x)$, and then determine the domain, range, vertical asymptote, and x-intercept. Show your work for each step.
