Study Guide: Logarithmic Equations

Overview

This section introduces logarithmic equations, which are equations where the variable appears inside a logarithm. A critical aspect of solving these equations is understanding and adhering to the domain restriction of logarithmic functions: the argument of a logarithm must always be positive. Consequently, every potential solution must be rigorously checked in the original equation to identify and reject any extraneous solutions.

Key Concepts

  • Definition of a Logarithmic Equation: An equation containing one or more logarithms where the variable is part of the logarithm's argument.
  • Domain Restriction: For any logarithm $\log_b(x)$, the argument $x$ must be greater than zero ($x > 0$). This is the most crucial rule for solving and checking logarithmic equations.
  • Strategy for Solving: A systematic four-step process for solving logarithmic equations:
    1. Isolate/Combine Logs: Simplify the equation by isolating a single logarithm or combining multiple logarithms into one using logarithm properties (e.g., product rule, quotient rule).
    2. Convert to Exponential Form: Rewrite the logarithmic equation $\log_b M = c$ into its equivalent exponential form $M = b^c$.
    3. Solve the Resulting Equation: Solve the algebraic equation (linear, quadratic, etc.) obtained from the conversion.
    4. Check Solutions: Substitute all proposed solutions back into the original equation. Reject any solution that makes the argument of any logarithm in the original equation less than or equal to zero.
  • Logarithm Properties: Often used in step 1. The Product Rule ($\log_b M + \log_b N = \log_b(MN)$) is explicitly shown in an example, but the Quotient Rule and Power Rule can also be applicable.
  • One-to-One Property: If an equation has a single logarithm on both sides with the same base, e.g., $\log_b M = \log_b N$, then their arguments must be equal: $M = N$.
  • Extraneous Solutions: Solutions that arise from the algebraic steps but do not satisfy the domain restrictions of the original logarithmic equation. These must be rejected.
  • Quadratic in Disguise: Some logarithmic equations can be transformed into a quadratic form (e.g., $(\log x)^2 - 3 \log x + 2 = 0$) by using substitution (e.g., $u = \log x$).

Important Terms

  • Logarithmic Equation: An equation that includes a variable within the argument of a logarithm.
  • Argument of a Logarithm: The expression inside the logarithm (e.g., in $\log_b(x)$, $x$ is the argument). It must always be positive.
  • Exponential Form: The equivalent form of a logarithm. If $\log_b M = c$, then its exponential form is $M = b^c$.
  • Extraneous Solution: A solution obtained through algebraic manipulation that does not satisfy the domain requirements of the original logarithmic equation, and therefore is not a valid solution.
  • Product Rule for Logarithms: The rule stating that the sum of two logarithms with the same base can be written as the logarithm of the product of their arguments: $\log_b M + \log_b N = \log_b(MN)$.
  • Natural Logarithm (ln): A logarithm with base $e$ (Euler's number), written as $\ln x$, which is equivalent to $\log_e x$.
  • Common Logarithm (log): A logarithm with base 10, typically written as $\log x$, which is equivalent to $\log_{10} x$.

Detailed Notes

The Core Principle: Domain Restrictions

Before diving into solving, it's paramount to remember that the domain of any logarithmic function $\log_b(x)$ requires $x > 0$. This means the argument of every logarithm in your equation must be strictly positive. This restriction leads to the critical final step: checking all proposed solutions.

General Strategy for Solving Logarithmic Equations

  1. Isolate the Logarithm or Combine Logs:

    • If you have a single logarithm, get it by itself on one side of the equation.
    • If you have multiple logarithms on one or both sides, use logarithm properties (Product Rule, Quotient Rule, Power Rule) to combine them into a single logarithm on each side, if possible.
    • Example: If you have $\log_2 x + \log_2(x-2)$, combine it to $\log_2[x(x-2)]$ using the Product Rule.

  2. Convert to Exponential Form:

    • Once you have an equation in the form $\log_b M = c$, convert it to the equivalent exponential form $M = b^c$.
    • Example: $\log_3(2x-1) = 4$ becomes $2x-1 = 3^4$.
    • If you have $\log_b M = \log_b N$, you can use the one-to-one property directly to $M=N$.

  3. Solve the Resulting Equation:

    • The equation obtained after conversion (or using the one-to-one property) will be an algebraic equation (linear, quadratic, etc.) that you can solve using standard techniques.

  4. Check All Proposed Solutions:

    • This is non-negotiable for logarithmic equations. Substitute each potential solution back into the original equation.
    • If a value causes the argument of any logarithm in the original equation to be zero or negative, that value is an extraneous solution and must be rejected.
    • Only solutions that result in all logarithm arguments being positive are valid.

Examples and Specific Solution Types

Type 1: Basic Logarithmic Equation

Example: Solve $\log_3(2x - 1) = 4$.

  • The logarithm is already isolated.
  • Convert to exponential form: $2x - 1 = 3^4$.
  • Solve: $2x - 1 = 81 \implies 2x = 82 \implies x = 41$.
  • Check: In the original equation, the argument is $2x-1$. For $x=41$, $2(41)-1 = 81$. Since $81 > 0$, $x=41$ is a valid solution.

Type 2: Logarithmic Equation Requiring Combination of Logs

Example: Solve $\log_2 x + \log_2(x-2) = 3$.

  • Combine logs using the Product Rule: $\log_2[x(x-2)] = 3$.
  • Convert to exponential form: $x(x-2) = 2^3 \implies x^2 - 2x = 8$.
  • Solve the quadratic equation: $x^2 - 2x - 8 = 0 \implies (x-4)(x+2) = 0$. This gives $x=4$ or $x=-2$.
  • Check:
    • For $x=4$: Arguments are $x=4$ and $x-2=2$. Both $4 > 0$ and $2 > 0$. So, $x=4$ is a valid solution.
    • For $x=-2$: Argument $x=-2$. Since $-2 \not> 0$, $\log_2(-2)$ is undefined. Thus, $x=-2$ is an extraneous solution and must be rejected.
  • Solution: $x=4$.

Type 3: Natural Log Equation

Example: Solve $\ln(x+3) = 4$.

  • The natural logarithm is isolated. Remember $\ln$ is $\log_e$.
  • Convert to exponential form: $x+3 = e^4$.
  • Solve: $x = e^4 - 3$.
  • Check: The argument is $x+3$. Since $e^4 \approx 54.598$, $e^4 - 3 \approx 51.598$. So $x+3 = e^4 > 0$. This is a valid solution.

Type 4: Equation with Logs on Both Sides

Example: Solve $\log(x+5) = \log(3x - 1)$.

  • Both sides have a single logarithm with the same base (base 10).
  • Use the one-to-one property: $x+5 = 3x-1$.
  • Solve: $6 = 2x \implies x=3$.
  • Check:
    • For $x=3$: Arguments are $x+5 = 3+5=8$ and $3x-1 = 3(3)-1=8$. Both $8 > 0$. So, $x=3$ is a valid solution.

Type 5: Quadratic Disguise in a Logarithmic Equation

Example: Solve $(\ln x)^2 - 3 \ln x + 2 = 0$.

  • Let $u = \ln x$. The equation becomes a quadratic in $u$: $u^2 - 3u + 2 = 0$.
  • Solve the quadratic: $(u-1)(u-2) = 0 \implies u=1$ or $u=2$.
  • Back-substitute to find $x$:
    • If $u=1$: $\ln x = 1 \implies x = e^1 = e$.
    • If $u=2$: $\ln x = 2 \implies x = e^2$.
  • Check:
    • For $x=e$: The argument is $e$. Since $e \approx 2.718 > 0$, $x=e$ is valid.
    • For $x=e^2$: The argument is $e^2$. Since $e^2 \approx 7.389 > 0$, $x=e^2$ is valid.
  • Solution set: ${e, e^2}$.

Crucial Reminder: Always double-check your solutions by substituting them into the original equation to ensure that all logarithm arguments remain positive. This step prevents the acceptance of extraneous solutions.

Review Questions

  1. Explain why checking solutions for logarithmic equations is a mandatory step, contrasting it with solving linear or simple quadratic equations where this step might be optional or less critical.
  2. Outline the four-step strategy for solving logarithmic equations. For each step, briefly describe its purpose and how it helps lead to a solution.
  3. Solve the equation $\log_5(x+1) + \log_5(x-3) = 1$. Be sure to show your check for extraneous solutions.
  4. Describe how an equation like $(\log x)^2 + 2 \log x - 3 = 0$ can be solved. What mathematical technique is employed, and what must be done after applying that technique?
  5. When is it appropriate to use the one-to-one property of logarithms (i.e., if $\log_b M = \log_b N$, then $M=N$) to solve an equation? Provide an example of such an equation and explain why this property simplifies its solution.
