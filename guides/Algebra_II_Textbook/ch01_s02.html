Solving Equations in One Variable Study Guide

Overview

This section introduces the fundamental concepts of equations, solutions, and solution sets. It primarily focuses on linear equations in one variable, outlining a step-by-step strategy for solving them using properties of equality. The section also covers special cases like equations with fractions, word problems, and distinguishes between identities (always true), contradictions (never true), and conditional equations (true for specific values).

Key Concepts

  • Equations and Solutions: An equation is a statement that two expressions are equal. A solution (or root) is a value of the variable that makes the equation true, and the solution set is the collection of all such values.
  • Linear Equations in One Variable: These are equations that can be written in the standard form $ax + b = 0$, where $a$ and $b$ are real numbers and $a \ne 0$. The goal is to find the specific value of the single variable that satisfies the equation.
  • Equivalent Equations: Equations are considered equivalent if they have exactly the same solution set. The process of solving equations involves transforming them into simpler equivalent equations.
  • Properties of Equality: These are fundamental rules (Addition, Subtraction, Multiplication, Division Properties) that allow you to perform the same operation on both sides of an equation without altering its solution set, thereby creating an equivalent equation.
  • General Strategy for Solving Linear Equations: A systematic approach involving clearing fractions, removing parentheses, combining like terms, collecting variable terms on one side, and finally isolating the variable. Checking the solution is a critical final step.
  • Identities and Contradictions: Special types of equations where the variable cancels out. An identity is always true (solution set is all real numbers), while a contradiction is never true (empty solution set).

Important Terms

  • Equation: A mathematical statement asserting that two algebraic expressions are equal.
  • Solution (or Root): A value of the variable that makes an equation a true statement when substituted.
  • Solution Set: The set of all values that are solutions to a given equation.
  • Linear Equation in One Variable: Any equation that can be written in the form $ax + b = 0$, where $a$ and $b$ are real numbers and $a \ne 0$.
  • Equivalent Equations: Equations that have exactly the same solution set.
  • Addition Property of Equality: If $a = b$, then $a + c = b + c$.
  • Subtraction Property of Equality: If $a = b$, then $a - c = b - c$.
  • Multiplication Property of Equality: If $a = b$, then $ac = bc$.
  • Division Property of Equality: If $a = b$ and $c \ne 0$, then $a/c = b/c$.
  • Identity: An equation that is true for every real number; its solution set is all real numbers ($\mathbb{R}$).
  • Contradiction: An equation that is never true for any real number; its solution set is the empty set ($\emptyset$ or {}).
  • Conditional Equation: An equation that is true for some but not all real numbers; most equations encountered in algebra fall into this category.
  • Literal Equation: An equation involving two or more variables, where the task is to isolate one specific variable in terms of the others (e.g., solving $A = lw$ for $l$ yields $l = A/w$).

Detailed Notes

Understanding Equations and Solutions

At its core, an equation is simply a statement that two mathematical expressions have the same value. For example, $3x - 7 = 14$ is an equation because it declares that the expression $3x - 7$ is equal to the expression $14$.

A solution (or root) to an equation is a specific value for the variable that makes the equation a true statement. In the example $3x - 7 = 14$, if we substitute $x = 7$, we get $3(7) - 7 = 21 - 7 = 14$, which is a true statement ($14 = 14$). Therefore, $x=7$ is a solution. The collection of all such solutions is called the solution set.

Linear Equations in One Variable

This section primarily focuses on linear equations in one variable. These are equations that can always be rewritten in the standard form $ax + b = 0$, where $a$ and $b$ are real numbers, and crucially, $a \ne 0$. The variable only appears to the first power, and there's only one type of variable (e.g., $x$, but not $x$ and $y$). Solving a linear equation means finding the value(s) of the variable that make the equation true.

The Role of Properties of Equality

When we solve equations, we're constantly transforming them into simpler forms. The key is that these new forms must be equivalent equations, meaning they have exactly the same solution set as the original. We achieve this by using the Properties of Equality:

  • Addition Property: You can add any real number ($c$) to both sides of an equation. If $a=b$, then $a+c=b+c$.
  • Subtraction Property: You can subtract any real number ($c$) from both sides of an equation. If $a=b$, then $a-c=b-c$.
  • Multiplication Property: You can multiply both sides of an equation by any non-zero real number ($c$). If $a=b$, then $ac=bc$.
  • Division Property: You can divide both sides of an equation by any non-zero real number ($c$). If $a=b$ and $c \ne 0$, then $a/c=b/c$.

These properties ensure that if you perform an operation on one side of an equation, you must do the exact same thing to the other side to maintain the equality and preserve the solution set.

General Strategy for Solving a Linear Equation

A systematic approach makes solving linear equations straightforward:

  1. Clear Any Fractions: If your equation contains fractions, find the Least Common Denominator (LCD) of all denominators. Multiply every single term on both sides of the equation by this LCD. This will eliminate all denominators, leaving you with an equation that contains only integers.
  2. Use the Distributive Property to Remove Parentheses: If there are any parentheses, distribute any numbers or signs outside them to the terms inside.
  3. Combine Like Terms on Each Side: On each side of the equation separately, combine terms that have the same variable part (e.g., $5x - 2x = 3x$) and combine any constant terms (numbers without variables).
  4. Collect Variable Terms on One Side and Constant Terms on the Other: Use the Addition or Subtraction Properties of Equality to move all terms containing the variable to one side (typically the left) and all constant terms to the other side (typically the right).
  5. Divide Both Sides by the Coefficient of the Variable: The variable should now be isolated on one side, multiplied by a single coefficient. Divide both sides of the equation by this coefficient (using the Division Property) to find the value of the variable.
  6. Check the Solution in the Original Equation: This is a crucial step! Substitute the value you found for the variable back into the very first equation you started with. If both sides simplify to the same number, your solution is correct.

Examples Demonstrating the Strategy:

  • Solving a Multi-Step Equation: Solve $3x - 7 = 14$.

    • Add 7 to both sides: $3x - 7 + 7 = 14 + 7 \implies 3x = 21$
    • Divide both sides by 3: $3x/3 = 21/3 \implies x = 7$
    • Check: $3(7) - 7 = 21 - 7 = 14$. This is true, so $x=7$ is the solution.

  • Variables on Both Sides: Solve $5x + 3 = 2x - 9$.

    • Subtract $2x$ from both sides: $5x - 2x + 3 = 2x - 2x - 9 \implies 3x + 3 = -9$
    • Subtract 3 from both sides: $3x + 3 - 3 = -9 - 3 \implies 3x = -12$
    • Divide both sides by 3: $3x/3 = -12/3 \implies x = -4$
    • Check: $5(-4) + 3 = -20 + 3 = -17$. Also, $2(-4) - 9 = -8 - 9 = -17$. Since $-17 = -17$, the solution is correct.

  • Equations with Fractions: Solve $\frac{x}{3} + \frac{1}{2} = \frac{5}{6}$.

    • The LCD of 3, 2, and 6 is 6. Multiply every term by 6: $6 \left(\frac{x}{3}\right) + 6 \left(\frac{1}{2}\right) = 6 \left(\frac{5}{6}\right)$ $2x + 3 = 5$
    • Subtract 3 from both sides: $2x = 2$
    • Divide both sides by 2: $x = 1$
    • Check: $\frac{1}{3} + \frac{1}{2} = \frac{2}{6} + \frac{3}{6} = \frac{5}{6}$. This is true, so $x=1$ is the solution.

Solving Word Problems

Many real-world situations can be translated into linear equations. To solve them:

  1. Define a variable: Assign a letter (e.g., $n$) to represent the unknown quantity.
  2. Translate the words into an equation: Carefully read the problem and write a mathematical equation that represents the given relationships.
  3. Solve the equation: Apply the general strategy for solving linear equations.
  4. Check your answer in the context of the problem: Ensure the numerical answer makes sense in the real-world scenario.
  • Example: Word Problem: "A number increased by 17 is equal to three times the number minus 5. Find the number."
    • Let $n$ represent the unknown number.
    • Translate: $n + 17 = 3n - 5$
    • Solve:
      • Subtract $n$ from both sides: $17 = 2n - 5$
      • Add 5 to both sides: $22 = 2n$
      • Divide by 2: $n = 11$
    • Check: $11 + 17 = 28$. And $3(11) - 5 = 33 - 5 = 28$. The answer makes sense.

Identities and Contradictions (Special Cases)

While most linear equations have a single, unique solution (these are called conditional equations), there are two special outcomes:

  • Identity: An equation that is true for every possible real number. When you try to solve an identity, the variable terms will completely cancel out, leaving you with a true numerical statement (e.g., $5 = 5$, or $0 = 0$). The solution set for an identity is all real numbers ($\mathbb{R}$).
  • Contradiction: An equation that is never true for any real number. When you try to solve a contradiction, the variable terms will also completely cancel out, but you'll be left with a false numerical statement (e.g., $2 = 4$, or $0 = 7$). A contradiction has no solution, and its solution set is the empty set ($\emptyset$ or {}).

Note: The key indicator for an identity or contradiction is when the variable terms ($x$, $2x$, etc.) completely disappear from the equation during the solving process. If the resulting statement is true, it's an identity; if it's false, it's a contradiction.

A literal equation is simply an equation with multiple variables where you are asked to solve for one specific variable in terms of the others. For example, solving the area formula $A = lw$ for $l$ means isolating $l$ to get $l = A/w$.

Examples of Identities and Contradictions:

  • Identity Example: Solve $2(x + 3) - 4 = 2x + 2$.

    • $2x + 6 - 4 = 2x + 2$
    • $2x + 2 = 2x + 2$
    • Subtract $2x$ from both sides: $2 = 2$.
    • Since $2=2$ is a true statement, this is an identity. Solution set = $\mathbb{R}$.

  • Contradiction Example: Solve $3(x - 1) + 5 = 3x + 4$.

    • $3x - 3 + 5 = 3x + 4$
    • $3x + 2 = 3x + 4$
    • Subtract $3x$ from both sides: $2 = 4$.
    • Since $2=4$ is a false statement, this is a contradiction. Solution set = $\emptyset$.

Review Questions

  1. Explain what it means for two equations to be "equivalent," and how the properties of equality are used to generate equivalent equations. Provide an example illustrating the use of one of these properties.
  2. Outline the general strategy for solving a linear equation in one variable, listing at least five distinct steps. Why is checking your solution an important final step?
  3. Describe the key differences between a conditional equation, an identity, and a contradiction. What does the solution set typically look like for each type?
  4. When solving an equation that contains fractions, what is the first recommended step, and why is it effective? Illustrate with a simple equation like $\frac{x}{4} + \frac{1}{2} = \frac{3}{4}$.
  5. Formulate a short word problem that can be solved using a linear equation. Then, demonstrate how you would define a variable, translate the problem into an equation, and solve it.
