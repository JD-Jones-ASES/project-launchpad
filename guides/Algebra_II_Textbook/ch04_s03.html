Study Guide: Square Root Functions

Overview

This section introduces square root functions, a type of function whose domain is inherently restricted to non-negative real numbers. We learn how to determine the domain and range of these functions, graph them, and understand how various transformations (shifts, stretches, reflections) affect their basic form. The key takeaway is recognizing the "starting point" of a square root function and how its equation relates to its graph.

Key Concepts

  • Domain Restriction: The most fundamental concept for square root functions is that the expression under the square root (the radicand) cannot be negative in the real number system. This restricts the function's domain.
  • Starting Point (Endpoint): Unlike lines or parabolas, square root functions have a definite starting point on their graph, which is the leftmost point of their domain.
  • Standard Form of Transformed Square Root Functions: The general form for transformed square root functions is $f(x) = a\sqrt{x-h} + k$. Understanding the roles of $a$, $h$, and $k$ is crucial for graphing and analyzing these functions.
  • Transformations: Square root functions undergo the same types of transformations as other functions:
    • Horizontal Shift (h): Moves the graph left or right.
    • Vertical Shift (k): Moves the graph up or down.
    • Reflection (a < 0): Flips the graph over the x-axis, causing it to curve downwards.
    • Stretch/Compression (a): Stretches or compresses the graph vertically.
  • Domain Calculation: To find the domain, set the radicand greater than or equal to zero and solve the resulting inequality.

Important Terms

  • Square Root Function: The basic function $f(x) = \sqrt{x}$. Its domain is $[0, \infty)$ and its range is $[0, \infty)$. Its graph starts at the origin $(0,0)$ and curves gradually upward to the right.
  • Radical Function: Any function containing a variable under a radical sign. The general transformed square root function in this section is of the form $f(x) = a\sqrt{x-h} + k$, where $a \ne 0$.
  • Radicand: The expression under the radical sign. For a real result, the radicand of a square root must be non-negative.
  • Starting Point (Endpoint): For a function in the form $f(x) = a\sqrt{x-h} + k$, the starting point of the graph is $(h, k)$.

Detailed Notes

Introduction to Square Root Functions

Square root functions are unique because they are the first type of function we encounter whose domain is naturally restricted. Since we cannot take the square root of a negative real number and get another real number, the input to a square root function must always be non-negative (zero or positive). Graphically, this means square root functions always have a definite starting point, unlike functions that extend infinitely in both directions (like lines and parabolas).

Domain of Transformed Square Root Functions

The most critical step in working with square root functions is determining their domain. Property: Domain of $f(x) = \sqrt{ax+b}$ To find the domain of $f(x) = \sqrt{ax+b}$:

  1. Set the radicand greater than or equal to zero: $ax + b \ge 0$.
  2. Solve the inequality for x.
  3. Write the answer in interval notation.

Examples of Finding the Domain:

  • $f(x) = \sqrt{x - 5}$

    • Radicand: $x - 5$
    • Inequality: $x - 5 \ge 0$
    • Solve: $x \ge 5$
    • Domain: $[5, \infty)$

  • $g(x) = \sqrt{3x + 12}$

    • Radicand: $3x + 12$
    • Inequality: $3x + 12 \ge 0$
    • Solve: $3x \ge -12 \implies x \ge -4$
    • Domain: $[-4, \infty)$

  • $h(x) = \sqrt{8 - 2x}$

    • Radicand: $8 - 2x$
    • Inequality: $8 - 2x \ge 0$
    • Solve: $-2x \ge -8$. When dividing by a negative number, flip the inequality sign: $x \le 4$.
    • Domain: $(-\infty, 4]$

  • $p(x) = \sqrt{x^2 - 9}$

    • Radicand: $x^2 - 9$
    • Inequality: $x^2 - 9 \ge 0$
    • Factor: $(x-3)(x+3) \ge 0$
    • This quadratic inequality holds when $x \le -3$ or $x \ge 3$.
    • Domain: $(-\infty, -3] \cup [3, \infty)$

Transformations of $f(x) = \sqrt{x}$

The general form for transformed square root functions is $g(x) = a\sqrt{x-h} + k$. The parameters $a$, $h$, and $k$ dictate how the base function $y = \sqrt{x}$ is transformed. The starting point (left endpoint) of the graph is $(h, k)$.

  • Horizontal Shift (h):
    • If $h > 0$ (e.g., $\sqrt{x-3}$), the graph shifts $h$ units to the right. The domain starts at $x=h$.
    • If $h < 0$ (e.g., $\sqrt{x+3}$ which is $\sqrt{x-(-3)}$), the graph shifts $|h|$ units to the left. The domain starts at $x=h$.
  • Vertical Shift (k):
    • If $k > 0$ (e.g., $\sqrt{x}+2$), the graph shifts $k$ units up. The range starts at $y=k$.
    • If $k < 0$ (e.g., $\sqrt{x}-2$), the graph shifts $|k|$ units down. The range starts at $y=k$.
  • Reflection (a):
    • If $a < 0$ (e.g., $-\sqrt{x}$), the graph is reflected over the x-axis. The graph will curve downwards from its starting point.
  • Stretch/Compression (a):
    • If $|a| > 1$, the graph is stretched vertically.
    • If $0 < |a| < 1$, the graph is compressed vertically.

Examples of Graphing Transformed Square Root Functions: For each function, state the starting point, domain, range, and describe the transformation from $y = \sqrt{x}$.

  • $f(x) = \sqrt{x - 3} + 2$

    • Here, $a=1$, $h=3$, $k=2$.
    • Starting point: $(3, 2)$
    • Domain: $[3, \infty)$
    • Range: $[2, \infty)$
    • Transformation: Shift $y = \sqrt{x}$ right $3$ units and up $2$ units.

  • $g(x) = -\sqrt{x + 1}$

    • Here, $a=-1$, $h=-1$, $k=0$.
    • Starting point: $(-1, 0)$
    • Domain: $[-1, \infty)$
    • Range: $(-\infty, 0]$ (due to reflection over the x-axis)
    • Transformation: Shift $y = \sqrt{x}$ left $1$ unit, then reflect over the x-axis. The graph curves downward.

  • $h(x) = 3\sqrt{x} - 6$

    • Here, $a=3$, $h=0$, $k=-6$.
    • Starting point: $(0, -6)$
    • Domain: $[0, \infty)$
    • Range: $[-6, \infty)$
    • Transformation: Stretch $y = \sqrt{x}$ vertically by a factor of $3$, then shift down $6$ units.

Writing an Equation from a Description

If given the starting point and another point the graph passes through, you can find the equation:

Example: Write an equation for the square root function whose graph starts at $(-2, 3)$, goes to the right, and passes through $(2, 7)$.

  1. Use the general form $y = a\sqrt{x-h} + k$. Since the starting point is $(-2, 3)$, we have $h = -2$ and $k = 3$. So, $y = a\sqrt{x - (-2)} + 3 = a\sqrt{x + 2} + 3$.

  2. Use the given point $(2, 7)$ to find $a$. Substitute $x=2$ and $y=7$ into the equation: $7 = a\sqrt{2 + 2} + 3$ $7 = a\sqrt{4} + 3$ $7 = 2a + 3$ Subtract 3 from both sides: $4 = 2a$ Divide by 2: $a = 2$.

  3. Write the final equation. Substitute $a=2$ back into the equation: $y = 2\sqrt{x + 2} + 3$.

Graphical Representation

The starting point $(h, k)$ is the "anchor" of the square root function graph. From this point, the graph sweeps upward or downward depending on the sign of $a$. For $g(x) = \sqrt{x-2}+1$, it starts at $(2, 1)$ and sweeps upward. For $h(x) = -\sqrt{x+3}$, it starts at $(-3, 0)$ and sweeps downward.

Caution

When solving inequalities of the form $ax + b \ge 0$ to find the domain, always remember to flip the inequality sign if you divide or multiply both sides by a negative number. For example, $-2x \ge -8$ becomes $x \le 4$.
