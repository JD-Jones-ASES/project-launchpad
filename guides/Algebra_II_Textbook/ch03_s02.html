Solving Equations by Taking Roots

Overview

This section introduces how to solve the simplest quadratic equations, those without a linear term, by taking the square root of both sides. It details the "Square-Root Property," explaining how the sign of the constant term determines whether solutions are real, zero, or complex. The method is then extended to equations where a binomial is squared, and finally to higher-root equations, emphasizing the importance of considering the context for real-world applications.

Key Concepts

  • Square-Root Property for (x^2 = c):
    • If (c > 0), there are two real solutions: (x = \pm \sqrt{c}).
    • If (c = 0), there is one real solution: (x = 0).
    • If (c < 0), there are two complex solutions: (x = \pm i\sqrt{|c|}).
    • This property applies generally to (u^2 = c), where (u) can be any expression.
  • The "Plus or Minus" ((\pm)) Symbol: When solving equations like (x^2=c), it's crucial to remember both the positive and negative roots. The radical symbol ((\sqrt{})) alone denotes the principal (non-negative) root, so (\pm) must be explicitly written for equations.
  • Isolating the Squared Term: Before applying the square-root property, ensure that the term being squared is isolated on one side of the equation.
  • Solving Equations of the Form ((ax+b)^2 = c): First apply the square-root property to the entire binomial, leading to two linear equations ((ax+b = \sqrt{c}) and (ax+b = -\sqrt{c})), which are then solved for (x).
  • Complex Conjugate Solutions: When solving an equation like ((ax+b)^2 = c) where (c < 0) and the original equation has real coefficients, the two complex solutions will always be complex conjugates of each other.
  • Higher-Root Equations ((x^n = c)):
    • If (n) is odd, there is one unique real solution: (x = \sqrt[n]{c}) (even if (c) is negative).
    • If (n) is even and (c > 0), there are two real solutions: (x = \pm \sqrt[n]{c}).
    • If (n) is even and (c < 0), there are no real solutions (only complex solutions).
  • Geometry and Physical Quantities: When solving for lengths, distances, or other physical quantities, only the positive root is meaningful, as these quantities cannot be negative.

Important Terms

  • Quadratic Equation: An equation that can be written in the form (ax^2 + bx + c = 0). This section focuses on the simpler form (x^2=c).
  • Square-Root Property: The rule that states if (u^2 = c), then (u = \pm \sqrt{c}).
  • Principal Square Root: The non-negative square root of a number, denoted by the radical symbol ((\sqrt{})). For example, (\sqrt{9} = 3), not (\pm 3).
  • Complex Solutions: Solutions to equations that involve the imaginary unit (i) ((i = \sqrt{-1})), typically arising when taking the square root of a negative number.
  • Complex Conjugates: A pair of complex numbers of the form (a+bi) and (a-bi). When an equation with real coefficients has complex solutions, they always appear as conjugate pairs.
  • Higher-Root Equation: An equation of the form (x^n = c), where (n) is an integer greater than 2.
  • Cube Root: The number that, when multiplied by itself three times, equals a given number. Denoted by (\sqrt[3]{}).

Detailed Notes

Introduction to Solving by Taking Roots

The simplest type of quadratic equation is (x^2 = c), where there is no linear term (no (x) term). To solve such equations, we use the inverse operation of squaring, which is taking the square root. However, it's crucial to account for the sign of (c) and remember that squaring both a positive and a negative number yields a positive result.

The Square-Root Property

This property is the foundation for solving equations of the form (x^2 = c) or (u^2 = c).

  • Case 1: (c > 0) (Positive (c))
    • If (x^2 = c) and (c) is a positive real number, there are two distinct real solutions: (x = \sqrt{c}) and (x = -\sqrt{c}). This is often written concisely as (x = \pm \sqrt{c}).
    • Example: For (x^2 = 49), (x = \pm \sqrt{49} = \pm 7).
  • Case 2: (c = 0)
    • If (x^2 = 0), there is exactly one real solution: (x = 0).
  • Case 3: (c < 0) (Negative (c))
    • If (x^2 = c) and (c) is a negative real number, there are no real solutions. Instead, there are two distinct complex (non-real) solutions: (x = \pm i\sqrt{|c|}). Remember that (i = \sqrt{-1}).
    • Example: For (x^2 = -9), (x = \pm \sqrt{-9} = \pm \sqrt{9 \cdot -1} = \pm 3i).

Important Note on (\pm): The symbol (\sqrt{}) always refers to the principal (non-negative) square root. Therefore, when solving an equation, you must explicitly include the (\pm) to account for both positive and negative solutions.

Solving Basic Equations

To solve equations like (x^2=c) or (Ax^2=C):

  1. Isolate (x^2): Manipulate the equation algebraically to get (x^2) by itself on one side.
  2. Apply the Square-Root Property: Take the square root of both sides, remembering the (\pm) sign.

  • Example 1: Solving (x^2 = 49)

    • (x^2 = 49)
    • (x = \pm \sqrt{49})
    • (x = \pm 7)
    • Solutions: (7, -7)

  • Example 2: Solving (3x^2 = 48)

    • First, isolate (x^2): (3x^2 = 48) (x^2 = \frac{48}{3}) (x^2 = 16)
    • Now, apply the square-root property: (x = \pm \sqrt{16}) (x = \pm 4)
    • Solutions: (4, -4)

Solving Equations of the Form ((ax+b)^2=c)

This is a common variation where the expression being squared is a binomial. The steps are:

  1. Isolate the squared expression: Ensure the entire term ((ax+b)^2) is by itself on one side.
  2. Apply the square-root property: Take the square root of both sides, remembering (\pm). This will result in (ax + b = \pm \sqrt{c}).
  3. Solve the two linear equations: Separate the equation into two cases:
    • (ax + b = \sqrt{c})
    • (ax + b = -\sqrt{c}) Solve each of these for (x).
  4. Check both solutions: Substitute your solutions back into the original equation to verify.

  • Example 1: Solving ((x-2)^2 = 25)

    • The squared expression ((x-2)^2) is already isolated.
    • Apply the square-root property: (x - 2 = \pm \sqrt{25}) (x - 2 = \pm 5)
    • Solve the two linear equations:
      • Case 1: (x - 2 = 5 \implies x = 7)
      • Case 2: (x - 2 = -5 \implies x = -3)
    • Solution set: ({-3, 7})

  • Example 2: Complex Solutions from ((2x+1)^2 = -9)

    • The squared expression is isolated.
    • Apply the square-root property (note (c < 0)): (2x + 1 = \pm \sqrt{-9}) (2x + 1 = \pm 3i)
    • Solve the two linear equations:
      • Case 1: (2x + 1 = 3i \implies 2x = -1 + 3i \implies x = \frac{-1 + 3i}{2})
      • Case 2: (2x + 1 = -3i \implies 2x = -1 - 3i \implies x = \frac{-1 - 3i}{2})
    • Solution set: ({\frac{-1+3i}{2}, \frac{-1-3i}{2}}). Notice these are complex conjugates.

Higher-Root Equations ((x^n = c))

The concept of taking roots extends to higher powers:

  • If (n) is odd: There is always exactly one real solution, (x = \sqrt[n]{c}), regardless of the sign of (c).
    • Example: (x^3 = -64 \implies x = \sqrt[3]{-64} = -4).
    • Example: (x^3 = 8 \implies x = \sqrt[3]{8} = 2).
  • If (n) is even:
    • If (c > 0), there are two real solutions: (x = \pm \sqrt[n]{c}).
    • If (c < 0), there are no real solutions (complex solutions exist, but are not always covered in this context).
    • If (c = 0), there is one real solution: (x = 0).
    • Example: (x^4 = 16 \implies x = \pm \sqrt[4]{16} = \pm 2).
    • Example: (x^4 = -16) has no real solutions.

Geometry Applications

In geometry and other real-world applications, quantities like length, distance, area, and volume are always non-negative.

  • If you find the area (A) of a square, its side length (s) is found by (s = \sqrt{A}). You only take the positive root because length cannot be negative.
  • Always consider the context of the problem. While the mathematical solution to an equation like (x^2=25) is (x=\pm 5), if (x) represents a side length, only (x=5) is a meaningful answer.

Review Questions

  2. What is the significance of explicitly writing "(\pm)" when applying the square-root property to solve an equation? Why is it important not to confuse this with the principal square root symbol (\sqrt{})?
  3. Outline the step-by-step process for solving an equation of the form ((ax+b)^2=c). Describe how the solution process changes if (c) is negative.
  4. Compare and contrast the number and type of real solutions you would expect for higher-root equations when the exponent (n) is odd versus when (n) is even. Use (x^3 = 27) and (x^4 = 81) as examples.
