Absolute Value Equations and Inequalities Study Guide

Overview

This section explores absolute value from both a geometric and algebraic perspective, defining it as distance from the origin on the number line. It then provides systematic methods for solving absolute value equations and inequalities, distinguishing between cases with one absolute value and those with two, and between "AND" (bounded interval) and "OR" (two rays) scenarios for inequalities. Special attention is paid to potential pitfalls like extraneous solutions and incorrect inequality notation.

Key Concepts

  • Absolute Value as Distance: The absolute value of a number $x$, denoted $|x|$, represents its distance from zero on the number line. Since distance is always non-negative, $|x| \ge 0$.
  • Algebraic Definition of Absolute Value:
    • $|x| = x$ if $x \ge 0$
    • $|x| = -x$ if $x < 0$
  • Fundamental Absolute Value Properties:
    • $|-x| = |x|$ (Symmetry)
    • $|xy| = |x||y|$ and $|x/y| = |x|/|y|$ (Multiplication/Division)
  • Solving Absolute Value Equations ($|E| = n$):
    • If $n > 0$, then $E = n$ or $E = -n$ (two solutions).
    • If $n = 0$, then $E = 0$ (one solution).
    • If $n < 0$, then there is no solution (absolute value cannot be negative).
  • Extraneous Solutions: Solutions that arise algebraically but do not satisfy the original equation, often occurring when a variable expression is on the right side of an absolute value equation (e.g., $|2x-1| = x+5$). Always check all candidate solutions.
  • Solving Equations with Two Absolute Values ($|A| = |B|$): This simplifies to $A = B$ or $A = -B$.
  • Solving Absolute Value Inequalities (Geometric Interpretation):
    • Less Than ($|E| < m$ or $|E| \le m$, for $m > 0$): Asks for numbers within distance $m$ of zero. Leads to an "AND" case, represented as a single bounded interval: $-m < E < m$ or $-m \le E \le m$.
    • Greater Than ($|E| > m$ or $|E| \ge m$, for $m > 0$): Asks for numbers farther than distance $m$ from zero. Leads to an "OR" case, represented as two outward-extending rays: $E < -m$ or $E > m$ (or with $\le, \ge$).
  • Special Cases for Absolute Value Inequalities:
    • If $m \le 0$, then $|E| < m$ or $|E| \le m$ has no solution.
    • If $m < 0$, then $|E| > m$ or $|E| \ge m$ is satisfied by all real numbers ($-\infty, \infty$).
  • Correct Notation for Inequality Solutions: "AND" cases use a single interval, e.g., $(-7/3, 3)$. "OR" cases use union notation, e.g., $(-\infty, -4] \cup [-1, \infty)$. Do not write "OR" solutions as a single compound inequality.

Important Terms

  • Absolute Value: The distance of a real number $x$ from $0$ on the number line, denoted $|x|$. It is formally defined as $x$ if $x \ge 0$ and $-x$ if $x < 0$.
  • Extraneous Solution: A value obtained during the algebraic solving process that does not satisfy the original equation. These must be identified and discarded, especially when the right side of an absolute value equation contains a variable.
  • Absolute Value Equation: An equation of the form $|E| = n$, where $E$ is an algebraic expression and $n$ is a real number. These are solved by considering $E = n$ and $E = -n$ (when $n \ge 0$).
  • Absolute Value Inequality: An inequality involving an absolute value, such as $|E| < m$, $|E| > m$, or their non-strict variants ($|E| \le m$, $|E| \ge m$). Solutions typically result in either a single bounded interval or the union of two rays.

Detailed Notes

Absolute Value as Distance

The most intuitive way to understand absolute value is geometrically: it's the distance a number is from zero on the real number line. For example, both $5$ and $-5$ are $5$ units away from $0$, so $|5|=5$ and $|-5|=5$. This non-negative nature of distance means $|x| \ge 0$ for any real number $x$.

The formal algebraic definition reflects this: [ |x| = \begin{cases} x & \text{if } x \ge 0 \ -x & \text{if } x < 0 \end{cases} ] Two important algebraic properties derive directly from this definition:

  1. $|-x| = |x|$: The absolute value of a number is the same as the absolute value of its opposite. This highlights the symmetry around zero.
  2. $|xy| = |x||y|$: The absolute value of a product is the product of the absolute values. This extends to quotients: $|x/y| = |x|/|y|$ (provided $y \neq 0$).

Absolute Value Equations

When solving an absolute value equation of the form $|E| = n$, where $E$ is an algebraic expression and $n$ is a real number, the approach depends on the value of $n$:

  • Case 1: $n > 0$ (Positive value) The expression $E$ must be equal to $n$ or its opposite, $-n$. So, $|E| = n \implies E = n \text{ or } E = -n$. This usually leads to two distinct solutions. Example: Solve $|2x - 3| = 7$ $2x - 3 = 7 \quad \text{or} \quad 2x - 3 = -7$ $2x = 10 \quad \text{or} \quad 2x = -4$ $x = 5 \quad \text{or} \quad x = -2$ Always check solutions in the original equation: $|2(5) - 3| = |7| = 7$ (Correct) $|2(-2) - 3| = |-4 - 3| = |-7| = 7$ (Correct) Solution set: ${-2, 5}$.

  • Case 2: $n = 0$ (Zero) The expression $E$ must be exactly $0$. So, $|E| = 0 \implies E = 0$. This leads to exactly one solution.

  • Case 3: $n < 0$ (Negative value) An absolute value represents a distance, which is never negative. Therefore, if $n$ is negative, there is no solution. Example: Solve $|x + 4| = -2$ Since an absolute value cannot equal a negative number, there is no solution. Solution set: $\emptyset$.

Extraneous Solutions: It's crucial to check all candidate solutions, especially when the right side of the equation involves a variable (e.g., $|2x - 1| = x + 5$). If a candidate solution makes the expression on the right side negative, it would be an extraneous solution because an absolute value cannot equal a negative quantity.

Equations of the Form $|A| = |B|$

When both sides of an equation have an absolute value, such as $|A| = |B|$, where $A$ and $B$ are algebraic expressions, it simplifies to two cases:

  1. $A = B$ (The expressions are equal)
  2. $A = -B$ (The expressions are opposites) Example: Solve $|x - 2| = |3x + 4|$ Case 1: $x - 2 = 3x + 4$ $-6 = 2x$ $x = -3$

Case 2: $x - 2 = -(3x + 4)$ $x - 2 = -3x - 4$ $4x = -2$ $x = -1/2$

Checking both solutions: For $x = -3$: $|-3 - 2| = |-5| = 5$. $|3(-3) + 4| = |-9 + 4| = |-5| = 5$. (Correct) For $x = -1/2$: $|-1/2 - 2| = |-5/2| = 5/2$. $|3(-1/2) + 4| = |-3/2 + 8/2| = |5/2| = 5/2$. (Correct) Solution set: ${-3, -1/2}$.

Absolute Value Inequalities

The geometric interpretation is key to understanding absolute value inequalities.

  • $|E| < m$ (or $|E| \le m$) - The "AND" Case (Sandwich) This asks for all numbers whose distance from zero is less than $m$. If $m > 0$, the solutions lie in a bounded interval between $-m$ and $m$. $|E| < m \implies -m < E < m$ $|E| \le m \implies -m \le E \le m$ Example: Solve $|3x - 1| < 8$ $-8 < 3x - 1 < 8$ Add $1$ to all parts: $-7 < 3x < 9$ Divide all parts by $3$: $-7/3 < x < 3$ Solution in interval notation: $(-7/3, 3)$.

  • $|E| > m$ (or $|E| \ge m$) - The "OR" Case (Alligator Mouth) This asks for all numbers whose distance from zero is greater than $m$. If $m > 0$, the solutions lie in two outward-extending rays. $|E| > m \implies E < -m \text{ or } E > m$ $|E| \ge m \implies E \le -m \text{ or } E \ge m$ Example: Solve $|2x + 5| \ge 3$ $2x + 5 \le -3 \quad \text{or} \quad 2x + 5 \ge 3$ Left branch: $2x \le -8 \implies x \le -4$ Right branch: $2x \ge -2 \implies x \ge -1$ Solution in interval notation: $(-\infty, -4] \cup [-1, \infty)$. Caution: Never write an "OR" solution as a single compound inequality like $-4 \ge x \ge -1$. This implies $x \ge -4$ and $x \ge -1$, which is incorrect.

Special Cases for Inequalities (when $m \le 0$):

  • $|E| < m$ or $|E| \le m$ when $m \le 0$: Since $|E|$ is always non-negative, it can never be less than or equal to a negative number or zero (unless $E=0$ and $m=0$). If $m < 0$: No solution. (e.g., $|x| < -5$) If $m = 0$: $|E| < 0$ has no solution. $|E| \le 0$ implies $E=0$.
  • $|E| > m$ or $|E| \ge m$ when $m \le 0$: Since $|E|$ is always non-negative, it will always be greater than or equal to any negative number. If $m < 0$: All real numbers are solutions. (e.g., $|x| > -5$) If $m = 0$: $|E| > 0$ means $E \ne 0$. $|E| \ge 0$ means all real numbers are solutions.

Real-World Application

The distance interpretation of absolute value is crucial in fields like engineering and science. For example, a "tolerance" or "error bound" specification like "$x$ must be within $0.05$ cm of target value $c$" can be directly translated into the absolute value inequality $|x - c| \le 0.05$.

Review Questions

  1. Explain the geometric meaning of absolute value. How does this interpretation help in understanding why $|x| = -2$ has no solution, while $|x| = 2$ has two solutions?
  2. Describe the steps for solving an absolute value equation of the form $|E| = n$. What special considerations are there if $n$ is negative or zero?
  3. What is an "extraneous solution" in the context of absolute value equations? Provide an example of an equation where an extraneous solution might arise and how you would identify it.
  4. Geometrically, how do the solutions to $|E| < m$ and $|E| > m$ (for $m > 0$) differ? Explain why one leads to an "AND" statement and a single interval, while the other leads to an "OR" statement and two rays.
  5. Solve the following problems and explain your reasoning for each step: a) $|3x + 6| = 15$ b) $|x - 5| = |2x + 1|$ c) $|4x - 2| \le 10$ d) $|x + 3| > 7$
