Mandatos Afirmativos con Tú: Study Guide

Overview

This section introduces the formation and use of affirmative tú commands in Spanish, which are used to tell someone informally to do something. It highlights how these commands are surprisingly simple to form for regular verbs by using a familiar present-tense conjugation, emphasizing their prevalence in everyday communication and instructions.

Key Concepts

  • Affirmative Tú Commands: These are direct commands given to a single person you address with "tú" (informal "you"), telling them to do something (e.g., "Eat!", "Speak!").
  • Formation Rule for Regular Verbs: For all regular -ar, -er, and -ir verbs, the affirmative tú command is formed by taking the él/ella/Ud. form of the verb in the present tense.
    • Example: For hablar (to speak), the él/ella/Ud. present tense is habla. The command is ¡Habla!
    • Example: For comer (to eat), the él/ella/Ud. present tense is come. The command is ¡Come!
    • Example: For vivir (to live), the él/ella/Ud. present tense is vive. The command is ¡Vive!
  • No Subject Pronoun 'tú': It is crucial not to use the subject pronoun "tú" with the command. The command form itself implies you are speaking to "you." Adding "tú" sounds unnatural and can even be considered rude in Spanish.
  • Punctuation: In written Spanish, affirmative commands (like all exclamations) must begin with an inverted exclamation mark (¡) and end with a regular exclamation mark (!).
  • Applicability: This formation rule works consistently for all regular -ar, -er, and -ir verbs.
  • Maintaining Accent Marks: If the él/ella present-tense form of a verb already has an accent mark, it will remain in the command form. For regular verbs, this usually means no new accent marks are added.
  • Attaching Nouns/Pronouns/Location Words: Nouns, object pronouns, or place words can be attached after an affirmative tú command to provide more detail (e.g., "Wash the dishes").

Important Terms

  • Mandato(s) afirmativo(s): Affirmative command(s). Instructions telling someone to do something.
  • Tú command: An informal command directed at a single person, equivalent to "you" (singular informal).
  • Infinitive: The base form of a verb, unconjugated (e.g., hablar, comer, vivir).
  • Present tense (él/ella/Ud. form): The verb conjugation used for "he," "she," or formal "you" in the present tense. This is the key to forming regular tú commands.
  • Los quehaceres del hogar: Household chores. A common context for giving and receiving commands.
  • ¡Ojo!: An expression meaning "Watch out!" or "Be careful!" Used to highlight important warnings.
  • Porque: A conjunction meaning "because." Used to explain the reason for a command.

Detailed Notes

Introduction to Commands

Commands are an essential part of daily communication, used to give instructions, make requests, or tell someone what to do. In English, we simply use the base form of the verb (e.g., "Open the door," "Eat your vegetables," "Write your name"). In Spanish, these are called mandatos afirmativos, and forming the informal "tú" commands is quite straightforward for regular verbs.

Discovering the Pattern

The key to understanding affirmative tú commands lies in recognizing a pattern with a verb form you already know: the present tense.

  • Consider the verb hablar (to speak). The él/ella/Ud. present-tense form is habla. Notice that the affirmative tú command is ¡Habla!
  • For the verb comer (to eat), the él/ella/Ud. present-tense form is come. The command is ¡Come!
  • Following this pattern, for estudiar (to study), the él/ella/Ud. present tense is estudia, so the command is ¡Estudia!
  • For vivir (to live), the él/ella/Ud. present tense is vive, making the command ¡Vive!

This pattern reveals a simple and consistent rule.

The Golden Rule for Regular Affirmative Tú Commands

The fundamental rule for forming regular affirmative tú commands is:

Tú command = él/ella/Ud. present-tense form of the verb

Simply take the conjugation you already know for "he/she/formal you" in the present tense, and use it as the command. The subject "tú" is dropped.

Examples:

  • -AR Verbs:
    • cantar (to sing) -> él/ella/Ud. canta -> ¡Canta! (Sing!)
    • escuchar (to listen) -> él/ella/Ud. escucha -> ¡Escucha! (Listen!)
    • preparar (to prepare) -> él/ella/Ud. prepara -> ¡Prepara! (Prepare!)
  • -ER Verbs:
    • aprender (to learn) -> él/ella/Ud. aprende -> ¡Aprende! (Learn!)
    • correr (to run) -> él/ella/Ud. corre -> ¡Corre! (Run!)
    • leer (to read) -> él/ella/Ud. lee -> ¡Lee! (Read!)
  • -IR Verbs:
    • abrir (to open) -> él/ella/Ud. abre -> ¡Abre! (Open!)
    • escribir (to write) -> él/ella/Ud. escribe -> ¡Escribe! (Write!)
    • subir (to go up) -> él/ella/Ud. sube -> ¡Sube! (Go up!)

Key Characteristics and Usage Notes

  1. No Subject Pronoun 'tú': This is a critical point. In Spanish, you never add the word "tú" to an affirmative tú command. The command form itself intrinsically communicates that you are addressing "tú." Saying "¡Habla tú!" sounds unnatural, redundant, and can even be perceived as rude or emphatic in an unwelcome way. Always just use the verb form: ¡Habla!
  2. Required Punctuation: When writing commands in Spanish, remember to use both an inverted exclamation mark at the beginning (¡) and a regular exclamation mark at the end (!). This is a standard rule for all exclamatory sentences in Spanish.
  3. Regular Verbs Only: This specific rule (using the él/ella/Ud. present tense) applies only to regular -ar, -er, and -ir verbs. There are a small number of irregular tú commands, which you will learn in a later section.
  4. Accent Marks: Any accent marks that are part of the original él/ella/Ud. present-tense verb form will remain in the command. For most regular verbs, this means no new accent marks are introduced.
  5. Attaching Elements: You can attach nouns, object pronouns, or place words directly to the end of an affirmative command to make it more specific. For example, instead of just ¡Lava! (Wash!), you might say ¡Lava los platos! (Wash the dishes!). More detailed rules for pronoun attachment will be covered later.

Vocabulary for Commands

The chapter provides useful vocabulary, particularly for Los quehaceres del hogar (Household Chores). These are common contexts for using commands.

  • Lavar los platos (to wash the dishes) -> ¡Lava los platos! (Wash the dishes!)
  • Limpiar el baño (to clean the bathroom) -> ¡Limpia el baño! (Clean the bathroom!)
  • Pasar la aspiradora (to vacuum) -> ¡Pasa la aspiradora! (Vacuum!)
  • Sacar la basura (to take out the trash) -> ¡Saca la basura! (Take out the trash!)
  • Organizar tu cuarto (to organize your room) -> ¡Organiza tu cuarto! (Organize your room!)

It's important to note that some common chore verbs, like poner (to put/set) -> pon and hacer (to do/make) -> haz, are irregular and will be addressed in a subsequent section.

Practice and Application

Applying these rules in practice is crucial for mastery. Try creating commands for everyday situations, such as giving instructions in the kitchen or telling family members about chores. Remember the structure: Command + (object/phrase) + (porque + reason). For example: ¡Lava los vegetales porque están sucios! (Wash the vegetables because they are dirty!).

Review Questions

  1. What is the basic rule for forming regular affirmative tú commands in Spanish? Provide an example for an -ar verb, an -er verb, and an -ir verb.
  2. Explain why it is incorrect to use the pronoun "tú" directly with an affirmative tú command (e.g., "¡Habla tú!"). What is the correct way to give the command "Speak!"?
  3. Besides the formation rule, what two crucial punctuation marks are required when writing an affirmative tú command in Spanish?
  4. Imagine you are helping a friend with their homework. Using three different regular verbs, create three affirmative tú commands you might give them (e.g., to read, write, or study something).
  5. Think of a simple instruction you might give to a younger sibling or friend using an affirmative tú command. Then, add a "porque" clause to explain the reason for the command.
