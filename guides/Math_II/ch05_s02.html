Solving One-Step and Two-Step Inequalities Study Guide

Overview

This section explains how to solve one-step and two-step inequalities, highlighting their similarities to solving equations. The crucial distinction lies in the need to reverse the inequality symbol when multiplying or dividing both sides by a negative number, a common pitfall for students. It covers the properties of inequality and provides examples for various scenarios, including practical applications.

Key Concepts

  • Inverse Operations for Isolation: To solve an inequality, you isolate the variable by using inverse operations, just as you would when solving an equation.
  • Addition and Subtraction Properties of Inequality: Adding or subtracting the same number on both sides of an inequality does not change the direction of the inequality symbol.
  • Multiplication and Division Properties of Inequality:
    • Multiplying or dividing both sides by a positive number does not change the direction of the inequality symbol.
    • Multiplying or dividing both sides by a negative number reverses the direction of the inequality symbol (e.g., < becomes >, ≥ becomes ≤).
  • Common Error: Forgetting to reverse the inequality sign when multiplying or dividing by a negative number is the most frequent mistake.
  • Order of Operations for Two-Step Inequalities: When solving two-step inequalities, first undo any addition or subtraction, then undo any multiplication or division.
  • Checking Solutions: To verify a solution, choose a value that satisfies your final inequality and substitute it into the original inequality to ensure it makes the statement true.

Important Terms

  • Inequality: A mathematical statement that compares two expressions using an inequality symbol (e.g., <, >, ≤, ≥).
  • Inverse Operations: Operations that undo each other, such as addition and subtraction, or multiplication and division. They are used to isolate a variable.
  • Isolate the Variable: The process of manipulating an equation or inequality to get the variable by itself on one side of the symbol.
  • Solution to an Inequality: The set of all numbers that make the inequality a true statement. This is often represented as an interval or by shading on a number line.
  • Inequality Symbol: The symbols used to show the relationship between two expressions:
    • < (less than)
    • > (greater than)
    • ≤ (less than or equal to)
    • ≥ (greater than or equal to)

Detailed Notes

Solving Inequalities: The Basics

Solving inequalities follows many of the same rules as solving equations. The goal is to isolate the variable on one side of the inequality symbol. This is achieved by applying inverse operations to both sides.

The Critical Difference: Multiplying/Dividing by a Negative Number

This is the single most important rule to remember:

  • If you multiply or divide both sides of an inequality by a negative number, you must reverse the direction of the inequality symbol.
    • For example, if you have a < b and you multiply by c where c < 0, the result is ac > bc.
    • Similarly, if you have a < b and you divide by c where c < 0, the result is a/c > b/c.

Properties of Inequality

  1. Addition and Subtraction Properties of Inequality:

    • Adding the same number to both sides of an inequality does not change its direction: If a < b, then a + c < b + c.
    • Subtracting the same number from both sides of an inequality does not change its direction: If a < b, then a - c < b - c.
    • These properties apply to all inequality symbols (<, >, ≤, ≥).

  2. Multiplication and Division Properties of Inequality:

    • By a Positive Number (c > 0): Multiplying or dividing both sides by a positive number does not change the direction of the inequality: If a < b and c > 0, then ac < bc.
    • By a Negative Number (c < 0): Multiplying or dividing both sides by a negative number reverses the direction of the inequality: If a < b and c < 0, then ac > bc.

    Caution: This reversal is the most common mistake! Always ask yourself: "Did I multiply or divide by a negative number?" If yes, flip the sign.

Solving One-Step Inequalities

One-step inequalities require a single inverse operation to isolate the variable.

  • Example 1: Addition/Subtraction Inequality Solve x - 4 > 7

    1. Add 4 to both sides: x - 4 + 4 > 7 + 4
    2. Simplify: x > 11
    • Graphing: An open circle at 11 (because x is greater than 11, not equal to) and shading to the right.
    • Check: Pick a value x > 11, like x = 12. Substitute into the original: 12 - 4 = 8. Since 8 > 7 is true, the solution is correct.

  • Example 2: Division by a Negative Number Solve -3x ≤ 18

    1. Divide both sides by -3. Since -3 is negative, reverse the inequality symbol: -3x / -3 ≥ 18 / -3
    2. Simplify: x ≥ -6
    • Check: Pick a value x ≥ -6, like x = -7 (a value not in the solution set) or x = -6 (a value in the solution set). Let's try x = -6: -3(-6) = 18. Since 18 ≤ 18 is true, the solution is correct. If you picked x = -7 (not in the solution set): -3(-7) = 21. Since 21 ≤ 18 is false, this confirms -7 is not a solution, and x ≥ -6 is correct.

Solving Two-Step Inequalities

Two-step inequalities involve two inverse operations. The process is identical to solving two-step equations:

  1. Undo addition or subtraction first.
  2. Undo multiplication or division second.

  • Example 1: Standard Two-Step Inequality Solve 2x + 3 > 11

    1. Subtract 3 from both sides (undo addition): 2x + 3 - 3 > 11 - 3 2x > 8
    2. Divide both sides by 2 (undo multiplication). 2 is positive, so the inequality symbol does not reverse: 2x / 2 > 8 / 2 x > 4
    • Check: Pick x = 5. 2(5) + 3 = 10 + 3 = 13. Since 13 > 11 is true, the solution is correct.

  • Example 2: Two-Step with a Negative Coefficient (Application) A cell-phone plan costs $25 per month plus $0.10 per text message. Maya wants her monthly bill to be at most $40. How many text messages t can she send?

    1. Set up the inequality: "At most $40" means less than or equal to $40: 25 + 0.10t ≤ 40
    2. Subtract 25 from both sides: 0.10t ≤ 15
    3. Divide both sides by 0.10 (a positive number, so no symbol reversal): t ≤ 150
    • Answer: Maya can send at most 150 text messages per month.
    • Check: If t = 150, 25 + 0.10(150) = 25 + 15 = 40. Since 40 ≤ 40 is true, the solution is correct.

General Tip for Checking Solutions

Always pick a value from the solution region of your final answer and substitute it back into the original inequality. If the original inequality remains true, your solution is likely correct. If it results in a false statement, you've made an error and need to review your steps, especially looking for forgotten sign reversals.

Review Questions

  1. What is the fundamental difference between solving an inequality and solving an equation? Explain the one critical rule you must remember for inequalities.
  2. Provide an example of an inequality where you would need to reverse the inequality symbol, and walk through the steps of solving it. Why is it important to reverse the symbol in this specific case?
  3. Describe the correct order of operations for solving a two-step inequality. If you were solving -2x + 5 < 17, what would be your first and second steps?
  4. You solved an inequality and got x < -3. How would you check if your solution is correct? Give an example of a test value you would use and explain why.
  5. A student solved 4 - 5y ≥ 24 and got y ≥ -4. Without solving it yourself, identify the likely mistake the student made and explain why it's a common error.
