Study Guide: Data Displays and Measures of Spread

Overview

This section serves as a review of basic statistical measures of center (mean, median, mode) and introduces important concepts for visualizing and understanding data distribution. You will learn about two powerful data displays—histograms and box plots—and how to calculate measures of spread like range and interquartile range (IQR). The section also covers how to identify potential outliers in a data set.

Key Concepts

  • Measures of Center:
    • Mean: The average of a data set, calculated by summing all values and dividing by the count of values. Represents the "balancing point."
    • Median: The middle value of a data set when it's ordered from least to greatest. If there's an even number of data points, it's the average of the two middle values. It's less affected by extreme values than the mean.
    • Mode: The value that appears most frequently in a data set. A data set can have one mode, multiple modes, or no mode.
  • Range: A simple measure of spread, calculated as the difference between the maximum and minimum values in a data set. It shows the total spread of the data.
  • Histograms: A bar graph that shows the frequency distribution of data across specific intervals (called "bins"). The bars touch to indicate continuous intervals, and their height represents the count (frequency) of data points within each bin.
  • Five-Number Summary: A set of five key values that describe the distribution of a data set:
    • Minimum: The smallest value in the data set.
    • First Quartile (Q1): The median of the lower half of the data. 25% of the data falls below Q1.
    • Median (Q2): The middle value of the entire data set. 50% of the data falls below Q2.
    • Third Quartile (Q3): The median of the upper half of the data. 75% of the data falls below Q3.
    • Maximum: The largest value in the data set.
  • Interquartile Range (IQR): A measure of spread that represents the middle 50% of the data. It is calculated as the difference between the third quartile (Q3) and the first quartile (Q1): IQR = Q3 - Q1.
  • Box Plot (Box-and-Whisker Plot): A visual representation of the five-number summary. It clearly shows the median, quartiles, and the overall spread and symmetry of the data, as well as potential outliers.
  • Outliers: Data values that are significantly different from other values in the set. Informally, a value is considered an outlier if it is more than 1.5 * IQR below Q1 or above Q3.

Important Terms

  • Measures of Center: Statistical values (mean, median, mode) that describe the "middle" or "typical" value of a data set.
  • Mean: The arithmetic average of a data set.
  • Median: The middle value in an ordered data set.
  • Mode: The most frequently occurring value in a data set.
  • Range: The difference between the maximum and minimum values in a data set, indicating the total span of the data.
  • Histogram: A type of bar graph that displays the frequency distribution of numerical data divided into intervals (bins).
  • Bins: The intervals or ranges into which data is grouped for a histogram.
  • Frequency: The number of times a particular data value or a value within a specific interval appears in a data set.
  • Five-Number Summary: A set of five descriptive statistics for a data set: minimum, first quartile (Q1), median (Q2), third quartile (Q3), and maximum.
  • Quartiles (Q1, Q2, Q3): Values that divide an ordered data set into four equal parts. Q1 is the 25th percentile, Q2 (median) is the 50th percentile, and Q3 is the 75th percentile.
  • Interquartile Range (IQR): A measure of statistical dispersion, equal to the difference between the upper (Q3) and lower (Q1) quartiles.
  • Box Plot (Box-and-Whisker Plot): A standardized way of displaying the distribution of data based on the five-number summary.
  • Outlier: A data point that lies an abnormal distance from other values in a random sample from a population.

Detailed Notes

Review of Measures of Center

Before diving into data displays and spread, it's crucial to recall how to find the center of a data set.

  • Mean ($x$): To calculate the mean, sum all the individual data points ($x_1, x_2, ..., x_n$) and then divide by the total number of data points ($n$).
    • Formula: $x = \frac{x_1 + x_2 + ... + x_n}{n}$
  • Median: This is the middle value when the data set is arranged in numerical order.
    • If the number of data points ($n$) is odd, the median is the single middle value.
    • If $n$ is even, there are two middle values; the median is the average of these two values.
  • Mode: The mode is the value that appears most often in the data set. A data set can have one mode (unimodal), multiple modes (multimodal), or no mode if all values appear with the same frequency.

Example: For the data set $12, 15, 18, 15, 22, 30, 15$:

  1. Order the data: $12, 15, 15, 15, 18, 22, 30$
  2. Mean: $(12+15+15+15+18+22+30) / 7 = 127 / 7 \approx 18.1$
  3. Median: The middle value (4th value) is $15$.
  4. Mode: $15$ appears 3 times, which is more than any other value. So, the mode is $15$.

Range

The range is the simplest measure of spread. It tells you the total extent of your data.

  • Formula: range = maximum value - minimum value

Example (continued): For the data set $12, 15, 15, 15, 18, 22, 30$:

  • Maximum value = $30$
  • Minimum value = $12$
  • Range = $30 - 12 = 18$

Histograms

A histogram is a graphical display that shows the frequency distribution of numerical data.

  • It looks like a bar graph, but the bars touch each other to signify that the data is continuous and grouped into intervals (often called bins).
  • The horizontal axis represents the data intervals.
  • The vertical axis represents the frequency (count) of data points falling into each interval.
  • The height of each bar corresponds to the frequency for that specific bin.

Example: Reading a Histogram Imagine a histogram of test scores with the following bins and frequencies:

  • $60-69$: 3 students
  • $70-79$: 8 students
  • $80-89$: 12 students
  • $90-100$: 5 students

From this, we can answer questions like:

  • How many students scored in the $80-89$ range? 12 students.
  • How many students took the test? $3 + 8 + 12 + 5 = \textbf{28 students}$.
  • Which interval has the highest frequency? $80-89$.

Box Plots and the Five-Number Summary

Box plots are powerful visual tools that display the five-number summary of a data set.

The five-number summary consists of:

  1. Minimum: The smallest value in the data set.
  2. First Quartile (Q1): The median of the lower half of the data (values below the overall median).
  3. Median (Q2): The overall middle value of the data set.
  4. Third Quartile (Q3): The median of the upper half of the data (values above the overall median).
  5. Maximum: The largest value in the data set.

The interquartile range (IQR) is a key measure of spread derived from the quartiles:

  • Formula: IQR = Q3 - Q1
  • The IQR represents the range of the middle 50% of the data, making it less sensitive to extreme values than the overall range.

Steps to create a Five-Number Summary:

  1. Order the data from least to greatest.
  2. Identify the Minimum and Maximum values.
  3. Find the Median (Q2) of the entire data set.
  4. Find Q1 by taking the median of all data points below the overall median.
  5. Find Q3 by taking the median of all data points above the overall median.

Example: Creating a Five-Number Summary Data set: $3, 5, 7, 8, 10, 12, 14, 15, 18, 20, 22$ (already ordered, $n=11$)

  • Minimum: $3$
  • Maximum: $22$
  • Median (Q2): The 6th value is $12$.
  • Lower half: $3, 5, 7, 8, 10$. Q1 (median of lower half) = $7$.
  • Upper half: $14, 15, 18, 20, 22$. Q3 (median of upper half) = $18$.
  • IQR: $Q3 - Q1 = 18 - 7 = 11$.

A box plot visually represents these five numbers:

  • A "box" is drawn from Q1 to Q3.
  • A line inside the box marks the median (Q2).
  • "Whiskers" extend from the box to the minimum and maximum values.

Comparing Data Sets

Box plots and the five-number summary are excellent for comparing two or more data sets.

  • Center: Compare the medians (Q2) to see which data set has a higher or lower typical value.
  • Spread: Compare the IQRs and overall ranges. A larger IQR or range indicates that the data is more spread out.
  • Symmetry/Skewness: The position of the median within the box and the length of the whiskers can give clues about the data's distribution.

Example: Comparing with Box Plots

  • Class A Scores: Min=55, Q1=65, Med=75, Q3=85, Max=95
    • IQR = $85 - 65 = 20$
    • Range = $95 - 55 = 40$
  • Class B Scores: Min=60, Q1=70, Med=80, Q3=82, Max=90
    • IQR = $82 - 70 = 12$
    • Range = $90 - 60 = 30$

Comparison:

  • Center: Class B has a higher median ($80$) than Class A ($75$), suggesting Class B generally performed better.
  • Spread: Class A has a larger IQR ($20$ vs. $12$) and a larger overall range ($40$ vs. $30$). This means Class A's scores are more spread out, indicating greater variability in student performance, while Class B's scores are more consistent.

Identifying Outliers

Outliers are data points that lie far away from the rest of the data. They can significantly affect the mean and range.

Informal Outlier Rule: A data value is considered an outlier if it is:

  • More than 1.5 * IQR below the first quartile (Q1).
    • Lower Fence = $Q1 - (1.5 \times IQR)$
  • More than 1.5 * IQR above the third quartile (Q3).
    • Upper Fence = $Q3 + (1.5 \times IQR)$

Any data point falling outside these "fences" is considered an outlier.

Example: Checking for Outliers Data set has $Q1 = 10$, $Q3 = 22$.

  1. Calculate IQR: $IQR = Q3 - Q1 = 22 - 10 = 12$.
  2. Calculate 1.5 * IQR: $1.5 \times 12 = 18$.
  3. Calculate Upper Fence: $Q3 + 1.5 \times IQR = 22 + 18 = 40$.
  4. Calculate Lower Fence: $Q1 - 1.5 \times IQR = 10 - 18 = -8$.

Now, check if the value $42$ is an outlier:

  • Is $42 > 40$? Yes. Therefore, $42$ is an outlier.

Review Questions

  1. Explain the difference between the mean, median, and mode. In what situations might you prefer to use the median over the mean to describe the center of a data set?
  2. Describe what a histogram visually communicates about a data set. How does it differ from a standard bar graph?
  3. What are the five components of the five-number summary? How is the interquartile range (IQR) calculated, and what does it tell you about the data?
  4. Imagine you are comparing the test scores of two different classes using their box plots. What specific features of the box plots would you look at to determine which class had a higher "typical" score and which class had more consistent (less spread out) scores?
  5. A data set has $Q1 = 50$ and $Q3 = 70$. Using the informal outlier rule, determine if a value of $95$ or a value of $15$ would be considered an outlier. Show your calculations.
