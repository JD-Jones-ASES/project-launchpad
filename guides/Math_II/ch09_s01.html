Chapter 9, Section 1: Relations and Functions Study Guide

Overview

This section introduces the fundamental mathematical concepts of relations and functions, explaining how they describe pairings between two quantities. It defines a relation as any set of ordered pairs and a function as a special type of relation where each input has exactly one output. The Vertical Line Test is presented as a visual method to determine if a graph represents a function, and methods for identifying domain and range from different representations are discussed.

Key Concepts

  • Relation: Any set of ordered pairs $(x, y)$. This is a general term for any way of pairing values.
  • Domain: The set of all possible input values (the $x$-values) in a relation.
  • Range: The set of all possible output values (the $y$-values) in a relation.
  • Function: A specific type of relation where every input ($x$-value) is paired with exactly one output ($y$-value). This means that for any given $x$, there can only be one corresponding $y$. An $x$-value cannot appear more than once with different $y$-values.
  • Vertical Line Test: A graphical method to determine if a relation is a function. If any vertical line intersects the graph at more than one point, the graph does not represent a function. This is because multiple intersections would mean one $x$-value has multiple $y$-values.
  • Identifying Domain and Range: From a set of points, the domain is simply the list of all $x$-coordinates, and the range is the list of all $y$-coordinates. From a graph, the domain covers all $x$-values spanned by the graph, and the range covers all $y$-values spanned by the graph.

Important Terms

  • Relation: A set of ordered pairs $(x, y)$.
  • Domain: The set of all $x$-values (inputs) in a relation.
  • Range: The set of all $y$-values (outputs) in a relation.
  • Function: A relation in which every input ($x$-value) is paired with exactly one output ($y$-value).
  • Input: The $x$-value in an ordered pair, representing an independent variable.
  • Output: The $y$-value in an ordered pair, representing a dependent variable that depends on the input.
  • Vertical Line Test: A visual test applied to the graph of a relation: if no vertical line intersects the graph at more than one point, the relation is a function.

Detailed Notes

Understanding Relations

Many situations in the real world involve one quantity depending on another. For example, your work hours determine your pay, or the outside temperature influences your choice of clothing. In mathematics, we represent these pairings using ordered pairs $(x, y)$, where $x$ is the input and $y$ is the output. Any collection of these ordered pairs is called a relation.

  • Domain and Range: For any relation, the set of all $x$-values is called the domain, and the set of all $y$-values is called the range. These sets define the scope of the relation.

Defining a Function

A function is a very important type of relation with a specific rule: every input ($x$-value) must be paired with exactly one output ($y$-value). This is the critical distinction.

  • What makes a relation a function?

    • Each unique $x$-value appears only once in the set of ordered pairs.
    • If you have a set like $(1, 3), (2, 5), (3, 7)$, this is a function because each $x$ (1, 2, 3) has only one corresponding $y$.
    • It's okay for different $x$-values to have the same $y$-value. For example, $(-2, 3), (0, 3), (4, 3)$ is still a function because each distinct $x$-value ($-2, 0, 4$) has only one $y$-value (which happens to be 3 for all of them).

  • What makes a relation NOT a function?

    • If a single $x$-value is paired with two or more different $y$-values.
    • For example, $(1, 4), (2, 5), (1, 6)$ is NOT a function because the input $x=1$ is paired with two different outputs, $4$ and $6$. This violates the "exactly one output" rule.

The Vertical Line Test

When a relation is represented graphically, we can use the Vertical Line Test to quickly determine if it's a function.

  • How it works: Imagine drawing vertical lines across the graph. If any vertical line intersects the graph at more than one point, then the relation is not a function.
  • Why it works: A vertical line represents a single $x$-value. If it crosses the graph at two or more points, it means that one $x$-value corresponds to multiple $y$-values, which contradicts the definition of a function.
    • Example: A circle is not a function because a vertical line passing through its center would intersect the circle at two points (an upper $y$-value and a lower $y$-value for the same $x$).

Identifying Domain and Range from Different Representations

  • From a Set of Ordered Pairs:

    • Domain: List all unique $x$-coordinates.
    • Range: List all unique $y$-coordinates.
    • Example: For the relation ${(1, 3), (2, 5), (3, 7)}$, Domain = ${1, 2, 3}$, Range = ${3, 5, 7}$.

  • From a Table:

    • Similar to ordered pairs, simply list the unique $x$-values from the input column for the domain and unique $y$-values from the output column for the range.
    • To check if it's a function, verify that no $x$-value appears more than once.

  • From a Graph:

    • Domain: Look at the horizontal extent of the graph. The domain consists of all $x$-values from the leftmost point to the rightmost point that the graph occupies. This is often expressed using inequalities or interval notation.
    • Range: Look at the vertical extent of the graph. The range consists of all $y$-values from the lowest point to the highest point that the graph occupies. This is also expressed using inequalities or interval notation.
    • Example: For a line segment from $(-3, 1)$ to $(5, 7)$:
      • Domain: The $x$-values go from $-3$ to $5$. So, $-3 \le x \le 5$ (or $[-3, 5]$ in interval notation).
      • Range: The $y$-values go from $1$ to $7$. So, $1 \le y \le 7$ (or $[1, 7]$ in interval notation).

Functions are Central to Algebra

Functions are not just an abstract concept; they are fundamental to algebra and beyond. Many equations you've encountered, such as linear equations in the form $y = mx + b$, are functions. This means that for every $x$ you choose, there's only one $y$ that satisfies the equation. You will continue to build upon your understanding of functions throughout your math education.

Review Questions

  1. Explain in your own words the difference between a "relation" and a "function." Why is the "exactly one output" rule so important for functions?
  2. Given the following relations, determine which ones are functions and explain your reasoning for each: a. ${(5, 2), (6, 3), (7, 2), (8, 4)}$ b. ${(-1, 0), (0, 1), (-1, 2), (2, 3)}$ c. A table where $x$ values are [1, 2, 3, 4] and $y$ values are [10, 20, 30, 40]
  3. Describe a real-world scenario that can be modeled by a function and another real-world scenario that can be modeled by a relation but is not a function. Justify your choices based on the definitions.
  4. A student is confused about why the Vertical Line Test works. Explain to them how it's connected to the definition of a function, particularly focusing on what it means when a vertical line intersects a graph at multiple points.
  5. Consider a relation represented by a square drawn on a coordinate plane. a. Would this relation be a function? Why or why not, using the Vertical Line Test? b. If the square has vertices at $(0,0), (2,0), (2,2), (0,2)$, what would be the domain and range of this relation?
