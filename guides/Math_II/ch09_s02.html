Chapter 9, Section 2: Function Notation Study Guide

Overview

This section introduces function notation, a standard way mathematicians write and work with functions, replacing the traditional "$y = $" format. It covers how to evaluate functions for specific numerical inputs and algebraic expressions, combine operations involving multiple functions, and identify restrictions on a function's domain to prevent undefined operations.

Key Concepts

  • Function Notation: A formal way to represent functions using f(x) instead of y. The f names the function, and x explicitly indicates the input variable. f(x) represents the output value, not "f times x."
  • Evaluating Functions: To find the output of a function for a given input, substitute the input value (number or expression) for every instance of the input variable (x) in the function's rule and then simplify the resulting expression.
  • Evaluating with Expressions: The process of substitution also applies when the input is an algebraic expression (e.g., a + 1), requiring careful algebraic simplification after substitution.
  • Working with Multiple Functions: Different functions can be defined and evaluated independently. Their outputs can then be combined (added, subtracted, etc.) according to the problem's requirements.
  • Domain Restrictions: Certain mathematical operations are undefined (e.g., division by zero, square root of a negative number). The domain of a function is the set of all allowable input values (x) that do not lead to these undefined operations.

Important Terms

  • Function Notation: A symbolic way to write functions, typically f(x), where f is the name of the function and x is the input variable.
  • Input: The value or expression that is substituted into a function (represented by x in f(x)).
  • Output: The result obtained after evaluating a function with a given input (represented by f(x)).
  • Evaluate: To find the value of a function for a specific input by substituting the input into the function's rule and simplifying.
  • Domain: The set of all possible input values for which a function is defined (i.e., produces a real number output without undefined operations).
  • Radicand: The expression or number under a radical sign (e.g., the x in sqrt(x)). For real numbers, the radicand of an even-indexed root must be non-negative.
  • Denominator: The bottom part of a fraction. The denominator of a fraction cannot be zero.

Detailed Notes

Introduction to Function Notation

Function notation is a powerful tool in mathematics. Instead of writing y = 3x - 2, we use f(x) = 3x - 2.

  • The f is the name given to the function. We can use other letters like g, h, etc.
  • The x inside the parentheses (x) explicitly tells us that x is the input variable for this function.
  • f(x) is read as "f of x" and represents the output of the function when x is the input. It does not mean "f times x."
  • This notation makes it very clear what the input is and what the resulting output represents.

Evaluating Functions

To evaluate a function f(x) at a specific value, say a, you write f(a). This means you replace every x in the function's rule with a and then simplify the expression.

Example 1: Evaluating a Linear Function Let f(x) = 3x - 2.

  • Find f(5):
    • Replace x with 5: f(5) = 3(5) - 2
    • Simplify: 15 - 2 = 13
    • So, f(5) = 13.
  • Find f(-1):
    • Replace x with -1: f(-1) = 3(-1) - 2
    • Simplify: -3 - 2 = -5
    • So, f(-1) = -5.
  • Find f(0):
    • Replace x with 0: f(0) = 3(0) - 2
    • Simplify: 0 - 2 = -2
    • So, f(0) = -2.

Example 2: Evaluating a Quadratic Function Let g(x) = x^2 + 4x - 1.

  • Find g(3):
    • Replace x with 3: g(3) = (3)^2 + 4(3) - 1
    • Simplify: 9 + 12 - 1 = 20
    • So, g(3) = 20.
  • Find g(-2):
    • Replace x with -2: g(-2) = (-2)^2 + 4(-2) - 1
    • Simplify: 4 - 8 - 1 = -5
    • So, g(-2) = -5.

Evaluating with Expressions

The input to a function doesn't always have to be a number; it can be another algebraic expression. The evaluation process remains the same: substitute the entire expression for x.

Example: Input Is an Expression Let f(x) = 2x + 5.

  • Find f(a + 1):
    • Replace x with the entire expression (a + 1): f(a + 1) = 2(a + 1) + 5
    • Distribute the 2: 2a + 2 + 5
    • Combine like terms: 2a + 7
    • So, f(a + 1) = 2a + 7.

Working with Multiple Functions

You can have multiple functions defined in a single problem, often with different names (e.g., f(x) and g(x)). You can evaluate each function independently and then combine their results as needed.

Example: Two Functions Let f(x) = x + 3 and g(x) = 2x^2.

  • Find f(4) + g(2):
    • First, evaluate f(4): f(4) = 4 + 3 = 7
    • Next, evaluate g(2): g(2) = 2(2)^2 = 2(4) = 8
    • Finally, add the results: f(4) + g(2) = 7 + 8 = 15
    • So, f(4) + g(2) = 15.

Domain Restrictions

The domain of a function is the set of all real numbers for which the function produces a real number output. Sometimes, certain input values must be excluded from the domain because they lead to undefined mathematical operations.

Common Domain Restrictions:

  1. Division by zero: You cannot divide by zero. If a function has a variable in the denominator of a fraction, any value of x that makes the denominator zero must be excluded.
    • For example, if f(x) = 1/x, then x cannot be 0.
  2. Square root of a negative number: For real numbers, you cannot take the square root of a negative number. If a function involves a square root (or any even-indexed root) with a variable expression inside, that expression (the radicand) must be greater than or equal to zero.
    • For example, if f(x) = sqrt(x), then x must be greater than or equal to 0.

Example: Finding Domain Restrictions

  • State the domain of f(x) = 5/(x - 3):
    • This function involves division, so the denominator cannot be zero.
    • Set the denominator equal to zero to find the excluded value: x - 3 = 0 implies x = 3.
    • Therefore, x cannot be 3.
    • Domain: All real numbers except 3.
  • State the domain of g(x) = sqrt(x + 2):
    • This function involves a square root, so the radicand must be non-negative.
    • Set the radicand greater than or equal to zero: x + 2 >= 0.
    • Solve for x: x >= -2.
    • Domain: All real numbers x such that x >= -2.

Connecting to Previous Knowledge: Remember that function notation f(x) = 2x + 3 is essentially the same as the linear equation y = 2x + 3 that you studied in Chapter 8 and Algebra I. Function notation provides a more formal and explicit way to describe the relationship between input and output, which is crucial as you explore more complex function types beyond linear equations.

Review Questions

  1. Explain in your own words why mathematicians use function notation (f(x)) instead of just y = .... What advantages does f(x) offer?
  2. Given the function h(x) = -x^2 + 5x - 3, evaluate h(-3) and h(1/2). Show your steps clearly.
  3. Let p(x) = 3x - 1 and q(x) = x^2 + 2. Find p(2) - q(-1).
  4. Describe the two most common types of domain restrictions encountered in algebra. For each type, provide an example of a function and explain how to determine its domain.
  5. Consider the function k(x) = sqrt(x - 4) / (x - 6). What steps would you take to find the domain of this function? What values of x would be excluded from its domain?
