Chapter 7, Section 3: Proportions in Similar Figures Study Guide

Overview

This section builds on the concept that similar figures have proportional sides by teaching you how to set up and solve these proportions to find unknown side lengths. These skills are essential for practical applications such as indirect measurement, which allows for calculating distances or heights that are difficult to measure directly.

Key Concepts

  • Similar Figures and Proportional Sides: When two figures are similar, the ratios of their corresponding side lengths are equal. This fundamental property allows us to set up equations (proportions) to find missing measurements.
  • Scale Factor: The ratio of any pair of corresponding side lengths between two similar figures. It indicates how much larger or smaller one figure is compared to the other. The order matters when stating the scale factor (e.g., A to B is different from B to A).
  • Setting Up Proportions: To solve for an unknown side, you must correctly identify corresponding sides and place them in the same relative positions within the proportion (e.g., numerator/numerator, denominator/denominator, or left side/left side, right side/right side).
  • Solving Proportions: The standard technique for solving proportions is cross-multiplication. If you have a/b = c/d, then ad = bc.
  • Indirect Measurement: A practical application of similar figures and proportions used to find lengths or distances that cannot be measured directly. Common examples include shadow problems (using similar triangles formed by objects and their shadows) and map scales.

Important Terms

  • Scale Factor: The ratio of the lengths of two corresponding sides of two similar figures. For example, if two similar triangles have corresponding sides of length 6 and 9, the scale factor from the first to the second is 6/9 = 2/3.
  • Proportion: An equation that states that two ratios are equal. For example, a/b = c/d is a proportion.
  • Indirect Measurement: A technique that uses proportions and similar figures to measure distances or heights that are difficult or impossible to measure directly.
  • Corresponding Sides: Sides that are in the same relative position in two similar figures. When figures are similar, their corresponding sides are proportional.

Detailed Notes

Introduction to Proportions in Similar Figures

Recall from Section 7-2 that similar figures have corresponding angles that are congruent and corresponding sides that are proportional. This section focuses on using the proportionality of sides to solve for unknown lengths. This skill is incredibly useful in real-world scenarios, particularly for indirect measurement.

Scale Factor

The scale factor is a crucial concept when working with similar figures.

  • Definition: It is defined as the ratio of any pair of corresponding side lengths.
  • Calculation Example: If triangle ABC is similar to triangle DEF (written as $ ABC DEF$) and side AB = 6 while its corresponding side DE = 9, then the scale factor from $ ABC$ to $ DEF$ is $6/9 = 2/3$.
  • Importance of Order: The scale factor depends on which figure is listed first. If the scale factor from $ ABC$ to $ DEF$ is $2/3$, then the scale factor from $ DEF$ to $ ABC$ would be $9/6 = 3/2$. Always be clear about which figure you are scaling from and to.

Solving Proportions with Similar Figures

When two figures are similar, you can set up a proportion using any two pairs of corresponding sides. The key is to be consistent in how you set up the ratios.

  • Standard Technique: Cross-Multiplication: To solve an equation where two ratios are equal, like $a/b = c/d$, you can cross-multiply: $a \times d = b \times c$.

  • Example: Finding a Missing Side Length Given: $ ABC DEF$. Known sides: $AB = 8$, $BC = 12$, $DE = 10$. Find: $EF$.

    1. Set up the proportion: Match corresponding sides. We'll set up the ratio of sides from $ ABC$ over $ DEF$: $AB/DE = BC/EF$ $8/10 = 12/EF$

    2. Cross-multiply: $8 \times EF = 10 \times 12$ $8 \times EF = 120$

    3. Solve for EF: $EF = 120 / 8$ $EF = 15$

  • Checkpoint Example: Given: $ PQR STU$. Known sides: $PQ = 6$, $QR = 9$, $ST = 10$. Find: $TU$.

    1. Set up the proportion: $PQ/ST = QR/TU$ $6/10 = 9/TU$

    2. Cross-multiply: $6 \times TU = 10 \times 9$ $6 \times TU = 90$

    3. Solve for TU: $TU = 90 / 6$ $TU = 15$

Applications of Proportions (Indirect Measurement)

Indirect measurement is a powerful application of similar figures, allowing you to find unknown distances or heights without measuring them directly.

  • Example: Shadow Problem (Indirect Measurement) Problem: A flagpole casts a shadow $12$ feet long. At the same time, a $5.5$-foot-tall student casts a shadow $3$ feet long. How tall is the flagpole?

    1. Identify Similar Triangles: The sun's rays hit both the flagpole and the student at the same angle, creating two similar right triangles (object, shadow, sun ray).
    2. List Knowns and Unknowns:
      • Student height = $5.5$ ft
      • Student shadow = $3$ ft
      • Flagpole shadow = $12$ ft
      • Flagpole height = $h$ (unknown)
    3. Set up the proportion: Be consistent. We can compare (height/shadow) for both objects: $h/12 = 5.5/3$
    4. Cross-multiply: $3 \times h = 5.5 \times 12$ $3h = 66$
    5. Solve for h: $h = 66 / 3$ $h = 22$ The flagpole is $22$ feet tall.

  • Example: Map Distance Problem: On a map, $1$ inch represents $25$ miles. Two cities are $3.5$ inches apart on the map. What is the actual distance between the cities?

    1. Identify the Scale/Ratio: The map scale is a ratio: $1$ inch / $25$ miles.
    2. List Knowns and Unknowns:
      • Map scale: $1$ in : $25$ mi
      • Map distance = $3.5$ in
      • Actual distance = $d$ (unknown)
    3. Set up the proportion: Match map distances to actual distances: $1/25 = 3.5/d$
    4. Cross-multiply: $1 \times d = 25 \times 3.5$ $d = 87.5$ The actual distance is $87.5$ miles.

  • Example: Finding a Side with Decimals Given: $ JKL MNP$. Known sides: $JK = 4.8$, $KL = 7.2$, $MN = 6$. Find: $NP$.

    1. Set up the proportion: $JK/MN = KL/NP$ $4.8/6 = 7.2/NP$

    2. Cross-multiply: $4.8 \times NP = 6 \times 7.2$ $4.8 \times NP = 43.2$

    3. Solve for NP: $NP = 43.2 / 4.8$ $NP = 9$

  • Checkpoint Example: Problem: A tree casts a shadow $15$ feet long. A $4$-foot fence post nearby casts a shadow $2.5$ feet long. How tall is the tree?

    1. Set up the proportion: (Tree height / Tree shadow) = (Post height / Post shadow) $h/15 = 4/2.5$

    2. Cross-multiply: $2.5 \times h = 15 \times 4$ $2.5h = 60$

    3. Solve for h: $h = 60 / 2.5$ $h = 24$ ft.

Important Caution

When writing a proportion, always ensure that corresponding parts are in the same position. For instance, if you put the height of the first object in the numerator, the height of the second object must also be in the numerator of its ratio. Similarly, if a denominator represents a shadow length for one object, the other denominator must also be a shadow length for the corresponding object. Mixing up these positions will lead to an incorrect answer.

Review Questions

  1. Explain what the "scale factor" of two similar figures represents and how its value changes if you reverse the order of the figures (e.g., from A to B vs. B to A).
  2. Given two similar polygons, describe the steps you would take to set up a proportion to find an unknown side length. What is the most common method for solving this proportion?
  3. Triangle ABC is similar to Triangle XYZ ($ ABC XYZ$). If $AB = 7$ cm, $BC = 10$ cm, and $XY = 21$ cm, what is the length of side $YZ$? Show your work.
  4. A model car is built to a scale where $1$ inch on the model represents $16$ inches on the actual car. If the actual car is $192$ inches long, how long is the model car?
  5. You are trying to find the height of a building. You stand $20$ feet away from the building and measure its shadow to be $100$ feet long. If you are $6$ feet tall and cast a $12$-foot shadow at the same time, how tall is the building? Explain your reasoning for setting up the proportion.
