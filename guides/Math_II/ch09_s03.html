Arithmetic Sequences and Linear Patterns Study Guide

Overview

This section introduces arithmetic sequences, which are ordered lists of numbers where the difference between consecutive terms is constant. This constant difference, known as the common difference, creates a linear pattern that can be described by an explicit formula. Understanding arithmetic sequences provides a foundational link to linear functions, allowing us to predict any term in the sequence or model real-world scenarios with constant rates of change.

Key Concepts

  • Sequences and Terms: A sequence is an ordered list of numbers, and each individual number in the list is called a term.
  • Arithmetic Sequence: A specific type of sequence where the difference between any two consecutive terms is always the same.
  • Common Difference ($d$): The constant difference between consecutive terms in an arithmetic sequence. It is calculated by subtracting any term from its succeeding term ($d = a_{n+1} - a_n$).
  • Explicit Formula for an Arithmetic Sequence: A formula that allows you to find any term ($a_n$) in an arithmetic sequence directly, given the first term ($a_1$), the common difference ($d$), and the term number ($n$). The formula is $a_n = a_1 + (n - 1)d$.
  • Connection to Linear Functions: Arithmetic sequences exhibit a linear relationship. The common difference ($d$) acts as the slope of the linear function, and the "zeroth term" ($a_1 - d$) acts as the y-intercept when the term number ($n$) is treated as the input variable.

Important Terms

  • Sequence: An ordered list of numbers.
  • Term: Each individual number within a sequence.
  • Arithmetic Sequence: A sequence where the difference between consecutive terms is constant.
  • Common Difference ($d$): The constant value obtained by subtracting any term from the term that immediately follows it in an arithmetic sequence ($d = a_{n+1} - a_n$).
  • Explicit Formula: A mathematical formula that directly calculates the value of the $n$th term of a sequence without needing to know the previous terms. For an arithmetic sequence, it is $a_n = a_1 + (n - 1)d$.

Detailed Notes

What is an Arithmetic Sequence?

A sequence is simply an ordered list of numbers. Each number in the list is referred to as a term. For example, in the sequence $3, 7, 11, 15, \dots$, the first term ($a_1$) is 3, the second term ($a_2$) is 7, and so on.

An arithmetic sequence is a special type of sequence where the difference between any two consecutive terms is always the same. This constant difference is called the common difference, denoted by $d$.

To find the common difference, you subtract any term from the term immediately following it: $d = a_{n+1} - a_n$

Examples of Identifying Arithmetic Sequences:

  1. Sequence: $3, 7, 11, 15, 19, \dots$

    • $7 - 3 = 4$
    • $11 - 7 = 4$
    • $15 - 11 = 4$
    • Since the difference is consistently 4, this is an arithmetic sequence with $d = 4$.

  2. Sequence: $2, 6, 18, 54, \dots$

    • $6 - 2 = 4$
    • $18 - 6 = 12$
    • The differences are not constant (4 then 12), so this is not an arithmetic sequence.

  3. Sequence: $10, 7, 4, 1, -2, \dots$

    • $7 - 10 = -3$
    • $4 - 7 = -3$
    • $1 - 4 = -3$
    • Since the difference is consistently -3, this is an arithmetic sequence with $d = -3$.

The Explicit Formula for an Arithmetic Sequence

While you can find subsequent terms by repeatedly adding the common difference, it's inefficient for finding terms far down the sequence (e.g., the 100th term). The Explicit Formula provides a direct way to calculate any $n$th term ($a_n$) in an arithmetic sequence:

$a_n = a_1 + (n - 1)d$

Where:

  • $a_n$ is the $n$th term you want to find.
  • $a_1$ is the first term of the sequence.
  • $n$ is the term number (e.g., for the 5th term, $n=5$).
  • $d$ is the common difference.

Using the Explicit Formula - Examples:

  1. Finding a specific term: An arithmetic sequence has $a_1 = 5$ and $d = 3$. Find $a_{10}$.

    • Here, $a_1 = 5$, $d = 3$, and $n = 10$.
    • $a_{10} = 5 + (10 - 1)(3)$
    • $a_{10} = 5 + (9)(3)$
    • $a_{10} = 5 + 27$
    • $a_{10} = 32$

  2. Finding $d$ and writing the formula: The sequence $12, 8, 4, 0, -4, \dots$ is arithmetic. Write the explicit formula and find $a_{20}$.

    • First, identify $a_1$ and $d$.
      • $a_1 = 12$
      • $d = 8 - 12 = -4$
    • Now, substitute $a_1$ and $d$ into the explicit formula:
      • $a_n = 12 + (n - 1)(-4)$
      • $a_n = 12 - 4n + 4$
      • $a_n = 16 - 4n$ (This is the explicit formula for this sequence)
    • To find $a_{20}$, substitute $n = 20$ into the formula:
      • $a_{20} = 16 - 4(20)$
      • $a_{20} = 16 - 80$
      • $a_{20} = -64$

Connection to Linear Functions

One of the most important aspects of arithmetic sequences is their direct relationship to linear functions. An arithmetic sequence is essentially a linear function where the input variable is the term number ($n$).

  • Common Difference as Slope: The common difference ($d$) of an arithmetic sequence plays the exact same role as the slope ($m$) in a linear function. It represents the constant rate of change between terms.
  • "Zeroth Term" as y-intercept: If you consider the explicit formula $a_n = a_1 + (n - 1)d = a_1 + nd - d = nd + (a_1 - d)$, you can see its resemblance to $y = mx + b$. Here, $d$ is the slope ($m$), and $(a_1 - d)$ is the y-intercept ($b$). The value $a_1 - d$ can be thought of as the "zeroth term" (the term before the first term).

Example: Arithmetic Sequence as a Linear Function

Write the arithmetic sequence $2, 5, 8, 11, \dots$ as a function $f(n)$ and find $f(50)$.

  • Identify $a_1 = 2$ and $d = 5 - 2 = 3$.
  • Using the explicit formula $a_n = a_1 + (n - 1)d$:
    • $f(n) = 2 + (n - 1)(3)$
    • $f(n) = 2 + 3n - 3$
    • $f(n) = 3n - 1$ (This is the linear function representation)
  • To find $f(50)$:
    • $f(50) = 3(50) - 1$
    • $f(50) = 150 - 1$
    • $f(50) = 149$

Applications of Arithmetic Sequences

Arithmetic sequences are useful for modeling real-world situations that involve a constant rate of increase or decrease.

Example: Savings Account

Maria opens a savings account with $200 and deposits $25 each week. Write a formula for the amount $A$ after $n$ weeks and find $A$ after 12 weeks.

  • This is a situation with a starting amount and a constant weekly addition.
  • The initial amount is $200. This can be considered the value at week 0 ($A_0$).
  • The weekly deposit of $25 is the common difference ($d$).
  • If we define $A(n)$ as the amount after $n$ weeks (starting with $n=0$ for the initial deposit), the formula is:
    • $A(n) = 200 + 25n$
  • To find the amount after 12 weeks ($n=12$):
    • $A(12) = 200 + 25(12)$
    • $A(12) = 200 + 300$
    • $A(12) = 500$

Note on Growth Patterns: Arithmetic sequences always change at a constant rate, resulting in a straight-line graph when plotted. In contrast, geometric sequences (which you might study in higher-level algebra) grow by a constant ratio rather than a constant difference, leading to exponential growth.

Review Questions

  1. Explain what defines an arithmetic sequence and how to calculate its common difference. Provide an example of a sequence that is arithmetic and one that is not.
  2. State the explicit formula for an arithmetic sequence. Describe each variable in the formula and explain how it contributes to finding the $n$th term.
  3. An arithmetic sequence has $a_1 = 15$ and $d = -4$. Find the 7th term ($a_7$) and write the explicit formula for this sequence.
  4. How is an arithmetic sequence connected to a linear function? Which part of the arithmetic sequence corresponds to the slope, and which corresponds to the y-intercept of a linear function?
  5. A gym membership costs $50 to join and $25 per month thereafter. If you represent the total cost after $n$ months as an arithmetic sequence, what would be the first term ($a_1$) and the common difference ($d$)? Write an explicit formula for the total cost after $n$ months and use it to find the total cost after 1 year.
