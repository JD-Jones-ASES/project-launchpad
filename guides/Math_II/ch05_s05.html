Study Guide: Absolute Value Equations

Overview

This section extends the concept of absolute value to equations containing more complex expressions within the absolute value bars. It focuses on the fundamental property that an absolute value equation, such as $|x| = k$, can have two solutions, one solution, or no solution, depending on the value of $k$. A critical first step for solving these equations is to isolate the absolute value expression before splitting it into separate cases.

Key Concepts

  • Definition of Absolute Value: The absolute value of a number represents its distance from zero on the number line, always resulting in a non-negative value. For example, $|5| = 5$ and $|-5| = 5$.
  • Property for Solving $|x| = k$:
    • If $k > 0$ (k is positive): The equation $|x| = k$ has two distinct solutions: $x = k$ or $x = -k$. This is because there are two numbers (one positive, one negative) that are $k$ units away from zero.
    • If $k = 0$ (k is zero): The equation $|x| = 0$ has exactly one solution: $x = 0$. Zero is the only number whose distance from zero is zero.
    • If $k < 0$ (k is negative): The equation $|x| = k$ has no solution. This is because the absolute value of any real number is always non-negative (zero or positive); it can never be equal to a negative number.
  • Isolating the Absolute Value: Before applying the property above, any terms outside the absolute value bars on the same side of the equation must be moved to the other side. The absolute value expression must be completely alone on one side of the equation.
  • Checking Solutions: Always substitute your solutions back into the original absolute value equation to verify their correctness.

Important Terms

  • Absolute Value Equation: An equation that contains an absolute value expression, such as $|x + 3| = 7$.
  • Absolute Value: The distance of a number from zero on the number line, denoted by vertical bars (e.g., $|x|$). It is always non-negative.
  • Solution (of an equation): A value (or values) for the variable that makes the equation true.
  • Isolate: To get a specific term or expression alone on one side of an equation by using inverse operations.

Detailed Notes

Understanding Absolute Value Equations

An absolute value equation is one where the variable is contained within absolute value bars. The fundamental idea is to find the value(s) that, when plugged into the expression inside the bars, make the equation true.

Recall that absolute value represents distance. So, an equation like $|x| = 5$ means "the distance of $x$ from zero is 5 units." There are two numbers that fit this description: $5$ and $-5$.

The Three Cases for Solving $| \text{expression} | = k$

Let expression represent any algebraic expression (like $x$, $x+3$, $2x-6$, etc.).

  1. Case 1: $k > 0$ (The right side is a positive number)

    • When the absolute value equals a positive number, there are always two solutions.
    • Rule: If $|\text{expression}| = k$ where $k > 0$, then $\text{expression} = k$ OR $\text{expression} = -k$.
    • Example: Solve $|x + 3| = 7$.
      • Here, $k = 7$, which is positive.
      • Split into two separate linear equations:
        • $x + 3 = 7$
        • $x = 4$
      • OR
        • $x + 3 = -7$
        • $x = -10$
      • Solutions: $x = 4$ or $x = -10$.
      • Check:
        • For $x=4$: $|4 + 3| = |7| = 7$. (Correct)
        • For $x=-10$: $|-10 + 3| = |-7| = 7$. (Correct)

  2. Case 2: $k = 0$ (The right side is zero)

    • When the absolute value equals zero, there is exactly one solution.
    • Rule: If $|\text{expression}| = 0$, then $\text{expression} = 0$.
    • Example: Solve $|2x - 6| = 0$.
      • Here, $k = 0$.
      • Set the expression inside the bars equal to zero:
        • $2x - 6 = 0$
        • $2x = 6$
        • $x = 3$
      • Solution: $x = 3$.
      • Check: For $x=3$: $|2(3) - 6| = |6 - 6| = |0| = 0$. (Correct)

  3. Case 3: $k < 0$ (The right side is a negative number)

    • When the absolute value equals a negative number, there is no solution.
    • Rule: If $|\text{expression}| = k$ where $k < 0$, then there is no solution.
    • Reasoning: The absolute value of any real number can never be negative. It represents a distance, and distance is always non-negative.
    • Example: Solve $|x - 1| = -5$.
      • Here, $k = -5$, which is negative.
      • Since absolute value cannot be negative, there is no solution.
      • Caution: Always look at the sign of the constant on the right side after isolating the absolute value. If it's negative, you can stop immediately.

Isolating the Absolute Value First

Sometimes, an absolute value equation might have other terms or coefficients outside the absolute value bars. Before you can apply the "split into cases" property, you must first isolate the absolute value expression. This means getting the absolute value expression by itself on one side of the equation.

  • Process: Use inverse operations (addition/subtraction, multiplication/division) to move any terms outside the absolute value bars to the other side of the equation.

  • Example (Conceptual, not in text, but crucial for understanding): Solve $2|x + 1| - 5 = 15$.

    1. Add 5 to both sides: $2|x + 1| = 20$
    2. Divide both sides by 2: $|x + 1| = 10$
    3. Now, isolate and solve as in Case 1 (since 10 > 0):
      • $x + 1 = 10 \implies x = 9$
      • OR
      • $x + 1 = -10 \implies x = -11$
    • Solutions: $x = 9$ or $x = -11$.

  • Application Example: A thermostat set to $72,^F$ turns on when the temperature differs by $3$ degrees. This is modeled by $|t - 72| = 3$.

    • The absolute value is already isolated, and the right side is $3$ (positive).
    • Split into two equations:
      • $t - 72 = 3 \implies t = 75$
      • OR
      • $t - 72 = -3 \implies t = 69$
    • Conclusion: The thermostat turns on at $69,^F$ and $75,^F$.

Remember to always check your solutions in the original equation to ensure they are valid.

Review Questions

  1. Explain in your own words why an absolute value equation can have two solutions, one solution, or no solution. Provide an example for each scenario.
  2. Solve the equation $|x - 8| = 15$. Show all steps and check your solutions.
  3. What is the first step you should take when solving an equation like $4|2x - 1| + 7 = 31$? Why is this step necessary?
  4. Solve the equation $|5x + 10| = 0$. How many solutions does it have, and why?
  5. Consider the equation $|y + 2| = -9$. Without performing any calculations, what can you say about its solution(s)? Explain your reasoning.
